# 大纲 — 《这个破碗觉醒黄金眼》第11-20集（纯致富主线）

> **重写日期**: 2026-06-25
> **核心原则**:
> 1. **只做一件事：鉴宝→捡漏→攒钱。** 不换地图，不搞组织对抗，不让反派知道秘密。
> 2. **对标火眼金睛的节奏**：每次捡漏 2-3 集，一个完整的"发现→拿下→变现→钱到账"循环。
> 3. **冲突来自"看不起乞丐"**，不是来自"想抢他的眼睛"。每次换一拨新的人看不起他，然后被打脸。
> 4. **钱慢慢涨，生活慢慢变好。** 衣服从破到新，住处从桥洞到租房到买房，交通工具从走路到电动车到买车。

## E14-E16：一件大货（3集）

### E14：大件

| 维度 | 内容 |
|------|------|
| **核心事件** | 一个老实巴交的农民抱着一个蛇皮袋进了林记古玩。打开——一件青铜器，通体绿锈，器型像鼎又像炉。农民说是修房子挖地基挖出来的，跑了三家店都不收——"说是不吉利，是冥器。"林北金瞳扫过——信息：西汉青铜温酒器，真品，出土痕迹与农民描述吻合，价值180w。但器身有一道细裂纹——运输中磕的。林北："这东西我收。但你这有道裂纹——修好了值钱，修不好就废了。我给你30w。不还价。"农民眼眶红了："30w够了——够给俺娘做手术了。"林北写了支票，农民千恩万谢走了。顾柔过来看到青铜器——"你收了件什么？"林北翻过器底给她看铭文："西汉的——如果没那道裂纹，能值250w。"顾柔："你能修？"林北的金瞳扫过那道裂纹——Lv2触发，闪回画面：2000年前的匠人铸造这尊温酒器，铜料来自云南，工艺是西汉宫廷造办处——"修不了原样。但能修到看不出来。" |
| **在场人物** | 林北、农民、顾柔 |
| **本集财富** | 30w收购→修好后估值180w，浮盈150w。但关键是——这是林北第一次收"大货"，从几十万的捡漏升级到百万级 |
| **爽点** | 信息碾压（三家店不收，林北一眼认出西汉真品）+ 财富碾压（30w→180w，六倍回报）+ 金瞳Lv2触发（看到铸造历史=能力在生意中自然进化） |
| **弹簧** | 压 — 收了件需要修的青铜器，能不能修好是悬念 |
| **钩子** | 悬念 — 裂纹能修好吗？修好了能卖多少？ |
| **字数** | ~550 |

## E17-E18：换种玩法——帮人鉴宝收费（2集）

### E17：第一个客户

| 维度 | 内容 |
|------|------|
| **核心事件** | 林记古玩开业一个月。林北发现一个问题：整天等客人上门，太慢了。他在门口贴了张纸——"代客鉴宝，一次500。看走眼双倍赔偿。"路人笑——"一个开破店的要饭的，还代客鉴宝？"下午，一个中年女人犹豫着进来，从包里掏出一个锦盒——"我老公花20w买的'清代翡翠镯子'，我觉得不对。你帮我看看。"林北金瞳扫过——信息：B货翡翠，酸洗注胶，价值200元。林北："假的。B货——酸洗过的，戴久了伤皮肤。"女人脸色变了——蹲在地上哭："那是他半年的工资……"林北等她哭完，倒了杯水："你老公在哪买的？"女人说了店名——聚珍阁，古玩街最大的店。林北："拿着这个鉴定结果去找市场管理处。他们不敢不退。"女人掏出500块，林北没收——"钱不用给了。你让你老公来我店里一趟——他要是喜欢翡翠，我帮他找真的。" |

| **爽点** | 打脸（门口嘲笑的路人看到鉴定结果后闭嘴）+ 信息碾压（聚珍阁的真面目被揭开一角）+ 身份反转（乞丐→帮人维权的鉴定师） |
| **弹簧** | 放 |
| **钩子** | 悬念 — 女人的老公会来吗？聚珍阁卖假货——林北要管吗？ |
| **字数** | ~550 |

## E19-E20：别人看不上的，就是他的金矿（2集）

### E19：废料

| 维度 | 内容 |
|------|------|
| **核心事件** | 江城赌石场有一批废料要处理——堆了两年，全是切垮的，按吨卖，一吨5w。没人要。林北路过时金瞳扫了一眼——废料堆深处，一块被切过两刀的石头，内部有一团翠绿。信息：冰种翡翠，价值120w。石头被切了两刀都没切到——因为切的角度不对，偏离了翡翠的位置三公分。林北找到赌石场老板："这堆废料我要了。不过我不按吨买——我挑几块，一块1000。"老板巴不得有人清垃圾——答应了。林北在废料堆里翻了一个下午，挑出七块石头——其中一块里面有翡翠（120w），两块有糯种（各值5-10w），四块确实是废的。总成本7000。当晚在自己店里切开那块关键的——冰种翡翠，完美无裂纹。顾柔接到电话赶来："你又从废料里切出东西了？"林北把翡翠放在灯下——绿光照亮顾柔的脸。"别人看不上的东西——就是我的金矿。" |
| **在场人物** | 林北、赌石场老板、顾柔 |
| **本集财富** | 废料堆挑货：成本7000→冰种翡翠120w+两块糯种15w=135w。回报率19200%。累计身价接近1000w |
| **爽点** | 实力碾压（别人扔了两年的废料，金瞳一眼看穿）+ 信息碾压（三公分偏差=120w的差距）+ 财富碾压（7000→135w）+ 打脸（赌石场老板以为他在清垃圾） |
| **弹簧** | 大放 |
| **钩子** | 悬念 — 林北身价马上突破1000w了。下一步是什么？ |
| **字数** | ~550 |

### E20：一千万

| 维度 | 内容 |
|------|------|
| **核心事件** | 冰种翡翠在聚宝斋寄售——三天内被一位省城来的藏家以130w收走。林北坐在林记古玩的柜台后面，用计算器把所有的数字加了一遍：宣德炉20w+玉扳指40w+赌石60w+拍卖赚的70w+开店以来各种捡漏买卖+青铜器210w+冰种翡翠130w……计算器跳出最终数字——1015万。他把计算器转过去给顾柔看。顾柔坐在他对面，端着一杯茶——她三个月前在这个位置给了林北20w的支票。现在，她在这个男人的店里喝茶。林北站起来，走到店门口，看着古玩街的灯火——三个月前，他在这条街的另一头，被古玩堂的保安扔出门，手里只有一个破碗。现在，这条街上有了一家"林记古玩"。他从怀里摸出那块破碗碎片——在灯光下，碎片上的金光已经不在了，但碗上的纹路还是当年的纹路。顾柔走到他旁边："想什么呢？"林北收起碎片，笑了笑："想明天再去趟废料堆——说不定还有漏。"——画面切至古玩街全景。林记古玩的灯笼亮着。街上人来人往。一个乞丐蹲在街角——林北路过时，在他碗里放了一百块钱。乞丐抬头，林北已经走远了。 |
| **在场人物** | 林北、顾柔、街角乞丐 |
