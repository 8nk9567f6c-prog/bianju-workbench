## 一、第1集结构 —— 黄金开局模板（四剧高度统一）

四部剧的第1集遵循**完全相同的剧本骨架**，这是经过市场验证的黄金公式：

```
[00:00-00:07]  黄金三秒：强烈身份冲突画面
[00:07-00:20]  身份建立：VO自述"我叫xxx，我是xxx，我处于最低点"
[00:20-00:45]  受辱事件：被反派殴打/羞辱 → 情绪压到最低
[00:45-01:10]  金手指激活：意外触发 → 眼中异象 → 能力初现
[01:10-01:30]  首次验证：立刻展示能力起作用 → 主角震惊
[01:30-01:50]  小反转/情绪上扬：主角从绝望到看到希望
[01:50-02:00]  钩子卡点：新悬念/新冲突的暗示
```

### 逐剧对照

#### 《黄金疯涨》第1集
```
00:00  矿工挖金+持枪守卫 → 压迫感画面
00:10  VO：我叫林凡，被骗到东南亚当黑工
00:20  林凡发现金条
00:30  沙东踹翻林凡，枪指头 → 极端羞辱
00:45  金龙入眼 → 黄金瞳激活
01:00  看到金条信息"天然金条，重11g"
01:10  扫视地面，看到地下埋的金子 → 能力验证
01:25  老马求情被打 → 冲突未完全解决
01:40  林凡朝沙东背影吐口水 → 埋下反抗种子
```

**钩子设计**：结尾是林凡"呸！"——情绪宣泄但非悬念钩。第2集靠"又挖到金子"自然延续。

#### 《觉醒透视眼后》第1集
```
00:00  黑矿区+持枪守卫+黑奴被枪杀 → 极高压迫感
00:10  VO：我叫林诺，被骗到境外黑矿区
00:20  黑奴偷石头被一枪击毙 → 生命的廉价
00:30  林诺发现发光石头 → 光进入眉心
00:45  眼中泛金光 → 看到"糯种翡翠，价值1000000"
01:00  扫视周围，所有石头都有价值面板 → 能力验证
01:15  玛卡踢倒林诺，掏枪 → 危险逼近
01:30  吉普车驶来 → 新人物即将出场
```

**钩子设计**：吉普车+高跟鞋下车 → "来的是谁？"悬念钩子。**比黄金疯涨更强**。

#### 《通天神眼》第1集
```
00:00  保安姜枫+女儿小小在古玩街 → 温情+身份
00:10  王宇举花瓶走路 → 潜在冲突
00:15  王宇摔倒，花瓶碎裂 → 冲突触发
00:25  王宇抓住小小，索赔50万 → 讹诈事件
00:35  姜枫与王宇扭打，手被碎片划伤
00:45  金光入眼 → 看到"民国仿制青花瓷瓶，价值三千元"
01:00  扭头看桌上古物，都能看到价格 → 能力验证
```

**钩子设计**：能力验证完成即结束。钩子较弱，靠"接下来会怎样"自然延续。

#### 《寻宝黄金瞳》第1集
```
00:00  赵桐翻垃圾桶 → 身份最低点
00:05  找到塑料金元宝 → 假希望
00:15  王大壮撞倒赵桐，骂"臭乞丐"
00:20  王大壮脚踢金元宝 → 金元宝化作光飞入赵桐眼
00:35  赵桐捂眼，OS"金元宝真的进入了我的眼睛"
00:40  睁眼，视野中看到树下金条 → 能力激活即验证
00:50  赵桐挖出金条，咬一口确认 → 小爽点
```

**钩子设计**：咬金条确认是真的 → 以确认能力为终点，钩子是"接下来拿金条去换钱"。

### 第1集关键发现

1. **四部剧的VO开场完全一致**："我叫xxx，我是xxx（最低身份），被骗/被命运逼到xxx"——这是类型铁律。
2. **能力激活方式决定辨识度**：
   - 黄金疯涨：金龙入眼 → 意象最强
   - 觉醒透视眼后：光入眉心 → 常规
   - 通天神眼：血+碎片 → 最弱
   - 寻宝黄金瞳：塑料金元宝入眼 → **最有记忆点**
3. **能力必须立即验证**：四部剧都在激活后30秒内展示能力效果。
4. **第1集反派必须露面**：四部剧100%在第1集让反派出现并制造伤害。

## 二、第2集结构 —— 能力二次验证 + 冲突升级

```
[00:00-00:15]  接上集，能力继续展示
[00:15-00:45]  能力加深理解：主角发现能力不止表面功能
[00:45-01:15]  与反派的二次冲突：更危险、更接近死亡
[01:15-01:45]  贵人/关键人物出场：改变局势
[01:45-02:00]  新钩子：更大的机会/危险
```

### 逐剧对照

#### 《黄金疯涨》第2集
```
林凡继续挖金 → 发现能吸收金子能量 → 力量变强 → 老张质疑"你还能挖到？"
→ 林凡又挖到（打脸质疑者）
钩子：林凡又挖到金子（以重复打脸为钩子）
```
**特点**：第2集的核心是"能力进化"——吸收能量→力量变强→挖更多。建立了能力成长的预期。

#### 《觉醒透视眼后》第2集
```
杜温丽下车 → 玛卡变脸讨好 → 林诺好奇"这人是谁？" → 交底：矿场老板
→ 玛卡检查玉石 → 骂众人废物 → 踢林诺的筐 → 杜温丽问"谁最差"
→ 玛卡指林诺 → 杜温丽"按规矩，杀了"
钩子：林诺捡起石头"这块石头，价值一百万" → 命悬一线的反杀预告
```
**特点**：节奏最快的第2集。贵人出场→规则建立(谁差杀谁)→死亡威胁→主角反击。情绪曲线最陡。

#### 《通天神眼》第2集
```
王宇继续索赔 → 小小护父 → 姜枫反击"花瓶不值五十万" → 赌约建立
→ 苏大山出场 → "如果能证明是假的，赔十倍鉴定费" → 姜枫接受挑战
钩子：姜枫举起瓷片让苏大山用放大镜看 → 即将揭晓
```
**特点**：建立"赌约"机制——50万鉴定费的赌注，给观众明确的期待。贵人（苏大山）是仲裁者角色。

#### 《寻宝黄金瞳》第2集
```
赵桐发现视野中有地图 → 能定位方圆500米宝藏 → 去典当行
→ 进门遇到王大壮 → 王大壮认出他"臭乞丐滚出去"
钩子：王大壮赶人 → 冲突即将爆发
```
**特点**：能力深化（从看到→定位）→ 场景转换（户外→室内）→ 巧合遇到仇人。爽点延迟到第3集。

### 第2集关键发现

1. **能力必须升级/深化**：第2集不能只是重复第1集的能力展示，必须有"原来还能这样！"的新发现。
2. **冲突必须升级**：从被骂→被威胁生命；从被推→被枪指；从索赔→赌约。
3. **贵人出场时机**：第2集中后段是最佳时机（四剧中有3部在此出场）。
4. **第2集结尾必须是生死赌注**："价值一百万" / "十倍鉴定费" / 即将被赶出门。

## 三、第3集结构 —— 小高潮/反转打脸

```
[00:00-00:20]  钩子回收：上集悬念揭晓
[00:20-00:50]  打脸现场：反派被事实打脸
[00:50-01:20]  贵人认可：获得资源/机会/金钱
[01:20-01:45]  能力再进化/新发现
[01:45-02:00]  新钩子：更大的世界/新威胁
```

### 逐剧对照

#### 《黄金疯涨》第3集
```
众人震惊"牛逼啊" → 老陈求教 → 林凡得意"你们学不会" → 老张反复质疑
→ 林凡反复挖到金子（打脸循环×3） → 守卫催促 → 老马提醒"要交公"
→ 林凡OS"要是能藏起来就好了" → 金子消失又出现 → 觉醒空间异能
钩子：林凡发现体内空间 → 沙东在远处盯着他看 → 暗线"被盯上了"
```
**特点**：打脸循环是最密集的（老张质疑5次，每次都被打脸）。能力再次进化（透视→空间）。钩子暗藏危机。

#### 《觉醒透视眼后》第3集
```
玛卡骂"放屁，破石头" → 杜温丽"等一下，拿上来" → 玛卡切石
→ 第一刀灰白 → 众人"什么都没有啊" → 杜温丽"那就杀了吧"
→ 林诺"等等！切的位置不对，再切一次一定能出翡翠"
钩子：绝境中的一线生机——"再切一次"
```
**特点**：最紧张的第3集。切石过程制造了"假失败"的过山车情绪——先让观众和主角一起绝望，再给希望。钩子最强。

#### 《通天神眼》第3集
```
苏大山用放大镜看瓷片 → 确认是仿制品 → 王宇崩溃 → 苏大山道歉
→ 转账50万 → 小小羡慕上学的小女孩 → 姜枫"明天带你去最好的学校"
钩子：校长"没房上什么学呀"——新障碍出现
```
**特点**：打脸完成得最彻底（收到50万现金+工作邀请）→ 建立核心驱动（女儿上学）→ 新障碍立刻出现。

#### 《寻宝黄金瞳》第3集
```
赵桐"我是来卖金条的" → 王大壮不屑"你要是能拿出来，我吃桌子"
→ 赵桐拍出金条 → 王大壮咬金确认是真 → 想骗低价收 → 许琴出场
→ 许琴鉴定"9999千足金" → 开除王大壮 → 10万收金条
钩子：最弱——交易完成，无新悬念
```
**特点**：打脸最完整（反派被开除+主角拿到10万）。但钩子设计弱于其他三部。

### 第3集关键发现

1. **打脸必须分层**：质疑→半信半疑→被打脸→恼羞成怒→彻底被打脸。不能一次到位。
2. **贵人必须给实质资源**：钱（50万/10万）、工作机会、信息。不能只是口头认可。
3. **第3集结尾必须有新目标/新障碍**：学区房/被盯上/再切一刀。否则观众不会看第4集。

## 五、钩子设计 —— 六种卡点手法（四剧验证）

### 手法1：揭晓前卡点（最常用）
```
用法：关键信息即将揭晓→黑屏
实例：
- 觉醒：切石第一刀灰白→"再切一次"→黑屏
- 通天神眼：举起瓷片→"用放大镜看"→黑屏
频率：每集结尾
强度：★★★★★
```

### 手法2：能力进化钩
```
用法：主角发现能力的新功能→展示未完成→黑屏
实例：
- 黄金疯涨：金子消失"啊？！"→黑屏（空间能力初现）
- 寻宝黄金瞳：看到地图→定位附近宝藏→黑屏
频率：第2-3集
强度：★★★★
```

### 手法3：冲突升级钩
```
用法：反派放出狠话/掏枪/威胁→黑屏
实例：
- 觉醒：玛卡掏枪"老子弄死你"→吉普车来了→黑屏
- 寻宝：王大壮"赶紧给我滚出去"→黑屏
频率：每集可用
强度：★★★
```

### 手法4：贵人出场钩
```
用法：关键人物出场但不露脸/不交底→黑屏
实例：
- 觉醒：高跟鞋从吉普车下来→黑屏
- 黄金疯涨：（较弱，无此类型）
频率：第2集
强度：★★★★
```

### 手法5：赌约/倒计时钩
```
用法：建立明确的赌注和截止时间→黑屏
实例：
- 通天神眼："能证明是假的，赔十倍"→黑屏
- 觉醒："价值一百万，你活；没有，就死"→黑屏
频率：第2-3集
强度：★★★★★
```

### 手法6：新障碍钩
```
用法：刚解决一个问题，新问题立刻出现→黑屏
实例：
- 通天神眼：拿到50万→校长"没房上什么学"→黑屏
- 黄金疯涨：觉醒空间→沙东在远处盯着→黑屏
频率：第3集结尾
强度：★★★★
```

### 钩子频率要求
- **每集必须有1个结尾钩子**
- **每3-5集需要1个强力钩子**（类型1/5/6）
- **连续两集不能使用同类型钩子**

## 六、人物配置 —— 前3集的固定班底

### 必需角色（四剧100%配备）

| 角色类型 | 功能 | 出场时间 | 示例 |
|----------|------|:------:|------|
| 主角 | 草根，最低身份 | 0秒 | 林凡/林诺/姜枫/赵桐 |
| 反派小头目 | 制造伤害，打脸对象 | 第1集 | 沙东/玛卡/王宇/王大壮 |
| 同伴/见证者 | 信息传递，反应镜头 | 第1-2集 | 老马/老张/小小/（无） |
| 贵人/老板 | 认可主角，给资源 | 第2-3集 | 老马(弱)/杜温丽/苏大山/许琴 |
| 质疑者/跟风者 | 打脸循环的燃料 | 第2-3集 | 老陈/老张/黑奴们/路人 |

### 角色功能公式
```
反派 = 制造伤害 + 被打脸
贵人 = 认可 + 给资源
同伴 = 信息传递 + 反应镜头（震惊/质疑/羡慕）
群众 = 打脸现场的"见证者"（"卧槽""牛逼"）
```

## 七、对话风格 —— 四剧共同特征

### 1. 每句不超过15字
```
✅ "妈的，快点！快点干！"
✅ "哎哟，卧槽。"
✅ "我去。"
❌ "你怎么能这样对我，我明明没有做错任何事情。"
```

### 2. 情绪词前置
```
"哎哟，卧槽" → 不是"我感到很震惊"
"妈的" → 不是"真是岂有此理"
"我去" → 不是"这简直不可思议"
```

### 3. 重复递进（打脸循环专用）
```
老张："林凡又挖到金子了。"
老张："林凡又又挖到金子了。"
老张："林凡又又又挖到金子了。"
```
每次重复加一个"又"，情绪逐级推高。

### 4. 信息只在VO里交代
```
VO："我叫林凡，三年前怀揣发财梦……"
对话：绝不说背景信息，只说当下反应。
```

### 5. 无废对话
每句台词要么推进情节，要么展示人物，要么制造情绪。没有"吃了吗""今天天气不错"。

## 八、场景设置的效率原则

### 四剧场景使用

| 剧名 | 第1集场景数 | 第2集场景数 | 第3集场景数 |
|------|:--------:|:--------:|:--------:|
| 黄金疯涨 | 1（矿区） | 1（矿区） | 1（矿区） |
| 觉醒透视眼后 | 1（矿区） | 1（矿区） | 1（矿区） |
| 通天神眼 | 1（街道） | 1（街道） | 2（街道+办公室） |
| 寻宝黄金瞳 | 2（垃圾桶+树下） | 2（街道+典当行） | 1（典当行） |

**铁律：前3集场景越少越好，最多2个场景。场景切换=观众流失风险。**

## 九、能力系统的设计规律

### 四剧能力对比

| 维度 | 黄金疯涨 | 觉醒透视眼后 | 通天神眼 | 寻宝黄金瞳 |
|------|:--------:|:--------:|:--------:|:--------:|
| 能力名 | 透视眼 | 透视神眼 | 鉴定之眼 | 寻宝黄金眼 |
| 激活方式 | 血+金龙入眼 | 光入眉心 | 血+碎片 | 塑料金元宝入眼 |
| 首次展示 | 看金条重量 | 看玉石价值 | 看花瓶真伪+价格 | 定位金条位置 |
| 第2集进化 | 吸收能量+变强 | （未展示） | （未展示） | 500m地图定位 |
| 第3集进化 | 空间存储 | （未展示） | （未展示） | （未展示） |
| 有无代价 | 暂无 | 暂无 | 暂无 | 暂无 |

### 能力设计铁律
1. **激活方式必须有视觉辨识度**：金龙/金光/金元宝——都要有"画面"
2. **能力必须立即验证**：激活后30秒内展示第一个能力效果
3. **能力展示必须有UI**：半透明面板+文字信息——这是类型惯例
4. **能力必须逐集进化**：第1集基础→第2集深化→第3集扩展
5. **前3集暂不需要展示代价**（四剧都没有在第1-3集展示代价系统）

## 十、与旧对标报告的差异（重要）

旧对标报告分析的是《神眼鉴宝》《双眼变异》《通天神眼》《乞丐赠我通天眼》——这些是**已完结的完整剧集**。
新对标的是四部**正在流行的爆款新剧**，差异显著：

| 维度 | 旧对标（成熟剧） | 新对标（流行新剧） |
|------|:-----------:|:-----------:|
| 场景 | 多场景切换 | **单场景**或极少切换 |
| 人物密度 | 每集3-5个有名有姓 | **2-3个**核心人物 |
| 对话长度 | 允许3-5句交锋 | **每句不超过15字** |
| 能力进化 | 缓慢，10集一次 | **每集都有新发现** |
| 爽点密度 | 每5-8集一次大爽 | **每集都有小爽点** |
| 钩子类型 | 6种均匀使用 | **揭晓前卡点为主** |
| VO使用 | 较少 | **大量使用**（交代背景） |
| 情绪节奏 | 30秒一波 | **更快，15-20秒一波** |

**核心发现：新剧比旧剧更快、更直接、更少废话。场景极简，对话极短，能力进化极快。**

## 十一、自主生成模板（可直接使用）

### 前3集标准化模板

```
=== 第1集：觉醒篇 ===

[场景：单一极限环境]

△【黄金三秒】极端身份画面（矿场/矿区/街头/垃圾桶）
主角VO：我叫xxx，我现在的身份是xxx（最低点），因为xxx来到这里。

△【受辱事件】
反派出场 → 对主角造成身体伤害（踹/打/枪指）→ 主角无法反抗

△【金手指激活】
意外触发（血/碰撞/物品入眼）→ 异象（金光/光柱/光入眉心）→ 眼中泛金

△【首次验证】
看到某物的信息面板 → "xxx，价值/重量xxx" → 扫视环境确认能力

△【小反转/情绪上扬】
主角意识到能力 → 从绝望转为有希望 → 但外在处境未改变

△【钩子】
反派再靠近 / 新人物即将出场 / 主角发现能力不止于此
→ 黑屏


=== 第2集：深化篇 ===

△【接上集钩子】
能力继续展示 → 反派或环境压力继续存在

△【能力新发现】
发现能力可以xxx（吸收/定位/鉴定更深层信息）→ "原来还能这样！"

△【冲突升级】
反派的威胁从语言升级到行动（掏枪/下杀手/驱逐）

△【贵人出场】
关键人物出现 → 改变力量对比 → 不一定是友，但改变了局面

△【赌约/倒计时】
建立明确赌注："如果能xxx，就xxx；如果不能，就死"

△【钩子】
即将揭晓赌约结果 / 贵人身份 / 能力的代价
→ 黑屏


=== 第3集：小高潮篇 ===

△【接上集钩子】
赌约揭晓的第一部分 → 可以是假失败（先绝望再反转）

△【打脸现场】
反派被事实打脸 → 从嚣张→怀疑→震惊→哑口无言

△【贵人认可】
给实质资源（钱/机会/信息）→ 主角获得第一桶金

△【核心驱动建立】
明确主角为什么要用这个能力（救家人/还债/上学/翻身）

△【能力再进化/新发现】
能力的新功能 → 建立"这个能力还有很多秘密"的预期

△【钩子】
新障碍 / 新威胁 / 更大的世界 → 必须让观众想看第4集
→ 黑屏
```

### 生成时的参数设定

```
- 每集事件数：4-5个
- 每集字数：800-1200字（剧本格式）
- 场景数：1-2个
- 有名有姓人物：2-3个
- 对话最长句：15字
- VO使用：仅用于交代背景/能力规则/内心重大决定
- 钩子类型：优先使用"揭晓前卡点"和"赌约/倒计时"
- 爽点：每集至少1个小爽点（能力展示/打脸/获得资源）
```

## 十二、关键教训（避坑指南）

1. **不要在第1集展示能力的全部功能**——四剧都在第2-3集才展示进化。留悬念=留观众。
2. **不要在第1集让主角反击成功**——主角第1集只能受辱。反击留到第2-3集。
3. **贵人不能是救世主**——贵人给机会，主角靠能力证明自己。四剧无一例外。
4. **打脸对象不能太强**——前3集的反派都是小头目（监工/保安/店员），不是大Boss。
5. **VO不能超过3句/集**——四剧每集VO使用1-2次，每次1-2句话。
6. **"巧合"只能用于相遇，不能用于解决问题**——主角靠能力解决问题，不靠运气。
7. **第3集结尾必须有"新问题"**——否则是完结感，不是连载感。
