## 第一章：市场格局

### 1.1 宏观赛道定位

2026年微短剧市场规模已突破**680亿元**，用户规模达**6.96亿**。男频在全品类微短剧中占比从早期不足10%升至**17.2%**，且仍在快速扩容。红果、抖音、快手三大平台均将男频列为2026年战略重点方向，红果男频剧本保底已提高至**20万**，分成比例提升至**40%**。

**透视鉴宝**品类在当前赛道结构中定位：

```
一级赛道：男频
 └─ 二级赛道：现代都市
      └─ 三级品类：都市逆袭剧 / 都市玄幻剧
           └─ 热门类型：鉴宝天师、透视异能、系统加持、空间异能
                └─ 核心爽点：身份反差 + 打脸闭环 + 超能力碾压
```

## 第六章：创作简报

### 6.1 类型与基调

| 项目 | 定义 |
|------|------|
| **类型** | 都市透视鉴宝 · 纯暴富爽剧 |
| **总集数** | 100集（对标《乞丐赠我通天眼》129集精简优化版） |
| **预估总字数** | ≥ 50,000字 |
| **单集字数** | 500-600字 |
| **基调** | 爽燃（70%）+ 紧张（20%）+ 喜剧（10%） |
| **目标情绪** | 爽感（打脸/暴富/碾压）为主，好奇（代价/身份）为辅 |

### 6.2 必须要素（Must Have）

1. **手心透视视觉锚点**：每集至少1次手心透视特效展示（主动扫描/深金鉴定/危险预警/穿透视野）
2. **貔貅神兽意象**：关键节点（觉醒/代价/升级）反复出现貔貅画面，形成视觉记忆
3. **捡漏→暴富循环**：每2-3集一次捡漏/暴富节点，每次捡漏倍数≥100倍
4. **王胖子喜剧爆破**：每3-5集一次王胖子搞笑/助攻场景
5. **掌柜变现仪式**：每次捡漏后通过掌柜完成变现，强化"钱到账"的爽感
6. **信息差打脸**：每集至少1个"主角知道/反派不知道"的信息差打脸场景
7. **代价倒计时**：E1-E9暗中铺垫代价线索→E10全面引爆→E11-100代价与能力博弈
8. **光头→幕后Boss升级线**：光头(E5-20)→幕后势力(E21-50)→终极Boss(E51-100)

### 6.3 规避要素（Must NOT Have）

- ❌ 感情线（女主角/恋爱/暧昧）
- ❌ 家庭线（父母/子女/亲情羁绊——除非作为代价/软肋出现）
- ❌ 复杂阴谋（超过一句话能讲清的阴谋一律删除）
- ❌ 主角被动超过1集（林北必须主动选择）
- ❌ 伏笔超过3集不回收
- ❌ 连续3集同类型钩子
- ❌ 人物超过6人
- ❌ 对白超过20字/句
- ❌ AI腔/说教/文艺对白

### 6.5 差异化要点（核心竞争力）

1. **手心透视** — 唯一视觉锚点，所有竞品均在"眼睛"，本剧拍摄时可设计特写手心裂开+眼球眨动的视觉奇观
2. **貔貅入掌** — 神话神兽赋予文化厚度，"只进不出"的貔貅性格与"暴富"主题天然契合
3. **王胖子** — 忠诚搞笑小弟，对标剧普遍缺乏的喜剧元素，填补赛道空白，增加"兄弟情"情感锚点
4. **零感情线** — 释放30%剧情空间用于纯暴富内容，对标剧中感情线平均占用20-30%时长
5. **代价E10后置** — 先让观众充分感受"透视有多强"，第十集再打出"代价有多大"，反差冲击力最大化
