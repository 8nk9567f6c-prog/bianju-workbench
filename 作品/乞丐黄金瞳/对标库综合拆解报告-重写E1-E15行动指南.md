### 2.3 SS版比旧版强在哪

1. **第1集就有打脸**：SS版林北追上保安队长，用透视能力揭穿他全身假货——观众在第一集就爽到了。旧版只是被动接受500元离开。
2. **每集结尾是画面不是OS**：SS版三个钩子全是画面化悬念（后备箱透光/保安围过来/炉底旋痕），旧版E1用OS收尾=弱钩子。
3. **冲突直面不绕**：SS版林北不逃跑——被压价就揭穿镇店之宝，被围就打。旧版林北更被动。
4. **贵人分层出场**：SS版顾柔从高跟鞋声→进门→裁决→给卡，出场即完成"公正权威+商业头脑+对林北好奇"三层塑造。
5. **场景极简**：SS版前3集=3个场景（街口/店内/店内），旧版=5个场景切换。

### 2.4 SS版比旧版弱在哪

1. **缺失情感锚点**：SS版没有"亡父遗物"的设定——这是旧版/自写版与所有竞品的核心差异。
2. **缺失能力视觉辨识度**：SS版"金光入眼"过于常规，旧版"手心裂开眼睛+貔貅奔入"记忆点更强。
3. **缺失草根幽默**：SS版林北严肃直接，缺少自写版"老板你拿我当日本鬼子整啊"的讨价还价幽默。
4. **人物过简**：SS版王二=一次性反派，没有持续性。旧版如果保留自写版的"王二贯穿10集"会更优。

### 4.3 自写版已丢失的好东西（需要在新版中恢复）

#### 丢失#1：顾柔的渐进式贵人线

自写版顾柔有五个层次：
- 第3-4集：出场解围 → 展现权威
- 第5集：追出来收宝 → 展现好奇
- 第8集：送VIP卡 → 展现商业布局
- 第9集：邀请拍卖会 → 展现合作升级
- 第10集：并肩坐第一排 → 关系递进

**当前版本替换为掌柜**：掌柜是"精明但讲信用"的亦商亦友——功能上能完成变现渠道，但缺失了贵人的"权威感+神秘感+商业资源"三角。

**建议**：在重写E3-E10时，恢复顾柔作为"隐藏贵人"——掌柜做日常变现（E3-E8），顾柔在E8首次出场（赌石场VIP卡+暗示知道更多），E9-E10正式露面。这样既保留掌柜"老狐狸"的趣味性，又有顾柔的层次感。

#### 丢失#2：王二的"反复打脸"喜剧效果

自写版王二：
- 第2-4集：聚宝斋掌柜→被辞退
- 第6集：带人巷战报复→失败
- 第10集：投靠沈均→"你当狗挺称职"

**当前版本替换为光头男+刘三**：
- 光头男是幕后型反派（打电话/指挥），缺少"被当场打脸"的爽点
- 刘三是执行型反派（围堵/追捕），缺少王二的"可笑感"

**建议**：重写时，光头男维持幕后，增加一个类似王二的"台前可打脸反派"——可以使刘三升级为有性格的混混头目（增加他被林北戏耍、反复失败的喜剧线）。

#### 丢失#3：沈均+王二的"双反派趣味配置"

自写版E10结尾："王二，你当狗挺称职"——自写版最佳台词。

当前E10光头男上台揭穿→这个设计合理度没问题，但缺少了自写版中"前老板的狗投靠新主子"的双重背叛趣味。

**建议**：在E10赌石大会，让一个类似王二的小角色（曾被林北打脸过的人）投靠光头男，在大会现场认出林北→光头上台揭穿。这样揭穿更有"人证"说服力。

#### 丢失#4：幽默密度断崖式下降

自写版前10集幽默点分布：
- E1: 无（压抑开场）
- E2: "日本鬼子"/双线赚翻
- E3: "想啥好事儿呢"
- E4: "很合理吧"
- E5: 无
- E6: "孙贼拜拜了您嘞"/"君子不立危墙之下"
- E7: "低调低调"
- E8: 无
- E9: 无
- E10: "王二你当狗挺称职"

当前版E7-E15幽默点：**几乎为零**。

**建议**：重写时确保每3集至少1个幽默点。王胖子是天然的喜剧担当（他话多/搞笑/有点怂），E6-E15应充分利用。
