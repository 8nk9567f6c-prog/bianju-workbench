## 一、四剧结构骨架 —— 四幕黄金模型

```
第一幕：觉醒篇（1-5集）
最低点受辱 → 获得金手指 → 首次验证能力 → 贵人/观众认可
功能：建置人设 + 亮出核心期待 + 给第一个小爽点

第二幕：崛起篇（6-20集）
捡漏攒第一桶金 → 还债/买房/学区房 → 打脸初级对手
功能：反复强化"能力=财富"认知 + 建立还债压力引擎

第三幕：对抗篇（21-40集）
赌石/拍卖会/鉴宝大赛 → 遭遇专业对手 → 冲突升级到产业级别
功能：从"打脸个人"升级到"对抗势力" + 赌石是爽点密度最高区

第四幕：巅峰篇（41集+）
身世揭晓/惊天阴谋/终极对决 → 身份最高点爆发
功能：伏笔全额催收 + 身份大揭秘 → 情绪最高潮
```

## 二、单集节奏模板 —— 四剧高度统一

```
0-7秒     黄金三秒：强冲突/强悬念/强视觉 → 决定"滑走or留下"
7-30秒    困境铺开：主角受辱/被挑衅/发现端倪
30-90秒   能力发动：透视开启/识破真相/冲突推进
90-120秒  反转打脸：当众揭露/财富暴增/反派哑口无言
最后5秒   钩子卡点：画面定格在即将揭晓的瞬间
```

### 情绪波动铁律：每30秒一个波动

```
压抑(受辱) → 期待(透视开启) → 爽(打脸) → 好奇(钩子)
   0-30s        30-60s          60-90s      90-120s
```


## 六、当前《乞丐黄金瞳》诊断

### 6.1 已有内容（第1-2集）

**第1集：觉醒篇** ✅ 基本合格
- ✅ 黄金三秒：乞丐被扔出酒店（视觉冲突）
- ✅ 金手指激活：血染碎片+金色貔貅入眼（有画面感）
- ✅ 能力展示：看到包、石狮、花瓶的价格
- ✅ 启动资金：500元（具象目标）
- ⚠️ 问题：能力激活方式较常规（摔碎碗+血），缺少独特记忆点
- ⚠️ 缺钩子：第1集结尾没有强力钩子（只是"我要改写命运"os）

**第2集：首次捡漏** ✅ 结构正确
- ✅ 地摊场景——对标四剧标准开局场景
- ✅ 信息差打脸：观众+主角知道香炉值120万，摊主不知道
- ✅ 双线"赚翻了"反差笑点
- ✅ 电话钩子自然
- ⚠️ 问题：捡漏过于顺利，缺少"差点被识破"的紧张感
- ⚠️ 问题：200元买120万香炉——倍数6000倍，过于夸张，可信度存疑

### 6.2 对比四剧同集定位

| 维度 | 四剧第1-2集标准 | 乞丐黄金瞳当前 | 差距 |
|------|:------------:|:------------:|:--:|
| 开场冲突 | 极端羞辱+身份最低点 | 被扔出酒店 ✓ | 持平 |
| 金手指激活 | 有视觉辨识度的方式 | 血+碎片+貔貅 | 可更强 |
| 首次验证 | 立刻展示能力起作用 | 看到价格 ✓ | 持平 |
| 首次打脸 | 小人物打脸完成 | 摊主亏大了 ✓ | 持平 |
| 核心驱动 | 有具象目标 | 改写命运（较空泛） | **需补充** |
| 第1集钩子 | 强力悬念卡点 | os独白（弱） | **需强化** |
| 第2集钩子 | 新冲突/新人物 | 摊主手机响 ✓ | 持平 |

### 6.3 紧急建议

1. **补充具象驱动目标**：对标《通天神眼》"凑学区房"——林北为什么需要钱？给一个下沉市场能共鸣的具体理由（救家人/治病/还债/被赶出家门时的最后期限）

2. **强化第1集结尾钩子**：不能只是os。参考方式——林北眼睛金光未散，突然看到某样普通物件上浮现出一行红字："洪武御制 传国玉玺 价值——"画面黑屏。或者他路过某处，眼睛突然剧痛/看到不该看到的东西。

3. **第2集捡漏增加紧张感**：摊主中间应该怀疑一次——"你一个乞丐怎么认识宣德炉？"——林北需要用乞丐身份巧妙化解，这样最后摊主知道真相时的反差更强烈。

4. **尽早建立"规则"**：黄金瞳的限制是什么？冷却时间？次数限制？看清越贵的东西代价越大？四部剧都有代价系统，没有限制的金手指=没有戏剧张力。

## 七、对标《乞丐赠我通天眼》的重点参考

这部剧是《乞丐黄金瞳》的**最直接对标**——连主角起点都相同（都是乞丐）。以下是关键差异点，决定你的差异化方向：

| 维度 | 乞丐赠我通天眼（竞品） | 乞丐黄金瞳（本剧） |
|------|:-------------------:|:-----------------:|
| 能力来源 | 乞丐报恩赠予 | 血染碎片+貔貅入眼 |
| 能力名称 | 通天神瞳 | 黄金瞳 |
| 核心爽点 | 捡漏+赌石+打脸 | 待定 |
| 结构 | 纯直线129集 | 待定 |
| 差异化 | — | **能力来源**已有差异（血+神兽） |

**建议差异化方向**：
- "貔貅入眼"这个设定可以深度开发——貔貅只进不出，所以林北的黄金瞳可能对"花钱"有特殊反应？或者能看到"财运流向"而不仅仅是价格？
- 《乞丐赠我通天眼》是纯逆袭直线，你可以加一条"貔貅反噬"的暗线做差异化——能力越大，貔貅对宿主的要求越高，形成代价系统。
