## 主编砍集诊断

### 致命问题

**1. 一场景拆多集**
- 聚宝斋 Scene 003 → E2+E3+E4 = 3集（应为1-1.5集）
- 废料切石 Scene 008 → E7+E8 = 2集（应为0.5集）
微短剧1集1-2分钟，一个连续场景跨3集=观众等结果等3-6分钟，必砍。

**2. 废镜头集**
- E5 玉扳指：独立捡漏，不推剧情、不引新人、不升级冲突。对标中每次捡漏都绑定新场景/人物/冲突。
- E6 王二巷战：打完什么都没变。没后果的打戏=水。
- E9 连切蒙太奇：零冲突、零张力、零反派。"全中"不是爽点，没被打脸就没有爽。

**3. 拍卖会过长**
E10-E18 = 9集 = 已写集数一半。E11-E13连续3集主角故意输。对标《透视眼捡漏致富》鉴宝会仅5-6节点≈3-4集。

**4. 钩子疲软**
多数结尾是模糊悬念（"那是什么？""你到底是什么人？"），对标钩子是明确方向预告（"钩子：一起去潘家园"）。

## 重写清单

1. **新E2**：E2+E3+E4合成——进门→压价→冲突→顾柔→辞退→成交。一气呵成。
2. **新E3**：E6后半+E7+E8合成——进赌石场→挑废料→质疑→切→涨→VIP。不拆两集。
3. **新E5-E6**：拍卖前半不能主角连输3集。压缩"让"到最多1集，立刻切入黑皮料反击。
