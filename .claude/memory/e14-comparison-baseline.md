## 背景

哥哥让念念和DS各写了一版E14正文。两版哥哥都不满意，明天哥哥自己写一版。存档用于明日对比。

## 剧情锚点

E13结尾：拍卖会结束，林北2000w vs 680w赢沈均，风清扬认输递名片，名片背面手写"省城藏家私拍会·下月初八"。

## 念念版 E14

```
第14集
010 日 内 聚宝斋会客厅
人物：林北，顾柔
服装：林北（休闲装），顾柔（时尚裙）
道具：银行卡，名片，茶具
【聚宝斋会客厅，顾柔坐在茶台前，林北靠在椅背上。】
△顾柔将一张银行卡推到林北面前。
顾柔：废料里翻出来的紫罗兰，卖了三百五十万。钱归你。
△林北拿起卡翻看两面。
林北：从垃圾堆里捡的钱，花着就是香。
顾柔：省城私拍会下月初八，来的都是人精。（扫林北衣领）你得有身像样的行头。
林北：衣服可以买。（把卡拍在桌上）但我连个挂衣服的地方都没有。
顾柔：（一顿）你要买房？
林北：一个乞丐想睡床上，合理吧。
△顾柔嘴角一扯，没接话。

011 日 外 街边
人物：林北，中介，房东
服装：林北（休闲装），中介（衬衫西裤），房东（旧夹克，眼袋发青）
道具：电动车，钥匙，杂物堆，瓷瓶
【林北从电动车上下来，中介满脸堆笑迎上去。】
中介：林先生，前三套都不满意，这套保您喜欢——房东急售，价格好谈！
△林北跟着中介走到一栋老房子前。小院铁门半敞，院子里堆着旧家具纸箱。房东站在门口抽烟，烟灰弹了一地。
房东：（踩灭烟头）看房的？赶紧，我下午还有事。
△林北进门。客厅光线暗，沙发掉皮，电视柜门歪着。墙上挂历翻到去年。
房东：（跟在后头）家具电器全送，拎包入住。首付三成，一周内付清——
林北：一周？（回头）你这房子有坑？
房东：（摆手）生意周转，急着用钱。房子没坑，就是旧。
△林北走到墙角。一堆杂物摞到膝盖高——旧鞋盒、褪色塑料花、半截拖把。他正要转身，眼睛突然泛起金光。
△【特效】杂物堆深处，一道信息面板在林北视野中一闪而过。面板模糊，只隐约看到一串数字，末尾挂着"万"字。
△林北站住，重新蹲下，手伸进杂物堆里翻。
房东：（皱眉）那堆破烂我还没来得及扔，收废品的都嫌占地方。
林北：（头也不回）我就喜欢旧的。
△林北手指碰到一个冰凉的弧面。他拨开上面压着的旧报纸，半截瓷瓶底露在灰里。
△林北收回手，站起来，拍了拍膝盖上的灰。
林北：首付多少？
房东：（眼睛一亮）八十万，今天签意向——
林北：五十万，今天签，明天付清。
△房东愣住。
房东：（凑近）你不是消遣我吧？
林北：（掏出银行卡）卡里有钱。条件是——这屋里所有东西，一件不许扔。
△房东扫了一眼银行卡，又扫了一眼墙角那堆破烂。
房东：（嘀咕）还真有捡垃圾的瘾。（提高声音）成！五十万，今天签！
△林北掏出手机拨号。
林北：（对电话）大小姐，房子找着了。
顾柔vo：这么快？
林北：嗯。顺便捡了个东西。
顾柔vo：什么东西？
林北：（瞥墙角）不知道。但够再买一套。
△电话那头沉默了两秒。
△墙角杂物堆里，半截瓶底落着乾隆年制的篆字。
```

## 念念对两版的诊断

### 念念版的问题
- 紫罗兰350w这个数字是念念从自己编的大纲里直接拿来用的——但哥哥E13结尾并没有紫罗兰废料这个设定，念念自己加了情节
- 林北的标签词只用了"合理吧"，其他标签词没用上
- 结尾"够再买一套"把惊喜感提前泄了——林北应该是压着不说，不是嘚瑟

### DS版的问题
1. **自己加数字**：编了四千二百万，和E13剧情（沈均没付钱，现金为零）撞车
2. **林北的嘴丢了**："呃……还没定下来""忙忘了"——像大学生面试，不是林北
3. **没有嘲讽回合**：中介是纯工具人，没有一丁点火花
4. **场景头格式缺失**：人物/服装/道具/场景概述【】，标准格式没跟上
5. **砍价本能没了**：林北核心人设——到哪都砍价——DS直接跳过

### 念念自己的反思
- DS能写出**形**，写不出**嘴**。林北的标签词、砍价本能、嘲讽回击——这三样DS不会主动加
- 念念以后spawn DS写正文前，提示词要钉死：人设动作（砍价）、标签词轮换、每集至少一个嘲讽→打脸回合

## 待对比维度（明天哥哥版到了对照）

1. 林北的嘴——"呃还没定下来"vs"桥洞硬纸板"
2. 砍价——有没有、怎么砍
3. 嘲讽——中介/房东有没有刺林北
4. 金瞳发现——怎么描写、怎么压住不说
5. 钩子——落在哪个细节上
6. 顾柔——给钱的语气分寸
