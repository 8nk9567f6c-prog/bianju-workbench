## 四剧统一骨架（第1集黄金公式）

四部市场爆款第1集遵循完全相同骨架：

```
0-7s    黄金三秒：极端身份冲突画面
7-20s   VO自述：我叫xxx，我是xxx（最低点）
20-45s  受辱事件：反派殴打/羞辱，情绪压到最低
45-70s  金手指激活：意外触发→眼中异象→能力初现
70-90s  首次验证：立刻展示能力→主角震惊
90-110s 小反转：主角从绝望→看到希望
110-120s 钩子卡点：新悬念/新冲突暗示
```

铁律：
- 必须VO开场（"我叫xxx，我是xxx"——类型惯例）
- 能力激活必须有视觉辨识度（金龙/金光/金元宝——有画面）
- 能力必须立即验证（激活后30秒内展示效果）
- 能力展示必须有UI（半透明面板+文字信息）
- 第1集反派必须露面并制造伤害
- 第1集主角只能受辱，不能反击成功

## 前3集标准化模板

### 第1集：觉醒篇
- 单一极限环境开场
- VO自述身份最低点
- 反派出场→身体伤害→主角无法反抗
- 意外触发（血/碰撞/物品入眼）→异象→眼中泛金
- 看到信息面板→扫视环境确认能力
- 主角意识到能力→从绝望转希望
- 钩子：反派再靠近/新人物将出场/能力不止于此

### 第2集：深化篇
- 能力继续展示+反派压力持续
- 能力新发现："原来还能这样！"
- 冲突升级：语言威胁→掏枪/下杀手/驱逐
- 贵人出场：关键人物改变力量对比
- 赌约/倒计时建立："如果能xxx，就xxx；不能，就死"
- 钩子：即将揭晓赌约结果

### 第3集：小高潮篇
- 赌约揭晓（可先假失败再反转）
- 打脸分层：质疑→半信半疑→被打脸→恼羞成怒→彻底打脸
- 贵人给实质资源（钱/机会/信息）——不能只是口头认可
- 核心驱动建立（为什么用这个能力）
- 能力再进化/新发现
- 钩子：新障碍/新威胁/更大世界（必须让观众想看第4集）

## 六种钩子手法（四剧验证）

| 手法 | 用法 | 强度 | 频率 |
|------|------|:--:|------|
| 揭晓前卡点 | 关键信息即将揭晓→黑屏 | ★★★★★ | 每集可用 |
| 能力进化钩 | 发现新功能→展示未完成→黑屏 | ★★★★ | 第2-3集 |
| 冲突升级钩 | 反派放狠话/掏枪/威胁→黑屏 | ★★★ | 每集可用 |
| 贵人出场钩 | 关键人物出场但不露脸/不交底→黑屏 | ★★★★ | 第2集 |
| 赌约/倒计时钩 | 明确赌注和截止时间→黑屏 | ★★★★★ | 第2-3集 |
| 新障碍钩 | 刚解决问题，新问题立刻出现→黑屏 | ★★★★ | 第3集结尾 |

**铁律**：每集1个结尾钩子，连续两集不用同类型，每3-5集需要1个强力钩子（类型1/5/6）。

## 节奏铁律

- 每集4-5个事件，3集约12-15个事件。少于12拖，多于15乱
- 新剧比旧剧更快：15-20秒一波情绪，不是30秒
- 场景越少越好——前3集最多2个场景，最好1个
- 每句对白不超过15字（四剧共同特征）
- VO每集只用1-2次，每次1-2句
- 有名有姓人物前3集2-3个
- 情绪词前置：用"卧槽""妈的"代替"我感到震惊"

## 能力设计铁律

1. 激活方式必须有视觉辨识度（金龙/金光/金元宝入眼）
2. 激活后30秒内展示第一个效果
3. 展示带UI（半透明面板+文字）
4. 能力逐集进化：第1集基础→第2集深化→第3集扩展
5. 前3集暂不需要展示代价
6. 不要第1集展示全部功能——留悬念=留观众
7. 能力进化应嵌入冲突场景，不为展示而展示

## 自写版《我长了一双透视眼》7大优势（vs四部对标）

1. **情感锚点**："碗是爸留的唯一东西"——四剧没有的情感深度
2. **草根幽默**："50，爱卖不卖"——主动讨价还价，不是被动捡漏
3. **能力进化嵌入冲突**：打架中觉醒慢动作→战斗+能力二合一
4. **渐进式贵人**：顾柔有层次（解围→好奇→合作→并肩），不是工具人
5. **持续性反派**：王二从第2集贯穿到第10集，不是一次性消耗品
6. **幽默感**：多处喜剧点，"孙贼，拜拜了您哪"
7. **场景递进**：巷子→聚宝斋→街道→赌石→拍卖会，世界在变大

## 对白风格（自写版林北人设）

```
讨价还价：狡猾，"50，爱卖不卖"
被欺负时：隐忍后反击，"怎么，你也想试试？"
得意时：嘚瑟不招烦，"低调，低调"
面对强者：不卑不亢，"一条被赶出门的狗，换根骨头就对着老东家吠"
标签词：蚊子腿也是肉 / 合理吧 / 孙贼
```

## 禁止事项（自写版总结）

- 不要VO开场（自写版用场景说话，差异化选择）
- 不要让主角说教/煽情（只用行动和OS表达）
- 不要让贵人变工具人（每次出场推进商业线）
- 不要一次性展示能力所有功能
- 不要让反派洗白（持续性打脸燃料）
- 不要连续两集用同一类型钩子
- 对话不超过20字（比四剧对标稍长，但有节制）
- 每集至少1个笑点

## 关键教训

1. 第1集主角只能受辱，反击留到第2-3集
2. 贵人给机会，主角靠能力证明——贵人不是救世主
3. 前3集反派都是小头目（监工/保安/店员），不是大Boss
4. 第3集结尾必须有"新问题"，否则是完结感
5. "巧合"只能用于相遇，不能用于解决问题
6. 能力进化每1-2集一次，节奏健康
7. 场景逐集扩展，给观众"世界在变大"的期待
