# 主编反馈记录

## 反馈 1：E11+ 偏离致富主线（2026-06-25）

**主编原话**：剧本应该是"致富剧"，但从 E12 之后偏离了财富积累主线，变成了鉴定对决、组织对抗、父亲身世之谜。

**问题诊断**：
- E13+ 沈均复仇线取代了赚钱线
- E14-E15 方鹤年/省城交易会变成了"跟大人物对决"
- E16+ 换地图到省城，完全脱离了江城草根致富的调性
- 陈九爷/父亲失踪等组织阴谋线喧宾夺主

**修正**：v3.0 纯致富主线。见 [[pure-getting-rich-formula]]。

## 反馈 2：一集一个事件太赶（2026-06-25）

**用户原话**："你没发现一件事情吗，你很多时候都是，一集梗概一件事，但是你看看对标账号，往往都是两三集才讲一件事，你得自己水起来啊。"

**问题诊断**：把"水"理解成了灌水，实际上"水"是戏剧扩展——讨价还价、旁观者反应、对手情绪渐变、环境描写。对标剧本 2-3 集讲一个事件。

**修正**：v3.0 详细版每集 7 拍，事件横跨 2-3 集。见 [[benchmark-episode-density]]。

## 反馈 3：完全抛弃旧大纲（2026-06-25）

**用户原话**："我们可以完全抛弃掉之前的大纲要求，只写主角不断鉴宝攒钱，不换地图也行，也不用让反派知道他的秘密，就慢慢攒钱就行。"

**含义**：不是修修补补，是方向性推翻。旧版所有的省城、陈九爷、方大师、方鹤年、组织阴谋、父亲身世——全部不要。

**修正**：v3.0 大纲。沈均 E11 退场。方鹤年/陈九爷/周师傅全部砍掉。全程江城。聚珍阁只做背景威胁。

## 教训

1. 致富剧的核心是"钱在画面中央"——每集必须有具体的财富数字变化。
2. 不要急着塞大事件和大反派——慢慢攒钱的节奏本身就是看点。
3. 对标剧本是最好的老师——不是抄内容，是抄结构（每集几拍、几场景、怎么结尾）。
