# 对标剧本每集信息密度规律

## 分析样本
- 都市：我有一双火眼金睛（王峰版，5集）
- ref_01_黄金疯涨（林凡版，3集）
- ref_02_觉醒透视眼后（林诺版，3集）

## 核心指标

| 指标 | 基准值 | v3.0 是否对标 |
|------|:------:|:------------:|
| 每集字数 | ~700 | ✅ |
| 每集拍数 | 7-9 | ✅ |
| 每集场景数 | 1-1.4 | ✅ |
| 非E1开篇 | 动作/对话直切 | ✅ |
| 结尾钩子 | 70%压力型 | ✅ |

## 五条铁律

1. **场景极简**：矿工类剧本三集锁死一个场景。古玩类每集最多换一次场景（去市场/回家）。不要频繁换场。

2. **一件事 2-3 集**：不是"一集一件事"。火眼金睛的"捡漏→变现"跨了E2-E4三集。每集只是这件事推进了 7-8 拍。

3. **700 字分配**：开篇~100字（建立空间/状态）→ 对话交锋~250字（讨价还价/冲突）→ 反转验证~200字（能力展示/爽点）→ 收尾钩子~150字（悬念台词，通常1-2行）。

4. **结尾 = 压力**：70%的集结尾是压力钩子。即使整集是"放"（主角赢了），结尾也必须埋一个新问题。火眼金睛E5结尾："三天500万"——这就是为什么观众点下一集。

5. **爽点在中段爆**：爽点不在结尾，在每集第3-5拍（中段）。结尾留给钩子。

## v3.0 的调整

旧版问题：一集一个完整事件 + 频繁换场景 + 大量"放"结尾。
v3.0 修正：2-3集一个事件 + 极简场景 + 集集中段爽点 + 结尾埋压。
