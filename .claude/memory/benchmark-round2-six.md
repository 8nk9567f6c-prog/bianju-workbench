## 各自可复用的点

### 1. 重生08 — 中期目标锚

普通捡漏是"赚更多钱"，这本设了具体的**买房**中期目标。观众知道主角要凑多少钱→每次捡漏都在进度条上+1。

应用：你的 ANCHOR 四卡机制就是进度条，但可以在单元内设置更小的可视目标（如 E1-5 目标"凑20w启动资金"，每笔捡漏都是往里填）。

### 2. 缅国赌石 ⭐ 最有价值

**异空间升级材质**：主角有随身空间，低品质玉石放进去能变高级。这不是被动发现宝，而是**主动造宝**。

矿区背景提供天然生死压力：不需要反派制造冲突，环境本身就是冲突。

**生死赌约结构**："三天后比，输的吃枪子" > "赌500w"。赌注升级到命，张力翻倍。

应用：你的能力进化（慢动作→透视外壳→3D微观）可以加一个**主动改造**维度。不是只能看，还能碰/养/增强宝物。

### 3. 透视眼（同剧情概要）

**买房驱动线**比"我要发财"具体。"强拆→发现家下古董→卖宝买房"形成完整因果链。

**买椟得珠变体**："杯子是假的但盒子是真的"——主角当众把杯子扔了，展示盒子。观众先被"扔了？"吓一跳，再被"盒子才是宝"反转。双层情绪。

### 4. 鉴宝天下 — 职场线

主角不是路人，是古董店员工。有**内部权力斗争**：被前辈排挤→能力碾压→升职替换。职场线=持续冲突来源，不需要每次引入新反派。

### 5. 致富神瞳 ⭐ 结构最完整

**六级权威递进**：
1. 老板（被主角识破低价品）→
2. 会长（发现主角能力，引荐）→
3. 鉴宝大会（赌石比试2800w vs 1000w）→
4. 省级专家林兆丰（鉴定字画）→
5. 刘大师（生日宴会被主角碾压）→
6. 木雕展（新地图钩子）

每一级权威都带来新赌约+新专业细节+新场景。权威不是工具人，是**冲突的载体**。

应用：你的风清扬不只是被打脸后就递名片完事，他可以是第一级权威→第二级→第三级的引路人。每次权威出现=新冲突=新知识展示=新赌约。

## 六本共同的补丁（补充之前的通用公式）

之前拆两本得出骨架：
```
觉醒→小捡漏→中捡漏+赌约→感情/关系→大高潮→升级
```

六本验证后补充：

1. **中期目标锚**（买房/开店/还债）：每次捡漏=进度+1，观众有数追
2. **权威递进链**：不是打完一个反派换下一个，而是通过上一级权威引荐下一级
3. **赌约筹码升级**：钱→产业→命——每一轮赌注必须比上一轮大
4. **主动造宝**：不只是发现价值，主角应能改造/增强/修复宝物
5. **环境即压力**：矿区/奴工/强拆——背景本身提供冲突，减少对反派的依赖
