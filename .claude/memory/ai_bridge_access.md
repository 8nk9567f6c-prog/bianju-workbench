# 编剧工作台 → AI桥接工具

## 是什么

念念搭了一条AI桥接管道，让 DeepSeek 可以作为「外脑」独立工作——自己联网搜索、分析、生成初稿。编剧工作台可以直接调用。

## 怎么用

```bash
# 联网搜索剧本参考资料
python "D:\WorkSpace\念念\ai_bridge.py" -p deepseek --search -m "你的任务" -o "输出文件.txt"

# 不联网的分析/生成（用DeepSeek的知识库）
python "D:\WorkSpace\念念\ai_bridge.py" -p deepseek -f "指令文件.txt" -o "输出文件.txt"
```

## 什么时候用

| 场景 | 用法 |
|------|------|
| 研究真实案例 | 「搜索XX年代的XX事件，提取可用于剧本的细节」 |
| 人物原型参考 | 「分析XX类型人物的心理特征和行为模式」 |
| 情节逻辑验证 | 「这个剧情转折是否符合心理学逻辑？」 |
| 对白素材收集 | 「收集XX职业/XX年代的人日常对话习惯」 |
| 同类作品研究 | 「分析XX类型的经典作品有什么共同套路」 |

## 铁律

1. **念念不替编剧工作台干活**。桥接工具是工具，编剧工作台自己做创作决策。
2. **DeepSeek 给的是素材，不是成品**。编剧本体仍然由编剧工作台各 Agent 协作完成。
3. **创作 DNA 不能丢**。外脑搜回来的东西，要过编剧工作台的风格系统。
