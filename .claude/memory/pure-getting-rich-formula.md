# 致富剧纯公式（v3.0 总结）

## 来源
主编拒绝 E11+ 后，用户给的新方向："完全抛弃掉之前的大纲要求，只写主角不断鉴宝攒钱，不换地图也行，也不用让反派知道他的秘密，就慢慢攒钱就行。"

## 核心公式

```
鉴宝 → 捡漏 → 变现 → 攒钱 → 下一个更大的捡漏
```

## 三个"不"

1. **不换地图**：全程一个城市。不是"主角变强→去更大的地图"，而是"世界不变，主角在同一个地方越做越大"。
2. **不让反派知道秘密**：反派可以存在，但他们的冲突来自"看不起主角"（身份碾压），不是来自"想抢金瞳"。金瞳秘密只有观众和主角知道。
3. **不急着搞大事件**：慢慢攒。每个事件 2-3 集。不需要每集一个高潮。

## 为什么有效

- **火眼金睛**就是纯致富公式：外卖员→捡漏→变现→重复。E1-E5都在同一个古玩市场，唯一压力是欠债。
- **黄金疯涨**也是：苦力→发现透视眼→不断挖金→体内空间。三集锁死在一个金矿场景。
- 观众追的不是"能不能打败大反派"，而是"下一集能赚多少钱"。
- "下一集能赚多少钱"本身就是一个永远不会失效的悬念钩子。

## 如何应用

写任何致富剧的开篇时，先问三个问题：
1. 这一集有没有具体的财富变化？
2. 冲突是来自"看不起主角"还是"想抢主角的超能力"？
3. 主角有没有换地图？如果换了——是不是真的必要？
