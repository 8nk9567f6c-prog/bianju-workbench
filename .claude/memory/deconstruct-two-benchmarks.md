## 剧一：《你真有透视眼，来捡漏致富赚大钱》（男频）

22个场景节点，约10集体量。外卖员捉奸坠楼→玉佩化金龙入眼→透视鉴宝→一路赌约打脸。

### 核心技法

**1. 赌约引擎。** 全篇每一次冲突都转化为赌约，没有例外：
- 反派说石头不值钱 → 赌约（10w为界，2倍价买下）
- 王虎说赝品是真品 → 赌约（500w，输了废李伟）
- 王海燕嘲讽没钱 → 赌约（比宝物价值，输了跪下+10倍买下）

赌约 = 微短剧发动机。观众知道主角会赢，但不知道"怎么赢"——等待开奖即爽点蓄力。

**2. 打脸三件套。** 每个冲突回合三步：嘲讽（反派嚣张）→ 主角低调接招 → 当众翻盘（群众/专家第三方认证）。从不自己喊赢，靠第三方反应让观众爽。

**3. 钩子是方向预告，不是谜团。** 22个节点19个标"钩子："——不是"主角不知道接下来会发生什么"，而是"钩子：他们一起去潘家园"。观众知道下一页有什么，所以翻页。期待拉着，不是谜团吊着。

**4. 专业知识作武器。** 鉴伪用具体细节："观"字最后一笔笔锋轻灵上挑→宋人书写习惯；弧度圆润流畅→现代工艺。不是"一眼看出假的"，给观众一个能记住的理由。双重爽：主角赢了+我懂了。

**5. 感情线三层递进。** 婉儿：看中能力（邀约）→ 产生好感（下厨+上床）→ 给资源（邀请函）。每次推进有具体动作，不靠对白说"我喜欢你"。

**6. 反派逐级升级。** 前男友→王虎（富二代）→王海燕（集团总裁），不能同级反复。

### 可复用模板

```
捡漏打脸场景模板（9步）：
1. 主角进入新场所（摊子/店铺/鉴宝会）
2. 发现宝物（金瞳识别）
3. 反派出现+嘲讽（身份压制/金钱压制）
4. 赌约成立（具体金额/条件，不能模糊）
5. 主角低调陈词，反派加码
6. 当众验证（开石/专家鉴定/手机亮证据）
7. 群众反应逆转（路人/大师/权威第三方）
8. 反派付钱/被打脸
9. 钩子指向下一场景
```

## 剧二：《我有一双财神眼》（女频）

30个场景节点。打工族李诗涵→财神赐"财神眼"→捡漏：花瓶(500→20w)→字画(9000→1600w)→古书+邮票(5000→1.2亿)→犀角杯(1000→2.2亿)→金店搭赠。升级：满1亿→解锁隐藏价值；满10亿→再升级。

### 女频独有技法

**1. 升级数值化。** 财神眼满1亿升级，满10亿再升级。给观众进度条——"还差多少到下一级"，手游经验条逻辑，观众会一直追。

**2. 添头捡漏（买椟得珠）。** 买正品→要赠品→服务员说画贵送玉→女主说送地上瓶子→买够10w送→买金表带走古董。让对方主动送宝，爽点来自信息差完全倒向主角。

**3. 闺蜜代替感情线。** 用闺蜜乐乐（陪逛街、父亲生日、引荐入圈）替代男女感情线。关系链=情节链：闺蜜父亲生日→捡漏古书→认识修复大师→发现邮票→拍卖1.2亿。

**4. 嵌套藏宝（俄罗斯套娃）。** 古书里翻出邮票夹层→大清邮票→再翻页又一套珍品邮票。一层宝物藏着更贵宝物，观众以为爽完又来更大，双重高潮。

**5. 五种买入方式全不同。**
1. 直接砍价（花瓶：500成交）
2. 迂回问价（甜品店画：先问别的再绕回真品）
3. 朋友帮腔（古书摊：闺蜜发火+托唱红脸）
4. 赌约收割（犀角杯：赌2亿vs100w→切开验证）
5. 买椟得珠（金店：买金搭赠天价花瓶）

每种买入方式不同，观众不腻。

## 两本合并：微短剧捡漏类通用公式

```
觉醒（1-2节）
  ↓
小捡漏×1-2次 → 建立能力可信度（买入方式A）
  ↓
中捡漏+赌约打脸 → 建立冲突模式（买入方式B）
  ↓
感情/关系线穿插 → 缓冲+资源入口
  ↓
大高潮×1 → 多方势力+专家认证+巨额收入（买入方式C）
  ↓
能力升级/新目标 → 续力下一轮
```

骨架完全一致，填充物分化：
- **男频填充**：赌约+反派+女人（弱者逆袭逻辑）
- **女频填充**：闺蜜+砍价+升级进度条（精明致富逻辑）

## 两本的共同铁律

1. 每次捡漏必须换品种（石→瓷器→字画→古书→邮票→犀角→金器），不重复
2. 每次捡漏必须换买入方式，不重复
3. 每次打脸必须第三方认证（群众/专家/权威），主角不自夸
4. 反派层级逐级升级，不反复打同级
5. 每个场景节点结尾必须有钩子——明确方向预告，不是谜团
6. 专业细节必须有至少一个"观众能记住的具体理由"
7. 感情/关系线每次推进要有具体动作（上床/给邀请函/引荐入圈），不靠对白
