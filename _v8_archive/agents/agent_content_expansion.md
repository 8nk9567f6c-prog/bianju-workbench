## v5.0 剧本格式

```
第x集
001 日/夜 内/外 场景名
人物：A B（≤4人）
服装：- A：描述 - B：描述
道具：- 道具1 - 道具2
【角色站位+动作关系+环境+氛围】
△ 动作描述（以△开头，只写可拍内容）
A：对白（口语短句，不加△，≤15字）
---
（第x集完）
```

### 格式要点
- 场景编号三位数（001/002/003...）
- 人物≤4人同场
- 道具必须预罗列，正文不出现未罗列道具
- 【】内写站位+动作关系+环境+氛围
- △ 写动作，不加△写对白
- `---` 分隔场景
- 每集结尾 `（第x集完）`

---

## 分场建议

- 每集通常2-3场，少即是多
- 场景越少越好——换场景=剧组重新布光搭景=烧钱
- 能在一个场景拍完80%的戏，就不要写第二个场景

## 技巧提示

**对白怎么写**：
- 每句话说出来，默念一遍——不顺口就改
- "你给我跪下"→"跪下"
- "我希望你能为自己的行为付出代价"→"你会后悔的"

**爽点怎么埋**：
- 不要集中在一处，散开在开场/中间/结尾
- 最强的爽点放结尾前10秒

**弹簧怎么端**：
- 压弹簧：主角被压制/误会加深/反派嚣张/危机逼近 → 观众憋屈
- 放弹簧：主角反转/打脸/真相大白/危机解除 → 观众爽
- 结尾最后一句决定压还是放

**钩子怎么收**：
- 上一集结尾埋的 → 这一集开头3秒内收
- 这一集结尾埋的 → 下一集开头收
- 超过2集不收 = 观众忘了 = 白埋
