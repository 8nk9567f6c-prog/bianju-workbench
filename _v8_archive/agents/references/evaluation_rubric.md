# 剧本质量评估框架（红果评级对标版）

> 来源：哥哥提供的红果平台评估维度
> 用途：6维评估框架，替代旧的8基因评分体系。每个维度按优/良/中/差四级评定，不编造权重，不拍脑袋打分。
> 原则：每条评价必须有原文依据，不凭感觉。

---

## 一、主角塑造（权重：待定）

### 1.1 人设清晰度

| 等级 | 标准 |
|------|------|
| 优 | 一句话能说清主角是谁、他要什么、他有什么特别。出场30秒内形成记忆点。 |
| 良 | 主角身份明确，但记忆点不够强。需要看到第2集才能记住他。 |
| 中 | 主角身份模糊，和其他同类剧主角区分不开。 |
| 差 | 看了3集不知道主角是谁、想干嘛。 |

**核心问题**：老张看完这集，能用一句话跟朋友介绍主角吗？

### 1.2 命运牵引力

| 等级 | 标准 |
|------|------|
| 优 | 老张会担心主角下一步会不会出事、能不能翻盘。每集结尾有"不得不看"的牵动力。 |
| 良 | 老张觉得主角挺有意思，想知道他后来怎样，但不会焦虑。 |
| 中 | 老张看完了，但划走也不觉得可惜。 |
| 差 | 主角的处境引不起任何情绪。老张甚至希望反派赢。 |

**核心问题**：老张关掉手机后，会惦记主角吗？

## 二、人物塑造（权重：待定）

### 2.1 人物关系

| 等级 | 标准 |
|------|------|
| 优 | 主角之间的关系有强烈的矛盾戏剧张力（对峙/依赖/误会/反差）。观众会磕CP或盼看下一场对手戏。 |
| 良 | 人物关系清楚，有互动，但张力不够强。 |
| 中 | 人物有互动但关系平淡，没有记忆点。 |
| 差 | 人物之间没有有效互动，关系可有可无。 |

**核心问题**：如果把两个主角分开拍，观众还会想看吗？

### 2.2 反派/配角

| 等级 | 标准 |
|------|------|
| 优 | 反派有存在感，不是工具人。观众记得住他干了什么坏事。群像配角各司其职，不重复。 |
| 良 | 反派清晰，但单薄。配角够用但不突出。 |
| 中 | 反派脸谱化，换个名字也不影响剧情。配角可有可无。 |
| 差 | 没有有效反派，或者反派出现观众也不知道他是谁。 |

**核心问题**：老张能说出反派的名字和干过的最坏一件事吗？

## 三、情绪价值（权重：待定）

| 等级 | 标准 |
|------|------|
| 优 | 有能让人代入/共情/畅快的核心情绪场。老张会看哭、看笑、看握拳。 |
| 良 | 情绪到位，能跟着走，但不出格。 |
| 中 | 情绪平，观众知道"这里该爽"但没真的爽到。 |
| 差 | 情绪不在对的位置上。该爽时憋屈，该虐时平淡。 |

**核心问题**：老张看这集，有没有某个瞬间真的"入戏"了？

---

## 四、故事情节（权重：待定）

### 4.1 主线与追看

| 等级 | 标准 |
|------|------|
| 优 | 每10集有一个阶段性看点（报仇/暴富/身份揭穿/关系突破）。全集看完不觉得中间在凑。 |
| 良 | 主线清晰，但某些阶段看点不够突出。 |
| 中 | 主线存在感弱，观众看一半忘了主角的目标是什么。 |
| 差 | 没有主线，或者主线不支撑100集。 |

**核心问题**：第10集、第30集、第50集、第80集——老张每个节点都有理由点开下一集吗？

### 4.2 节奏与反转

| 等级 | 标准 |
|------|------|
| 优 | 节奏快到不用拖进度条。反转在"大方向猜得到"的基础上，小细节"没想到"。 |
| 良 | 节奏正常，不拖。反转有一个算一个，但都在意料之内。 |
| 中 | 节奏明显慢了——某场戏删掉也不影响剧情。 |
| 差 | 拖沓、注水。反转烂俗或生硬。 |

**核心问题**：老张有没有某个瞬间划了一下进度条？

### 4.3 逻辑与合理性

| 等级 | 标准 |
|------|------|
| 优 | 人物行为符合自己的设定。情节推进有因果链。观众不会出戏。 |
| 良 | 没大毛病，但有些地方需要观众自己脑补理由。 |
| 中 | 有明显逻辑漏洞，但因为爽感强观众选择性忽略。 |
| 差 | 逻辑断裂到观众出戏，爽感盖不住。 |

**核心问题**：老张看到某个情节会说"等等，他为什么这么做？"吗？

## 五、差异化（权重：待定）

| 等级 | 标准 |
|------|------|
| 优 | 在同类型赛道里，有明显让人记住的不同点。从人设/能力/结构/手法至少一个维度有新意。 |
| 良 | 不是完全复制，但核心框架和同类剧一致。 |
| 中 | 换皮——把另一部剧的人物名字改了就是这部。 |
| 差 | 完全复制对标剧，没有自己的东西。 |

**核心问题**：如果老张同时看三部同类型剧，他能分出这部跟另外两部有什么不一样吗？

---

## 六、当下性（权重：待定）

| 等级 | 标准 |
|------|------|
| 优 | 踩中当前主流用户的情绪刚需（反权威/付出有回报/好人赢/穷人也能翻身）。价值观不落后。 |
| 良 | 不犯错。符合主流价值观但不出彩。 |
| 中 | 价值观有点老了（比如"男人就该怎样怎样"的旧模板），但不至于被骂。 |
| 差 | 价值观与当下主流情绪冲突，或者完全过时。 |

**核心问题**：老张看完这部剧，会不会觉得"说出了我心里的话"？

---

## 使用方式

1. 评自己剧本时：逐维度对照原文，标注等级 + 一句依据
2. 拆对标剧时：用同一框架，方便自己和竞品同维度对比
3. 不编造分数——只给等级，等级背后必须有原文证据
