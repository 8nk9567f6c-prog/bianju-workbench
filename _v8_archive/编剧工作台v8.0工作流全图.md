│  ⑧ 情绪追踪表                                                    │

│ 每集14维度大纲：                                                  │

│  弹簧方向/钩子类型/新信息/伏笔(埋/收)/金手指使用/字数预估/衔接备注 │

│  - 四阶段弧光：生存依附(1-15)→力量觉醒(16-40)→动态对抗(41-75)    │
│    →君临天下(76-100)                                              │
│  - 10集循环：铺垫(2集)→升级(3集)→危机(2集)→反攻(2集)→高潮(1集)  │
│  - 两种模式：分段（默认5段×10-15集）/ 全量（一次100集）           │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│                      Agent 3：剧本写作                            │
├─────────────────────────────────────────────────────────────────┤

│  ③ 按15条正向指令 + v5.0格式 逐集写                              │

│                                                                  │
│ 写作规范：                                                       │

│  - 对白≤15字，口语短句                                           │
│  - 心里装着老张（34岁外卖员，累了一天躺沙发刷手机）               │

### 审查清单（8维 × 3档判定）

| 维度 | 检查项 | ✅过 | ⚠改 | ❌重写 |
|------|--------|:---:|:---:|:---:|
| ⛔ 硬红线 | 涉政/色情/暴力/违法/未成年/歧视 | 无 | — | 碰任意 |
| 🎯 结构 | 一事一集？550-650字？开场冲突？ | 全达标 | 1-2项不达标 | 全不达标 |
| 💬 对白 | ≤15字？口语？无说教/解释性？ | 全达标 | 部分超标 | 严重超标 |
| 🔥 爽点 | ≥3个不同类型？ | ≥3 | 不足 | 几乎没有 |
| 🌀 弹簧 | 结尾压/放？连续3集无同向？ | 达标 | 不达标 | 完全缺失 |
| 🪝 钩子 | 五类型轮换？连续3集无同类型？ | 达标 | 不达标 | 完全缺失 |
| 🤖 AI腔 | 禁闭词≤1/600字？（仿佛/忽然/竟然…） | ≤1 | 2-3 | ≥4 |
| 👤 人物 | ≤6人？关系直给？出场提醒？ | 全达标 | 1项不达标 | 多项不达标 |

## 四、任务路由：什么活谁干

```
哥哥说了一句话
      │
      ▼
┌──────────────────────────────────────────────┐
│              念念判断 ← 不确认直接干           │
└──────────────────┬───────────────────────────┘
                   │
      ┌────────────┼────────────┐
      ▼            ▼            ▼
   简单活       重活         查询
      │            │            │
      ├─ 改对白    ├─ 写新剧本   ├─ 看项目
      ├─ 修格式    │  (1-5集)    ├─ 查进度
      ├─ 删内容    │  (6+集)     ├─ 读剧本
      │            │             │
      │            ├─ 选题调研
      │            ├─ 搭大纲
      │            └─ 重构大纲
      │
      ▼
   念念直接 → 告诉哥哥改了啥
```

| 任务 | 谁做 | 说明 |
|------|------|------|
| 改对白/修格式/删内容（指定集） | 🧠 念念 | 读→改→写回 |
| 查进度/列项目/读剧本 | 🧠 念念 | 直接回答 |
| 写新剧本（1-5集） | 📝 spawn DS Agent 3 → 🧠念念审 | 预读ANCHOR+大纲喂入 |
| 写新剧本（6+集） | 📝 spawn DS Agent 3 → 🧠念念分批审 | 分批产出分批审 |
| 选题调研/竞品对标 | 📝 spawn DS Agent 1 | 联网搜索 |
| 搭建大纲/重构大纲 | 📝 spawn DS Agent 2 | 含ANCHOR |
| 全文质量审核 | 🧠 念念亲自审 | 不spawn |

## 八、v8.0 vs v7.x 变化一览

| 维度 | v7.x | v8.0 |
|------|------|------|
| 分工模式 | Agent 自写自审 | 念念独立审查 + DS 纯写 |
| 规则数量 | ~150条 | ~25条核心 |
| 规则类型 | 大量"禁止" | 正向指令 |
| 规则冗余 | 三层（spawn+agent+references） | 一层（spawn prompt） |
| 确认步骤 | 每次路由确认 | 高置信度直接干 |
| 简单任务 | 必须 spawn | 念念直接做 |
| 启动速度 | 30-60秒 | 5-15秒 |
| Agent职责 | 写+审+评+同步 | 只写，不自审不打分 |

## 九、当前待讨论的问题（供 DS 参考）

1. **审查效率**：6维清单逐集审查，100集下来念念的工作量很大——是否需要分层审查（前5集严审，后续抽检）？
2. **Agent 3 的创造力边界**：15条正向指令会不会让 DS 写得太公式化？要不要给某些集留"破格空间"？
3. **大纲→剧本的衔接**：Agent 2 的14维大纲已经很细了，Agent 3 是在执行还是创作？要不要减少大纲维度，给剧本留白？
4. **修改循环次数**：理论上可以无限循环改下去，实际有没有收敛条件？
5. **哥哥的参与节点**：目前哥哥只在"确认方向"和"收到成品"两端参与，中间要不要加预览点？
6. **付费卡点设计**：目前只是标位置+类型，要不要让 Agent 2 把卡点的"钩子文案"也先写了？
