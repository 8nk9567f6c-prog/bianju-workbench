# 通用铁律 v2.0

> 便携提示词包的固定层。三个提示词共享的铁律从这里提取。
> 对标吸收后如有更新，在此文件标注版本号和变更。
>
> **v2.0 重要变更**：冲突铁律 + 人物声音锁 + 事件意外性 已从通用铁律提取并嵌入三份提示词的【创作引擎】板块。提示词不再只是格式说明书——它们是自包含的创作引擎。

---

## 一、格式铁律（不可变）

每集格式：

```
第X集
编号 日/夜 内/外 场景名
人物：A，B，C
服装：A（描述），B（描述）
道具：X，Y，Z
【一句话场景概述】
△动作描写
角色A：对白
角色Aos：内心独白
△【特效】特效描述
---
（第X集完）
```

规则：
- 动作必须 △ 开头，△后面只写摄像机能拍到的身体动作
- 对白用 角色名：对白。可标注情绪：角色名（咬牙）：对白
- 内心独白用 角色名os：
- 特效用 △【特效】
- 每集头部必列人物、服装、道具。人物≤4人同场
- 场景编号三位数（001/002…）
- --- 分隔场景
- 续场（下一集继续同一场景）不重新写场景头

---

## 二、字数铁律（不可变）

每集 450-600 字。硬性红线，不可违背。

---

## 三、禁词清单（不可变）

以下词绝不在剧本中出现：

仿佛 / 忽然 / 竟然 / 原来如此 / 突然之间 / 殊不知 / 却不知 / 只见 / 就这样 / 其实 / 原来 / 终于

替代方式：动作直接写，不加修饰词。出现 ≥2 次 = 退稿。

---

## 四、结构铁律

- 每集 2-3 场，开场 1-3 句必须进冲突，不准铺垫
- 每集只讲一件事
- 对白每句 8-15 字为主（情绪爆发可到 25 字），口语短句，可用脏话（tm的/老子/孙贼/臭要饭的/妈的）
- 禁用无意义寒暄

---

## 五、爽点铁律

- 每集 ≥3 个爽点
- 类型池：捡漏 / 暴富 / 打脸 / 能力升级 / 身份反差
- 散开布置，最强爽点放结尾前 10 秒

---

## 六、钩子铁律

- 每集结尾必须硬钩子
- 类型池：新发现 / 狠话 / 新危机 / 巨额财富 / 能力升级 / 新人物登场
- 连续 3 集不得重复同类型
- 钩子要有方向预告（"接下来要发生什么"），不是纯谜团

---

## 七、弹簧铁律

- 压弹簧：主角被压制 / 误会加深 / 反派嚣张 / 危机逼近
- 放弹簧：主角反转 / 打脸 / 真相大白 / 危机解除
- 连续 3 集不得同向

---

## 八、人物红线（所有项目通用）

1. 人物不可违背自身人设——写每句对白/动作前先问：这人是谁？他会这么说/做吗？
2. △动作层 = 纯动作——摄像机拍不到的删。禁形容词、禁议论、禁心理描写
3. 无主叙述禁止——剧本里每句话必须有归属（△动作 / 对白 / OS / 【场景概述】）

---

## 九、叙事引擎（通用场景骨架）

每集捡漏/打脸场景默认遵循九步骨架：

进入新场景 → 金瞳发现异常 → 被质疑（反派/路人/权威）→ 赌约/挑战建立 → 低调验证（不说服，用结果说话）→ 当众翻盘 → 第三方权威认证 → 反派被打脸 → 获利 → 钩子

换场景不换骨架。每一轮捡漏的品类、买入方式、对手层级必须比上一轮递进。

---

## 十、事件升级曲线

事件从小到大，波浪式上升：

大事件 → 缓冲集（生活场景/人物关系）→ 更大事件 → 缓冲集 → 最大事件

- 金额递增：10w → 100w → 1000w → 亿级
- 对手升级：路人/摊主 → 小反派 → 家族/行业反派 → 国际对手
- 每次大高潮后插入 1-2 集缓冲（情感线/生活兑现/能力消化）

---

## 十一、角色驱动原则

事件从主角出发，而非事件推着主角走。

- ✅ 正确：林北金瞳看到 X → 林北决定做 Y → 事件展开
- ❌ 错误：发生了一个拍卖会 → 林北被动参加

写大纲时先问：**主角在这个阶段会主动做什么？** 再设计事件。

---

## 十二、赌约引擎（12部剧100%验证）

每一次正面冲突必须转化为赌约。赌约必须有具体金额/条件，不能模糊。赌注每轮升级：钱 → 产业 → 命。观众等开奖 = 爽点蓄力。

---

## 十三、买入方式轮换池（12部剧100%验证）

每次捡漏从以下5种中选一种，相邻两次不重复：
1. 直接砍价
2. 迂回问价（先问别的再绕回真品）
3. 赌约收割（"如果是真品，你出双倍"）
4. 添头捡漏（买正品要赠品，赠品才是宝）
5. 信息差碾压（摊主不识货，主角识货）

---

## 十四、专业知识鉴伪（12部剧100%验证）

每次打脸至少给一个观众能记住的具体理由。不是"一眼看出是假的"，而是"这个观字最后一笔笔锋轻灵上挑，是宋人书写习惯，现代工艺做不出来"。

---

## 十五、权威递进链（12部剧验证）

打脸认证方逐级升级：路人→摊主→掌柜→会长→省级专家→国家级。通过上级权威引荐下级，不是随机换反派。

---

## 十二、水字数自查

每句对白/动作/场景问：推进剧情？展示人物？制造冲突？三者都不沾 → 删。

典型水字数（禁止）：
- 无信息位移：△林北走到窗边，看了看窗外，然后又走回桌边坐下。
- 寒暄对白：A：今天天气真热啊。B：是啊，秋老虎。

### 已吸收的核心模式：

**第1集黄金公式（精确到秒）**：
0-7秒身份冲突画面 → 7-20秒VO自述最低点 → 20-45秒受辱事件 → 45-70秒金手指激活 → 70-90秒首次验证 → 90-110秒小反转 → 110-120秒钩子卡点

**九步捡漏打脸模板**：
进入 → 发现 → 反派嘲讽 → 赌约成立 → 低调陈词 → 当众验证 → 群众反转 → 反派付钱 → 钩子

**能力进化铁律**：
前3集逐步激活 → 30秒内验证 → UI展示 → 逐集进化。第3集前不展示代价。不一次性展示全部功能。

**节奏铁律**：
每集4-5个事件，场景越少越好，对白不超过15字（对标四剧共同特征），情绪词前置。
