## 一、12部剧交叉验证：真正通用的叙事引擎

### 赌约引擎（12/12 剧验证，100%）

**每一次冲突都转化为赌约，没有例外。**

- 反派说石头不值钱 → 赌约（10w为界，2倍价买下）
- 反派说赝品是真品 → 赌约（500w，输了废人）
- 反派质疑能力 → 赌约（比宝物价值，输了跪下+10倍买下）

赌约 = 短剧发动机。观众知道主角会赢，但不知道"怎么赢"——等待开奖即爽点蓄力。

**铁律：赌约必须有具体金额/条件，不能模糊。"如果你赢了就xxx"→不，"如果你赢了，我出500w买下，如果输了，你跪下道歉。"**

### 九步打脸骨架（12/12 剧验证，100%）

```
进入新场景 → 金瞳发现异常 → 被质疑（反派/路人/权威）
→ 赌约/挑战建立 → 主角低调接招（不说服，用结果说话）
→ 当众验证（切石/专家鉴定/手机亮证据）
→ 第三方权威认证（群众/专家/会长）
→ 反派被打脸（付钱/下跪/离开）
→ 主角获利 → 钩子指向下一场景
```

男频和女频完全一致。换场景不换骨架。

### 第三方认证（12/12 剧验证，100%）

**主角从不自夸。** 每次打脸必须有围观者/对手/权威来认证。认证方每次要比上一次更权威：路人→摊主→掌柜→会长→省级专家→国家级。

### 反派逐级升级（12/12 剧验证，100%）

路人/摊主 → 小反派（掌柜/富二代）→ 中反派（家族/集团）→ 大反派（行业/国际）。不能同级反复。

### 捡漏品种不重复（12/12 剧验证，100%）

石→瓷器→字画→古书→邮票→犀角→金器→玉器→青铜→木器。每次不同品类、不同买入方式。

### 钩子=方向预告（12/12 剧验证，100%）

不是"主角不知道接下来会发生什么"（谜团）。是"他们一起去潘家园"（方向预告）。观众知道下一页有什么，所以翻页。期待拉着，不是谜团吊着。

## 二、事件升级体系（12部剧共同规律）

### 波浪式升级曲线

```
大事件 → 缓冲集（生活场景/情感线/能力消化）
  → 更大事件 → 缓冲集
    → 最大事件
```

### 三维同步升级

| 维度 | 初期(1-10) | 中期(11-50) | 后期(51-100) |
|------|-----------|------------|-------------|
| 金额 | 万→百万 | 百万→亿 | 亿→十亿 |
| 对手 | 路人/摊主 | 家族/集团 | 行业/国际 |
| 场景 | 古玩街/店铺 | 拍卖会/交流会 | 国际拍行/矿区 |
| 赌注 | 钱 | 产业 | 命 |

### 大高潮后必须缓冲

每次大对决（拍卖会/赌石大赛）结束后，插入1-2集缓冲：买房/带父母住新家/关系线推进/能力消化。不给缓冲 = 观众累。

## 四、可直接写入提示词的浓缩指令

### 赌约铁律
每次冲突必须变成赌约。赌约有具体金额/条件。赌注每轮升级（钱→产业→命）。观众等开奖即爽点蓄力。

### 买入方式轮换池
每次捡漏从以下池中选一种，相邻两次不重复：
1. 直接砍价（"50，爱卖不卖"）
2. 迂回问价（先问别的再绕回真品）
3. 赌约收割（"如果是真品，你出双倍"）
4. 添头捡漏（买正品要赠品，赠品才是宝）
5. 信息差碾压（摊主不识货，主角识货）

### 专业知识鉴伪
每次打脸至少给一个观众能记住的具体理由。不是"一眼看出是假的"，而是"这个'观'字最后一笔笔锋轻灵上挑，是宋人书写习惯，现代工艺做不出来"。

### 升级数值化
给能力设置明确的"进度条"——透视眼每吸收N件古董的金光解锁新功能。观众知道"还差多少到下一级"就会一直追。

### 权威递进链
不是打完一个反派换下一个。而是通过上一级权威引荐/引出下一级：
风清扬（市级）→ 省城私拍会（省级）→ 全国鉴宝大会（国家级）→ 国际拍行（国际级）
