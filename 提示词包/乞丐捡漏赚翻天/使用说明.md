# 乞丐捡漏赚翻天 · 提示词使用说明

> 三份独立提示词，大纲/正文/审核各管各的。底部自由粘贴，不依赖上下文。

---

## 模型选择（重要）

| 任务 | 文件 | 模型 | 深度思考 | 原因 |
|------|------|------|---------|------|
| 大纲生成 | `大纲生成.txt` | R1 | ✅ 开 | 因果链推演是R1强项 |
| 正文生成 | `正文生成.txt` | V3 | ❌ 关 | 格式合规+文笔输出是V3强项 |
| 审核 | `审核.txt` | R1 | ✅ 开 | 逻辑一致性检查是R1强项 |

---

## 日常流程

### 写大纲（每模块一次）
```
1. 打开 大纲生成.txt → 全选复制
2. DS网页版 → 新建对话 → 深度思考 ON → 粘贴
3. 在底部粘贴区填入：目标集数、当前进度、人物设定
4. 发送
```

### 写正文（每集一次）
```
1. 打开 正文生成.txt → 全选复制
2. DS网页版 → 新建对话 → 深度思考 OFF → 粘贴
3. 在底部粘贴区填入：本集大纲、前情提要（最近3集各一句话）、特殊要求
4. 发送
```

### 审核（写完即审）
```
1. 打开 审核.txt → 全选复制
2. DS网页版 → 新建对话 → 深度思考 ON → 粘贴
3. 在底部粘贴区粘贴剧本正文
4. 发送
```

### 返工
```
同一DS对话内 → 直接输入"改：[具体问题]" → 不重贴提示词
```

## 粘贴区填空模板

### 正文生成粘贴区（复制填写）
```
本集大纲：[一句话核心事件+钩子]
前情提要：
E[X]：[一句话事件+钩子]
E[Y]：[同上]
E[Z]：[同上]
人物在場：[本集出场人物+特殊状态]
伏笔回收：[本集需回收的伏笔]
```

### 大纲生成粘贴区（复制填写）
```
目标：生成第[X]集到第[Y]集大纲
当前进度：已完成[X]集，上集结尾：[简述]
人物：林北（乞丐逆袭，透视眼），顾柔（聚宝斋大小姐），赵四爷（古玩城地头蛇），沈均（已失势）
伏笔总账：[待回收列表]
最近5集钩子：[EXX:类型, ...]
弹簧状态：上集[压/放]，已连续[X]集同向
```
