## 二、叙事引擎（可复用模板）

### 单事件九步打脸法（与林北项目一致，验证了通用性）

```
进入新场景 → 金瞳发现异常 → 被质疑（反派/路人/权威）
→ 赌约/挑战建立 → 低调验证（不说服，用结果说话）
→ 当众翻盘 → 第三方权威认证 → 反派被打脸 → 获利 → 钩子
```

**佛瞳神眼每段都严格遵循此模板**：赌石店、拍卖会、古玩街、售楼处、翡翠交流会——换场景不换骨架。

### 女频特有技法（可吸收到林北项目）

1. **救猫咪时刻**：柳烟扶起被推倒的林萧 → 奠定好人属性。林北项目已有类似（顾柔给林北机会）
2. **情感线交织**：前男友+前闺蜜持续出现 → 不只是一次性反派，是持续性情感反派。林北项目可以借鉴：让王二不只是功能性反派，增加个人恩怨层次
3. **帮助他人支线**：医院驱邪扇 → 展示主角能力不只用来赚钱，也用来帮人。增加主角的"值得被喜欢"感
4. **生活兑现场景**：买房+带父母住新家 → 观众看到"能力带来的实际生活改变"，增强代入感

## 三、人物驱动 vs 事件驱动

### 佛瞳神眼的做法（值得学习）

每个事件链的起点都是**林萧主动使用金瞳**：
- 她看到垃圾桶有金光 → 她去捡
- 她看到唐舒身上有邪气 → 她主动说
- 她看到画是赝品 → 她拦下柳烟
- 她看到三个鸮尊都是赝品 → 她主动提出
- 她看到废料区有真品 → 她去拿

**不是"事件找到林萧"，而是"林萧的眼睛找到了事件"。**

### 这对林北项目的启示

林北同样应该是：
- 林北的眼睛看到了什么 → 林北决定怎么做 → 事件展开
- 不是：发生了一个拍卖会 → 林北被动参加
