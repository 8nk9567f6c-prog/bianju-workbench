# 使用说明

## 这是什么

三份可以**直接复制粘贴**到 DeepSeek / 豆包 / 任何 AI 对话窗口的提示词。每一份都是自包含的——粘贴进去，AI 就知道要干什么、怎么干、不能干什么。

## 三个提示词怎么用

| 你要干什么 | 复制这个 | 粘到新窗口 |
|-----------|---------|-----------|
| 写剧本正文 | `正文生成.txt` | DeepSeek / 豆包 |
| 规划后续大纲 | `大纲生成.txt` | DeepSeek / 豆包 |
| 优化已有剧本语言 | `语言优化.txt` | DeepSeek / 豆包 |

> **重要**：每次开**新对话窗口**。不要在同一个窗口里切换任务——会污染上下文，触发幻觉。

## 每天的工作流程

```
早上到公司：
  1. 打开提示词包文件夹
  2. 看今天要干什么 → 复制对应的 .txt 全文
  3. 打开 DeepSeek / 豆包 → 新对话 → 粘贴 → 发送
  4. AI 开始干活

下班回家：
  5. 打开 Claude Code → 把今天 AI 的输出粘贴给念念
  6. 念念审查 + 更新项目状态 + 刷新提示词
  7. 第二天到公司 → 拿最新版提示词继续
```

## 喂爆款剧本给念念

如果你看到好的爆款短剧，把以下任一种发给念念：
- **正文段落**（念念会提取格式和语言风格）
- **"事件+钩子"梗概**（念念会提取结构模式）

念念会分析、吸收，然后更新通用铁律和提示词，下次你拿到的提示词就自动带上了新学的招。

## 提示词更新机制

三份 .txt 文件不是一成不变的——念念会根据以下情况刷新它们：

| 触发条件 | 念念做什么 |
|---------|-----------|
| 你汇报了新进度（写了新集数） | 更新"当前状态"和"本次任务"部分 |
| 你喂了爆款剧本 | 分析吸收 → 更新通用铁律 → 刷新固定层 |
| 你反馈了问题（格式漂移/人设偏移/字数不对） | 调整对应规则 → 刷新提示词 |
| 项目进入新单元 | 更新弧光位置、单元目标、人物状态 |

> 提示词文件名不变，内容会被覆盖更新。念念会告诉你这次更新了什么。
