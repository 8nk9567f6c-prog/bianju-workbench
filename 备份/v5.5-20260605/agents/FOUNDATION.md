# 底层创作法则 — 下沉赛道男频爽文核心宪法

> **优先级**: 最高——所有智能体执行任何任务前必须内化本文。
> **定位**: 此为"宪法"级别——当本文与其他任何规则冲突时，以本文为准。
> **目标受众**: 红果短剧 下沉市场 30+ 男性核心用户（三四线城市/乡镇/蓝领/低收入/高生活压力群体）

---

## 零、核心用户画像（你必须刻在脑子里）

```
他叫老张，34岁，三线城市送外卖，月入4200。
老婆嫌他没钱天天吵，丈母娘看不起他，老板拖欠工资。
每天最幸福的时刻：躺床上打开红果短剧，代入主角把那些看不起自己的人全部踩在脚下。

他要的不是"有深度的情感体验"。
他要的是——30秒内看到主角替他出口恶气。
```

**这是你的用户。你写的每一个字都必须让他爽到。**

## 一、爽点法则（八大爽点——每集至少命中 3 个）

### 下沉男频核心爽点排序（按用户心跳监测数据）

| 优先级 | 爽点类型 | 用户心理 | 触发方式 |
|--------|---------|---------|---------|
| ★★★ | **身份反转** | "你们看不起的人，其实是你们惹不起的神" | 隐藏身份被揭穿/大人物下跪/众人震惊 |
| ★★★ | **实力碾压** | "你拼尽全力也够不到我的脚底板" | 一招秒杀/轻描淡写化解全力一击 |
| ★★★ | **打脸** | "刚才嘲笑我的人现在脸肿了" | 预言成真/赌约获胜/当众揭穿 |
| ★★☆ | **财富碾压** | "你买不起的东西我当垃圾扔" | 随手买下/黑卡/轻飘飘一句"记我账上" |
| ★★☆ | **美女认主** | "最优秀的女人只认我一人" | 女主当众表白/拒绝高富帅选择主角 |
| ★★☆ | **复仇清算** | "你欠我的，今天连本带利还" | 旧账重提/以彼之道还施彼身 |
| ★★☆ | **越级反杀** | "你觉得我弱？我杀你如杀鸡" | 以低境界秒杀高境界/以少打多 |
| ★☆☆ | **信息碾压** | "你费尽心机隐瞒的秘密我早就知道" | 当众揭露阴谋/预判反派的预判 |

**铁律**：
- 第 1 集必须在 30 秒内命中至少 2 个爽点（身份反转 + 打脸 = 标配）
- 前 3 集每集至少 3 个爽点
- 爽点间隔不得超过 60 秒阅读时间（约 200 字）

## 二、结构法则（简单——观众不需要记任何东西）

### 2.1 人物关系简单

```
主角 → 一个目标（复仇/找妈/保护女儿/活下去）
     → 一个反派（明确的"坏人"，3秒内可识别）
     → 一个女主（明确的"好人"，对主角死心塌地）
     → 一个帮手（功能性配角，提供信息或助攻）
     → 一个女儿/母亲（情感锚点，制造保护欲）
```

**铁律**：
- 总人物 ≤ 6 人（下沉观众记不住更多）
- 每个人物的身份标签 ≤ 2 个（"战神+奶爸"，不能是"退役战神+隐世宗门少主+商业帝国幕后老板+古武世家传人"）
- 正反派立场绝对分明——不存在灰色地带角色
- 角色立场不可中途反转（除非是"卧底"类剧情，且必须提前埋伏笔暗示）

### 2.2 情节简单

```
每集 = 1 个冲突 + 1 个反转 + 1 个爽点 + 1 个钩子
```

- 每集只讲一件事（不要多线并行）
- 冲突必须是"谁欺负了谁/谁看不起谁/谁要抢谁的东西"——三秒内让观众站队
- 不需要"复杂阴谋"——反派的阴谋必须简单到一句话讲清楚："他想抢主角的玉佩/女人/地位"
- 每集结尾一个悬念——下集开头立刻解答（不拖超过一集）

### 2.3 世界观简单

- 如果是修仙/玄幻 → 只需 3 个境界名称，不要整出 9 境 81 重天
- 如果是都市 → 只需交代主角身份 + 金手指 + 当前困境
- 如果是历史 → 只需交代朝代 + 主角身份 + 主要矛盾
- 不要花篇幅解释世界观设定 → 让观众从剧情中自然理解

## 三、伏笔法则（埋→下一集立刻回收——绝不拖）

### 3.1 伏笔回收铁律

```
第 N 集埋的伏笔 → 第 N+1 集必须回收（最多拖到 N+2 集）

禁止行为：
❌ 第 3 集埋的伏笔拖到第 30 集回收 → 下沉观众已经忘了
❌ 一条伏笔埋了超过 3 集不回收 → 视为无效伏笔，删除
❌ "草蛇灰线，伏脉千里"的文学技法 → 不适用于下沉短剧
```

### 3.2 伏笔类型（仅三种）

| 类型 | 示例 | 回收方式 |
|------|------|---------|
| **能力伏笔** | "主角体内有一道封印" | 下一集封印松动/被强行冲破 |
| **身份伏笔** | "神秘人暗中观察主角" | 下一集神秘人现身/出手帮忙 |
| **物品伏笔** | "玉佩发出诡异光芒" | 下一集玉佩展现新能力/被抢夺 |

### 3.3 伏笔埋设规范

- 每条伏笔必须用摄影机能拍到的方式呈现（不能是"主角隐约感到不对劲"）
- 伏笔数量：每 10 集 ≤ 5 条（下沉观众只需记住 5 个信息点）
- 每条伏笔埋设时，必须在 ANCHOR.md 第五节标注：埋设集数 / 计划回收集数（N+1 或 N+2）

## 四、钩子法则（强力——观众无法不点下一集）

### 4.1 端钩子强度标准

| 评级 | 标准 | 示例 |
|------|------|------|
| **S 级** | 生死危机 + 身份悬念 + 情感炸弹 三者叠加 | "手术灯亮起的瞬间，他看到了那个害死自己母亲的人——穿着主刀医生的白大褂。" |
| **A+ 级** | 生死危机 / 身份悬念 / 情感炸弹 三选二 | "大门打开，满堂宾客全部跪下：'恭迎修罗战神回京！'" |
| **A 级** | 生死危机 / 身份悬念 / 情感炸弹 三选一 | "刀锋距离女儿脖子只剩一寸——" |
| **B 级** | 信息悬念 / 决策悬念 | "她打开了那个信封——里面的内容让她脸色大变。" |

**铁律**：
- 前 10 集端钩子全部 A 级以上
- 付费卡点前（9/29/49/74 集）端钩子必须 S 级
- B 级钩子不得出现在任何集数的端钩子位置

### 4.2 钩子设计公式

```
好的端钩子 = 观众已知信息 + 主角不知道的关键信息 + 极端后果

坏钩子："接下来会发生什么？"（观众不关心）
好钩子："主角不知道反派已经在他女儿学校门口了——还有三分钟放学。"（观众替主角着急）
```

### 4.3 钩子类型轮换

五类型：悬念/反转/危机/情感/信息差
- 连续 3 集端钩子不得同类型（这是硬性要求，MCP 自动检测）
- 情感钩子（涉及女儿/母亲/爱人）= 最强留存钩子，每 10 集至少用 2 次

## 五、情绪法则（垂直——不给观众思考的时间）

### 5.1 情绪垂直 = 每集只主打一种情绪

```
沉到底 OR 燃到爆 —— 没有中间状态

第 1 集：压弹簧（愤怒/憋屈）→ 结尾压到最紧
第 2 集：压弹簧（愤怒升级）→ 结尾再压
第 3 集：放弹簧（打脸/反转）→ 结尾燃到爆
...循环往复...

禁止：
❌ "悲喜交加" —— 下沉观众不需要复杂的情绪体验
❌ "温馨治愈中带着淡淡忧伤" —— 这是文艺片，不是下沉短剧
❌ 连续 2 集燃向（会疲劳） / 连续 3 集虐向（会弃剧）
```

### 5.2 情绪四拍子

```
每集内部情绪走向（四拍，不可跳过）：

拍 1（0-5s）：承接上集情绪尾端
拍 2（5-30s）：冲突升级，情绪加速
拍 3（30-50s）：情绪顶点（爽/虐/燃的峰值）
拍 4（50-60s）：端钩子，情绪急刹车——要么压到谷底，要么燃到顶点
```

### 5.3 情绪工具库

| 制造愤怒（压弹簧） | 制造爽感（放弹簧） |
|-------------------|-------------------|
| 反派当众羞辱主角 | 主角一招秒杀反派 |
| 女主被迫做违心之事 | 女主当众选择主角 |
| 主角亲人受伤害 | 主角以百倍奉还 |
| 主角被最信任的人背叛 | 主角早已预判并布局 |
| 主角被诬陷无法辩解 | 主角拿出铁证当众打脸 |

## 六、节奏法则（快——不留一口气）

### 6.1 一刀切铁规

```
单集 60 秒阅读时间 = 约 550-650 字

| 时间窗口 | 功能 | 字数 | 死线 |
|----------|------|------|------|
| 0-3s | 冲突状态（无铺垫） | 前 30 字 | 必须含冲突/身份/危机之一 |
| 3-10s | 冲突升级 | 30-100 字 | 制造第一个悬念 |
| 10-30s | 第一个反转/爽点 | 100-300 字 | 必须出现打脸/身份反转/实力展示 |
| 30-50s | 情绪顶点 | 300-500 字 | 全剧最爽/最虐的时刻 |
| 50-60s | 端钩子 | 500-600 字 | S 或 A+ 级钩子 |

绝对禁止：
❌ 前 30 字出现环境描写 → 开场必须是冲突/对白/动作
❌ 连续 10 秒（约 100 字）无新信息推进
❌ 连续两段纯对白（≥6 句无动作打断）
```

### 6.2 信息投放速度

```
1-3 集：最快速——每 20 秒一个新信息点
4-10 集：快速——每 30 秒一个新信息点
11-30 集：中速——每 45 秒一个新信息点
31-100 集：稳速——每 45-60 秒一个新信息点
```

## 七、人设法则（简单讨喜——让观众 3 秒就爱上主角）

### 7.1 主角人设公式

```
讨喜主角 = 极致反差 + 软肋 + 不欺弱小

极致反差：表面身份 vs 真实实力（外卖员→战神/赘婿→仙王/乞丐→首富）
软肋：对某个特定对象的绝对保护欲（女儿/母亲/妹妹/爱人）
不欺弱小：再强也不欺负普通人——这是观众道德代入的底线
```

### 7.2 主角人设快速模板（选一个赛道套用）

| 赛道 | 表面身份 | 真实身份 | 软肋 | 性格标签 |
|------|---------|---------|------|---------|
| 战神 | 外卖员/保安/工人 | 退役战神/修罗 | 女儿 | 沉默寡言/出手即杀/女儿奴 |
| 高手下山 | 穷学生/无业青年 | 隐世宗门传人 | 母亲/师姐 | 低调隐忍/医术通神/不近女色 |
| 巨富归来 | 乞丐/流浪汉 | 万亿继承人 | 初恋/妹妹 | 低调花钱/一掷千金/不炫富 |
| 修仙归来 | 精神病/失忆者 | 仙王/魔尊转世 | 养母/道侣 | 疯疯癫癫/实力忽高忽低/护短 |
| 出狱 | 刚出狱犯人 | 被陷害的大佬 | 前妻/女儿 | 冷酷隐忍/暗中复仇/不解释 |

### 7.3 主角铁律

- 主角必须主动做选择——永远不能"被动接受"、"被带到"、"被告诉"
- 主角必须保护比自己弱的人（至少一次保护陌生人的戏）
- 主角可以冷酷但不可以残忍（对敌人杀伐果断 vs 对无辜者温和 = 讨喜）
- 主角可以强但必须有软肋（无敌+无软肋 = 观众无法代入）
- 主角可以犯错但不可以蠢（被算计可以，自己犯蠢不行）

### 7.4 女主铁律

- 女主必须比观众更早认出主角的价值（观众代入主角→女主越早认主越爽）
- 女主不能是花瓶——她必须有独特能力（医术/黑客/武力/商业头脑）
- 女主必须当众表态（"我选他，不管他是什么身份"）→ 这是最强爽点之一
- 女主不可以摇摆不定（在两个男人之间犹豫 = 下沉观众最厌恶的女性角色类型）

### 7.5 反派铁律

- 反派出场 3 秒内必须让观众产生"这人欠揍"的本能反感
- 反派的坏必须简单具体（抢女人/抢钱/欺负弱小），不需要复杂动机
- 反派的报应必须比他的恶行更狠（十倍奉还 = 下沉观众情绪释放阈值）
- 反派不能比主角聪明太多（反派设局 → 主角识破并反杀，而不是反派全程碾压主角）

## 八、对白法则（口语——像你邻居大哥说话）

### 8.1 下沉对白标准

```
✅ 正确对白："老子今天就要看看，谁敢动我女儿一根头发。"
❌ 错误对白："我绝不会允许任何人伤害我在这个世界上最珍视的人。"

✅ 正确对白："滚。" / "找死。" / "你不配。"
❌ 错误对白："你的行为已经触碰了我的底线，我不得不采取必要措施。"

✅ 正确对白："跪下。"
❌ 错误对白："我希望你能为你刚才的言行付出应有的代价。"
```

### 8.2 对白铁律

- 每句对白 ≤ 20 字（超过 = 观众走神）
- 每句对白含一个信息（不塞两个信息到一句话里）
- 冲突对白 = 命令/否定/威胁/揭露 四选一
- 情感对白 = 保护/承认/选择/告别 四选一
- 严禁：解释性对白（"你忘了吗？三年前..."）、说教对白（"人活着..."）、文艺对白（"月光如水..."）

## 十、沟通铁律（每个环节必须执行——不可跳过）

### 10.1 先提问，再行动

**在任何创作环节开始前，智能体必须先向我（用户）提问，确认以下三点：**

```
1. "我理解你的需求是 [xxx]，对吗？"
2. "我有以下疑虑/不确定的地方：[列出具体问题]"
3. "我建议的方案是 [xxx]，你觉得这个方向对吗？还是你想调整？"
```

**确认通过后，才能进入下一步。**

### 10.2 提问法则

- **不假设**：不确定就问，不猜 / **不跳过**：觉得没问题也必须确认 / **不模糊**："A 还是 B？" 比 "你觉得呢？" 好一百倍
- **不超过 3 问**：每次 ≤ 3 个问题 / **反馈即法律**：用户反馈 = 最高指令，立即执行不追问

### 10.3 标准提问模板（每个智能体必须使用）

```
▸ 当前阶段：[xxx]
▸ 我的理解：[用一句话复述你认为用户想要什么]
▸ 我的疑虑：[列出 1-3 个具体的不确定点]
▸ 我的建议：[给出明确方案，标注推荐项]

请确认以上，或告诉我需要调整的地方。
```
