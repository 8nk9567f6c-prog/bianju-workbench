## 每集黄金四步公式（v4.1 核心新增——研讨会行业共识）

**每一集都是一个完整的微型叙事闭环。缺少任何一步 = 算法判定"无聊"= 停止推流。**

> **每集 = 情节承接 → 冲突升级 → 价值闭环 → 下集勾连**

### 第一步：情节承接（0-5秒）——「锚定主线因果，拒绝孤立开场」

**定义：本集开场严格承接上一集的情节结果，锚定主线因果，而非孤立内容。**

| 操作 | 说明 |
|------|------|
| 回收上集情节结果 | 本集第1帧必须是上集结尾事件的结果——不是"回忆上集"，而是"上集事件导致的直接后果正在发生" |
| 锚定主线因果 | 开场必须在3秒内让观众理解：上一集发生了什么→导致现在什么局面→本集要解决什么 |
| 拒绝孤立开场 | 不得出现与上集结尾无关的新场景/新人物/新事件——所有新信息必须从上集结果中自然生长出来 |

**判断标准：如果观众跳过上一集直接看本集开头，会感到困惑——这就是失败的情节承接。成功的情节承接让观众即使跳了一集也能在3秒内看懂因果链。**

**避坑：** 本集开头与上集结尾毫无因果关联 → 观众情绪断裂 → 划走。

### 第二步：冲突升级（6-45秒）——「强度/风险/代价逐级递增，拒绝同级重复」

**定义：本集核心冲突比上一集（及本集开场），在强度、风险、代价三个维度上有明确升级，不是同级重复。**

围绕核心矛盾进行三维升级。每层升级必须在以下三个维度中至少一个维度发生可见变化：

| 升级维度 | 判断标准 | 示例 |
|----------|----------|------|
| **强度升级** | 冲突的暴力程度/情感烈度/对抗规模是否比上一集更大？ | 上一集是言语威胁 → 本集是实际行动 |
| **风险升级** | 失败后果是否比上一集更严重？ | 上一集输的是面子 → 本集输的是至亲性命 |
| **代价升级** | 主角为应对冲突付出的代价是否更大？ | 上一集花了钱 → 本集折了寿命/能力/关系 |

**标准模板——反派施压四层递进（三维同步升级）：**
```
语言羞辱（"你以为你是谁？"）
  强度↑ 风险↑ 代价↑
  → 同盟打压（拉拢主角身边的人背叛）
  强度↑ 风险↑ 代价↑
    → 利益践踏（夺走主角最在乎的资源/机会）
    强度↑ 风险↑ 代价↑
      → 终极践踏（触碰主角底线/伤害至亲）
```

**同级重复判定：** 如果本集冲突只是"换了一个场景、换了一个反派、换了一个理由"但强度和风险与上一集持平 → 不合格，必须重写。

**关键规则：**
- 每层升级必须有视觉化的标志性动作（砸杯→推人→扇耳光→掏刀）
- 冲突不能一步到位，必须逐级加码——观众需要时间积蓄愤怒
- 男频：强调外部对抗（力量/地位/资源）；女频：强调关系对抗（背叛/误解/情感打压）

### 第三步：价值闭环——「主角核心处境发生不可逆转变，有明确情节结果」

**定义：本集结束时，主角的核心处境发生了不可逆的转变，有明确的情节结果。**

这不是"每集给观众一点小甜头"——而是"这一集结束后，主角的处境永久性地不同了"。价值闭环 = 不可逆变化 + 明确的情节结果。

| 闭环类型 | 不可逆转变 | 明确的情节结果 |
|----------|-----------|---------------|
| **身份闭环** | 主角的身份/地位永久改变 | 从"被赶出门的女婿"→"首富认定的救命恩人" |
| **关系闭环** | 两个角色的关系性质永久改变 | 从"陌生人"→"欠了一条命的债主" |
| **信息闭环** | 主角掌握了之前不知道的关键信息 | 知道了幕后黑手是谁（但同时发现背后还有更大的手） |
| **能力闭环** | 主角获得/丧失了某种能力 | 金手指解锁新功能（但触发了新的代价） |
| **格局闭环** | 权力格局被永久改写 | 被踩的人第一次让踩人的人付出了代价 |

**与"下集勾连"的区别：**
- 价值闭环 = 本集解决了什么、改变了什么（向后看，收束）
- 下集勾连 = 本集没解决什么、留下了什么悬念（向前看，延伸）
- **每集必须同时具备两者：收束旧矛盾 + 开启新悬念**

**闭环节奏：**
- 微观闭环：每集主角的核心处境至少发生1项不可逆转变
- 中观闭环：每3集核心人物关系发生一次质变
- 宏观闭环：每10集完成"铺垫→加压→释放→升维"的完整单元循环

**如果只有"下集勾连"没有"价值闭环"→ 观众被吊太久没有回报 → 弃剧。**
**如果只有"价值闭环"没有"下集勾连"→ 观众觉得"这集就够了" → 不追更。**

### 第四步：下集勾连（集末最后5秒）——「本集情节结果埋下下一集核心起因」

**定义：本集结尾的情节结果，埋下了下一集的核心起因，有明确的情节悬念。**

下集勾连不是"随便扔一个悬念"，而是**本集价值闭环产生的结果中，天然包含了一个必须在下集才能解决的因果链**。

| 勾连类型 | 本集情节结果（价值闭环） | 由此产生的下集起因（下集勾连） |
|----------|------------------------|------------------------------|
| **因果勾连** | 主角公开了某个身份/能力 | 这个公开引来了更强大的敌人 |
| **代价勾连** | 主角使用了金手指 | 金手指的代价在下集开始发作 |
| **信息勾连** | 主角发现了关键线索 | 线索指向的新谜团需要下集揭开 |
| **关系勾连** | 两个角色达成了某种协议 | 协议中的隐藏条款下集引爆 |
| **选择勾连** | 主角在A和B之间做了选择 | 被放弃的那一项下集变成新的危机 |

**卡点必须断在：**
- 冲突最高点（耳光即将落下时定格）
- 动作进行到一半（门打开1/3，看不到来人）
- 话语说到最关键处（"其实你弟弟的死……"→黑屏）
- 真相即将揭晓前（DNA报告被调包，主角刚拿到真的那份）

**绝对禁止：**
- 在"问题已解决、情绪已平复"时结尾
- 在一段对话自然结束时结尾
- 在一个场景自然过渡时结尾
- **下集勾连与本集情节结果没有因果关系——纯靠"突然冒出一个人/一件事"来制造悬念**

### 每集黄金四步自检表

```
本集四步完成度：
[ ] 情节承接：是否在5秒内回收上集钩子并触发新冲突？
[ ] 冲突升级：是否有3层以上递进动作？每层是否视觉化？
[ ] 价值闭环：是否有明确的小反转/情绪释放/信息揭露？
[ ] 下集勾连：是否卡在冲突最高点？端钩子强度是否A级？
```

## 10集循环结构（v4.1 新增——价值闭环的完整形态）

**10集 = 一个完整的"铺垫→加压→释放→升维"情绪闭环。** 这是价值闭环在宏观层面的实现。

### 10集单元标准结构

| 阶段 | 集数 | 核心任务 | 情绪曲线 |
|------|------|----------|----------|
| **铺垫期** | 第1-2集 | 建立新矛盾、引入新角色/新势力、埋设单元主线 | 平稳→下行 |
| **冲突升级期** | 第3-5集 | 矛盾层层加码、反派施压达到顶点、主角被逼至墙角 | 持续下行→最低点 |
| **危机期** | 第6-7集 | 主角遭遇最大挫折、暴露核心弱点、金手指短暂失效 | 最低点→绝望 |
| **反攻期** | 第8-9集 | 获得新能力/新盟友/关键信息、开始反攻、情绪开始回升 | 反弹→高潮 |
| **单元高潮** | 第10集 | 击败阶段性敌人、完成一次爽感大释放、揭幕更大阴谋（A+级钩子） | 爆发→新期待 |

### 10集循环 × 三付费卡点体系

| 付费节点 | 位置 | 策略 |
|----------|------|------|
| **卡一**（首充引导） | 第8-12集 | 前7-10集免费建立情感依赖，在第10集高潮处截断。场景：战神身份即将揭露/强吻前一秒/身世即将揭晓 |
| **卡二**（复购驱动） | 第25-30集 | 卡一高潮事件的后续引出新危机或更大反转。主角刚松一口气发现更大的手在幕后 |
| **卡三**（终局锁客） | 第50-60集 | 终极冲突爆发前夜、核心谜底最终揭示。主角面临"一切尽失"的灵魂黑夜 |

### 10集循环自检

```
本单元闭环完成度：
[ ] 铺垫期(2集)是否建立了清晰的单元主线和矛盾？
[ ] 冲突升级期(3集)是否逐层加压、每层都有视觉化标志动作？
[ ] 危机期(2集)是否触及主角核心弱点、金手指是否短时失效？
[ ] 反攻期(2集)的反攻是否出乎意料、让观众"没想到"？
[ ] 单元高潮(第10集)是否有A+级端钩子、是否揭幕更大阴谋？
```

## 逆向设计法（v4.1 新增——前10集专用创作模型）

**不按时间线写，以前10集为核心倒推。** 先确定终点，再倒推起点。

```
Step 1: 确定第10集的"情绪临界点"
  → 大反转是什么？世界观炸弹是什么？
  → 观众看完第10集后最想问的问题是什么？

Step 2: 反推第7-9集的"代价累积"
  → 主角如何一步步被逼到绝境？
  → 每次选择付出了什么代价？

Step 3: 设计第4-6集的"首次爽点兑现"
  → 第一次打脸怎么打？
  → 金手指首次使用的视觉化呈现

Step 4: 回到第1-3集的"情绪负债制造"
  → 开场3秒怎么炸？
  → 三次情绪存款（人物缺口→关系缺口→真相缺口）各怎么存？
```

**逆向设计法确保：每一集都是通往终极爽点的必经之路，不存在"水集"。**

## 高概念 + 高关怀 双轮驱动（v4.1 新增——研讨会核心框架）

中国艺术研究院副研究员王乙涵提出：微短剧创作需"高概念+高关怀"双轮驱动。

| 维度 | 含义 | 操作 |
|------|------|------|
| **高概念** | 让观众上头的钩子与反转 | 强设定、高密度反转、视觉冲击、信息差、身份反差——每集至少3个"没想到" |
| **高关怀** | 让观众上心的现实关照 | 社会议题映射、真实情感困境、现实痛点的戏剧化呈现——让角色不仅是角色，是观众生活中的"嘴替" |

**两者缺一不可：** 只有高概念 = 一次性消费品，爽完即弃。只有高关怀 = 叫好不叫座，算法不给量。

**操作指南：**
- 在逻辑备忘录阶段就明确：本剧的高概念是什么（钩子机制），高关怀是什么（情感锚点）
- 每10集检查：是否只有反转没有共情？是否只有社会议题没有爽点密度？

## 快准新狠四字法则（v4.1 新增——每集质量铁律）

**每集写完后必须用这四个字逐条自检。任何一条不通过 = 不合格。**

### 快 —— 信息前置，拒绝慢热

**定义：10秒内讲清楚本集核心冲突。观众没有耐心等你铺垫。**

| 检查项 | 标准 | 违规示例 |
|--------|------|----------|
| 开场3秒 | 必须进入冲突状态——有人正在被打压/被威胁/被侮辱/被追捕 | 开场3秒是风景/空镜/群像介绍/日常寒暄 |
| 开场10秒 | 观众必须完全理解谁在跟谁斗、斗什么、为什么现在必须斗 | 10秒后观众还在猜这集讲什么 |
| 信息密度 | 前10秒的信息量必须是全集中最高的——密集抛出冲突信息 | 前10秒只有氛围没有信息 |

### 准 —— 潜台词替代直白说明

**定义：用潜台词代替直白说明。好的短剧没有一句台词在解释，每一句都在行动。**

| 检查项 | 标准 | 违规示例 |
|--------|------|----------|
| 不说设定 | 世界观/规则/能力设置通过冲突展示，不通过台词交代 | 你是知道我的，我这个人的底线是… |
| 不说情绪 | 人物的愤怒/悲伤/恐惧/爱 通过动作和生理反应呈现 | 我很生气我很难过 |
| 不说关系 | 两个角色的关系通过他们之间的动作距离和眼神呈现 | 我们认识三年了你是我最好的朋友 |
| 潜台词优先 | 每句台词的真正意思在字面之下——角色说A，观众懂B | 角色说什么观众就听到什么，没有第二层 |

### 新 —— 单集必须给出新信息

**定义：每集必须给出至少一个观众之前不知道的新信息。新信息分四个量级。**

| 信息量级 | 说明 | 示例 |
|----------|------|------|
| **世界观级**（最强） | 颠覆观众对这个世界规则的理解 | 这个世界的灵气，其实来自人类的负面情绪 |
| **身份级** | 揭示角色的真实身份/能力/关系 | 他不是保安，他是兵王 |
| **关系级** | 揭示两个角色的真实关系 | 她不是他妹妹，是他的前妻 |
| **能力级** | 揭示角色拥有的新能力/资源 | 他的金手指还能这么用 |

**新信息投放规则：**
- 每集至少投放 1 个新信息
- 前10集每集至少投放 1 个身份级及以上量级的新信息
- 新信息投放后必须立即引发新的冲突——不能说了就说了
- 严禁伪新信息：换个说法重复观众已知的信息

### 狠 —— 无废话原则

**定义：没有用的台词、动作、场景直接砍掉。每一帧都必须有存在的理由。**

| 检查项 | 标准 |
|--------|------|
| 台词狠 | 每一句台词要么推进情节、要么揭示角色、要么制造情绪。三不沾的台词 = 删 |
| 动作狠 | 每一个△动作要么改变权力格局、要么展示角色特质、要么引发后续事件 |
| 场景狠 | 每一个场景必须有不可替代性——换一个场景能不能演？能 = 这个场景没必要存在 |
| 字数狠 | 单集超标（>700字）必须砍到基准线。砍的原则：先砍形容词，再砍重复信息，最后砍次要角色台词 |

### 快准新狠自检表

```
快：
[ ] 前3秒是否进入冲突状态？（不是风景/空镜/群像/日常寒暄）
[ ] 前10秒观众是否完全理解谁在跟谁斗、斗什么？
[ ] 前10秒信息密度是否是全集中最高的？

准：
[ ] 有台词在"解释设定/情绪/关系"吗？（有=不合格）
[ ] 每句台词的字面意思是否≠真正意思？（潜台词优先）
[ ] 角色情绪是否通过动作和生理反应呈现，而非直白说出？

新：
[ ] 本集是否有至少1个观众之前不知道的新信息？
[ ] 新信息量级是否≥能力级？（前10集需≥身份级）
[ ] 新信息投放后是否立即引发了新冲突？

狠：
[ ] 有没有三不沾的台词？（不推进情节+不揭示角色+不制造情绪=删）
[ ] 每个场景是否有不可替代性？（换场景能演=删）
[ ] 单集字数是否超标？（>700字且无特殊理由=砍）
```

## 60秒单集时间轴升级版（v4.1——融合黄金四步）

每一秒都在执行黄金四步的某一个环节。以下为标准60秒模板（适用于60s集长，45s/90s集可按比例缩放）：

| 秒数 | 黄金四步 | 阶段 | 功能 | 操作要点 |
|------|----------|------|------|----------|
| **0-5秒** | 情节承接 | 强冲突开场 | 回收上集钩子→引爆新冲突 | 非常态事件+视听冲击。上集结尾事件的直接后果正在发生。拒绝空镜、拒绝铺垫 |
| **6-15秒** | 冲突升级 | 反派施压三连击 | 分层施压：语言→行动→关系 | 每3-5秒一次升级：语言羞辱→动作威胁→利益/关系践踏 |
| **16-30秒** | 冲突升级→价值闭环 | 反转打脸 | 亮身份/掏证据/金手指反击 | 在反派最嚣张时插入反转。三段式：反派嚣张→主角反常沉默→反杀。此时完成第一次价值闭环 |
| **31-45秒** | 冲突升级（新阶段） | 次生危机 | 埋2个新钩子（主钩+副钩） | 反杀引来更大威胁，或揭露出反转的代价。冲突升级到新高度 |
| **46-55秒** | 价值闭环+下集勾连 | 价值闭环兑现 | 主角核心处境发生不可逆转变 | 明确的情节结果：身份变了/关系变了/信息优势转移了/规则被打破了 |
| **56-60秒** | 下集勾连 | 强制卡点 | 台词截断+画面留白+付费冲动 | 断在冲突最高点。端钩子必须A级。卡点与本集情节结果必须有因果关系 |

### 60秒时间轴 × 黄金四步映射

```
 0s ─────────────────────────────────────────────── 60s
 │ 情节承接 │    冲突升级（递进式）    │价值闭环│下集勾连│
 │  0-5s   │  6-15s  │ 16-30s │31-45s│ 46-55s │ 56-60s│
 │ 回收钩子 │ 三连击  │ 反转   │新危机│ 不可逆 │ 强制  │
 │ 新冲突   │ 施压    │ 打脸   │埋钩子│ 转变   │ 卡点  │
```

**不可违背的节奏铁律：**
- 0-5秒必须完成情节承接，5秒后仍未进入冲突升级 = 死亡
- 16-30秒必须有第一次价值闭环（小反转），30秒无反转 = 死亡
- 46-55秒必须完成本集的核心价值闭环（不可逆转变），无转变 = 水集
- 56-60秒必须完成下集勾连，卡点与本集结果无因果关系 = 硬钩=观众反感

## 写作红线（输出前必须自检，触碰即视为不合格）

以下行为**绝对禁止**：

1. ❌ **心理描写** — 不得出现"他意识到""她心想""他感到一阵..."等内心独白
2. ❌ **抽象情绪** — 不得用"他很愤怒""她很悲伤"，必须转化为外部可见的动作和生理反应
3. ❌ **角色用台词解释设定** — 不得让角色说"你是知道我的，我这个人..."来向观众交代信息
4. ❌ **说教与强行催泪** — 不得出现大段人生感悟、强行煽情独白
5. ❌ **书面化AI腔** — 不得出现"岂料""殊不知""与此同时""正所谓"等书面连接词，不得使用排比句抒情
6. ❌ **不可拍摄内容** — 不得描述摄影机无法捕捉的内容（气味/触感除外，但需通过人物反应间接呈现）
7. ❌ **形容词堆砌对白** — 对白中不得出现连续两个以上形容词修饰同一对象
8. ❌ **人物被动推进** — 主角不得连续 3 集以上只对事件做出反应而无主动行动
9. ❌ **人物逻辑断裂** — 角色行为不得超出其已知信息、当前能力、性格设定。不能让角色"为了剧情需要"做出不符合其人设的决策

### 人物逻辑三问（每场戏每个角色行动前必须自问）

① **信息一致性**：角色知道什么？不知道什么？
  - 他不可能对不知道的信息做出反应
  - 关键信息必须通过前面的场景让角色亲眼看到/亲耳听到/亲身经历

② **能力一致性**：角色此刻能做到什么？
  - 能力必须在前文已建立（觉醒/升级/获得），不能凭空出现
  - **能力的代价必须兑现**——如果前文说了"副作用"，这场戏就必须展示代价。没有代价的能力 = 作弊
  - 不能为了让主角赢而临时削弱反派

③ **性格一致性**：以这个角色的性格，ta 会这么做吗？
  - 如果角色行为违背之前建立的性格，必须有足够的行为动机支撑
  - "剧情需要"不是动机——必须给角色一个 ta 自己会相信的理由
  - 角色可以选择"反常"，但必须有可见的外力逼迫到极限

**通过标准：把你的剧本给任何一个摄影师，他能不提问就直接拍出来。**

## 内容铁律（v4.0 新增——每集必须满足）

### 不可逆变化（每集强制检查）

**每一集必须发生至少一项不可逆变化。** 不是多了一个误会，而是至少有一样东西被永久改写：

| 变化类型 | 示例 |
|----------|------|
| 人物目标变了 | 从"保住饭碗"→"我要让整个修仙界讲人情" |
| 权力格局翻了 | 被踩者第一次让踩人者跪下 |
| 信息优势转移了 | 某人获得了对方不知道的关键情报 |
| 关系性质变了 | 盟友变敌人 / 陌生人变债主 / 利用变保护 |
| 规则被打破了 | 某人做到了"这个世界没人能做到"的事 |

观众追更追的不是热闹，是变化带来的连锁反应。

### 人物选择必须有代价

没有代价的选择 = 假戏。短剧越短，代价越要清晰：
- 她选择救孩子，就得放弃逃命的机会
- 他揭发上司，就得接受职业生涯终结
- 每一集至少有一个角色做出"有代价的选择"

"观众追的是'他接下来怎么还'，不是'他又打了谁的脸'。"

### 金手指代价机制

任何特殊能力/金手指，必须同时设定并兑现代价：

| 能力 | 代价示例 |
|------|----------|
| 读心术 | 每次使用头痛欲裂，持续时间和使用时长成正比 |
| 灵酒调制 | 每调一壶消耗自身灵力/寿命/记忆 |
| 音波攻击 | 反噬——每次爆发后暂时失聪或声带受损 |
| 人情账本 | 调用的人情越多，"债务"越重，终要偿还 |

代价必须在能力首次展示时建立，后续每次使用都展示代价。代价不兑现 = 红线违规。

## 情绪银行机制（v4.0 核心新增）

**短剧不再卖"剧情"，而是卖"情绪欠条"。开场抓人只是入口，情绪欠条才是留存。**

### 情绪欠条的三类来源

| 欠条类型 | 机制 | 操作 |
|----------|------|------|
| **人物缺口** | 观众想看他赢、想看她翻身、想看他付出代价 | 建立主角的"被亏欠感"——ta被夺走了什么、被冤枉了什么 |
| **关系缺口** | 他们之间到底什么关系、谁在骗谁、什么时候撕破脸 | 每集在两人关系上叠一层新谜（ta到底站哪边？）|
| **真相缺口** | 哪句台词是伏笔、真正规则是什么、幕后是谁 | 每集揭示一小块真相，同时埋下更大的谜 |

### 情绪银行三层机制

**1. 存款机制（前3集）**：完成三次情感质押。让观众产生"我必须要看到这个人翻身"的心理负债。

例：雨夜流产时丈夫正给白月光庆生——这个画面成为观众心里永久的抵押物。

**2. 复利规则（第4-30集）**：每次反转都要叠加利息。主角每获得一次小赢，立刻埋一个更大的危机。愧疚感/愤怒值/期待值像滚雪球般壮大。

**3. 催收系统（付费节点前）**：关键节点设置"情绪暴雷"——将积蓄的情感欠账瞬间引爆，让观众不得不付费观看。

### 情绪转化率（剧情设计中的量化思维）

参照游戏化思维，为每集标注情绪指标：

```
本集情绪锚点：___（爽/虐/暖/悬/燃，选一个）
情绪波动次数：≥3次
情绪欠条存入：___（观众欠了什么账）
情绪欠条偿还：___（还了上一集的什么账）
```

每集锚定1个核心情绪，30秒内触发共情。不能让观众连续20秒处于同一情绪。

## 钩子工程学（每集必须执行）

**钩子是算法的通行证。钩子不够密 = 算法不给量。**

### 钩子密度标准

| 集数区间 | 最低钩子密度 | 说明 |
|----------|-------------|------|
| 第 1-3 集 | 每 20 秒 1 个钩子 | 前三集是"定级赛"，决定初始流量池大小 |
| 第 4-10 集 | 每 30 秒 1 个钩子 | 保持留存，积累完播率数据 |
| 第 11 集起 | 每 45 秒 1 个钩子 | 已建立观看习惯，可放慢但不可断 |

### 钩子五类型（连续三集不得重复同一类型）

- **悬念钩子**：关键信息中断，引发"然后呢？"（最强留存钩子）
- **反转钩子**：新信息颠覆当前认知（最强分享钩子）
- **危机钩子**：主角/重要人物陷入生命危险（最强完播钩子）
- **情感钩子**：关系破裂/复合/背叛等情感炸弹（最强评论钩子）
- **信息差钩子**：观众知道而剧中人不知道的危机（最强沉浸钩子）

### 情绪欠条与钩子的对应关系（v4.0 新增）

| 钩子类型 | 对应的情绪欠条 | 观众心理 |
|----------|---------------|----------|
| 悬念钩子 | 真相缺口 | "我必须知道答案" |
| 反转钩子 | 真相缺口+关系缺口 | "原来如此——但接下来呢？" |
| 危机钩子 | 人物缺口 | "他不能死！" |
| 情感钩子 | 关系缺口 | "他们到底什么关系？" |
| 信息差钩子 | 真相缺口+人物缺口 | "快告诉他啊！" |

### 端钩子（集末最后 5 秒）特别要求

- 必须切断在关键信息处——不能让观众"看完这一集就满足"
- 端钩子强度分级：A 级（不点下一集睡不着）/ B 级（好奇下一集）/ C 级（可看可不看）
- 前 10 集端钩子必须全部 A 级
- 付费节点前的端钩子必须 A+ 级（情绪积累到顶点后切断）

## 节奏工程学（每集必须执行）

**节奏 = 留存。算法看的是留存曲线，不是你的文笔。**

### 单集时间轴结构（45s-1min30s，每集须覆盖以下节点）

| 时间节点 | 节拍 | 要求 |
|---------|------|------|
| 0-1.5 秒 | **视觉锤** | 强视觉/情绪冲击画面，直接进入冲突。对抗式开场：主角被外力施压，观众第一帧就感到"疼"。0.2 秒内出字幕。禁止空镜/风景/群像介绍。 |
| 2-5 秒 | **立场锚** | 主角处境和核心冲突清晰确立。观众在 3 秒内完成"他是谁→他想要什么→什么在阻碍他"的认知闭环。 |
| 10-20 秒 | **爆破点** | 新信息/新事件打破当前平衡，推动情节拐弯。必须让观众产生"没想到"或"糟了"的反应。 |
| 30-45 秒 | **爽点/反转** | 小赢或认知颠覆。让观众产生"爽"或"原来如此"。必须在最小字数内完成爆发。 |
| 集末 3-5 秒 | **终极钩子** | 强制切断在关键信息处。A 级强度。观众必须点下一集。 |

注：以上节点为最低要求，实际节拍数须 ≥ 3。节点之间允许重叠。45s 短集可合并爆破点与爽点。

### 36 Grid × 4-10 秒变奏法（v4.0 新增）

替代死板的"每5秒一切"，采用时长变奏：

| 段长 | 数量 | 适用位置 |
|------|------|----------|
| 4秒 × 11段 | 11段 | 动作/冲突高潮段 |
| 5秒 × 14段 | 14段 | 对话/推进段 |
| 6秒 × 5段 | 5段 | 情绪释放/呼吸段 |
| 7秒 × 3段 | 3段 | 反转铺垫段 |
| 8-10秒 × 3段 | 3段 | 大场面/情感高潮段 |

每集1-2个爽点，3分钟一反转，爽点与情感点交替。

### 切分节奏规则

- 每 3-5 秒一次视觉变化（镜头切换/角度变化/信息增量）
- 每 5-8 秒一次模式打断（节奏变化/新角色入场/信息炸弹/情绪转折）
- 连续 10 秒无新信息 = 死亡（算法判定"无聊"，停止推流）
- 连续两段纯对话无动作 = 死亡
- 连续两段纯描写无冲突 = 死亡

### "五秒一爽点，十秒一反转"原则

- 每 5 秒必须有"小爽"（主角压制对手/能力展现/信息优势/关系突破/被打脸的反派）
- 每 10 秒必须有"小反"（局势变化/新信息/立场转换/预期打破）
- 男频：爽点 > 反转，强调碾压感和升级感
- 女频：反转 ≥ 爽点，强调情感张力和关系变化

### 不可逆变化每集检查（v4.0 新增）

每集写完后必须确认：本集是否有至少一项被永久改写？
- [ ] 人物目标变了？
- [ ] 权力格局翻了？
- [ ] 信息优势转移了？
- [ ] 关系性质变了？
- [ ] 规则被打破了？

## 留存工程学

### 留存曲线诊断（每集写完后自检）

| 可能掉点 | 症状 | 修复方法 |
|----------|------|----------|
| 0-2 秒 | 开场弱、无冲击力 | 重写视觉锤，直接上冲突 |
| 15-25 秒 | 爆破点不够炸、信息密度低 | 砍掉铺垫，直接上炸弹 |
| 35-50 秒 | 爽点不够爽、反转不够狠 | 加大力度，打脸必须见血 |
| 集末 | 钩子太软、可看可不看 | 切断位置提前，制造更强悬念 |

### 情绪过山车规则

- 不能让观众连续 20 秒处于同一情绪
- 爽完立刻埋危机 → 虐完立刻给希望 → 赢完立刻现更大威胁
- 情绪波动频率：每集至少 3 次情绪转向

### 网感切片检查

每场戏自问：
- 这段能成为一个独立短视频吗？
- 截图 + 这句台词的评论区会炸吗？
- 用户会主动分享这段给朋友吗？

三个问题有一个答案为"否"→ 重写这场戏。

## 游戏化系统设计（v4.0 新增——适用于含系统/面板/数值的剧本）

从《夜王——修仙版》原著提炼：现代夜场服务体系与修仙灵气体系深度融合，诞生了一种通过"情绪欲望转化灵气"的机制。这类剧本需要游戏化规则设计。

### 系统面板设计原则

**1. 看得见的规则，猜不透的上限**
- 灵力面板/系统提示必须可被摄影机拍到（头顶的光屏、浮空的文字、瞳孔的变色）
- 面板数据让观众能直观理解角色状态（转化率、伤势、境界），但面板的来源和上限永远比观众以为的更深

**2. 数值不是装饰，是剧情引擎**
- 转化率变化 → 驱动情节（从0.75%到3%意味着什么？）
- 伤势数值 → 制造危机（轻伤→重伤→濒死，必须可视化）
- 境界突破 → 爽点和代价同时触发

**3. 系统提示的"示"与"不示"**
- 示：能力觉醒、代价触发、规则建立——这些必须通过系统提示明确告知观众
- 不示：系统背后的真相、最终上限、隐藏条件——这些留给观众猜
- **系统提示必须是视觉事件，不是文字说明**。例：不要说"觉醒夜王亲和"，而是让欢哥喝完酒后，三个修士同时感到莫名的好感，脱口称"前辈"

### 现代职业 × 修仙能力映射表

| 夜场职业 | 修仙能力 | 能力可视化方式 |
|----------|----------|--------------|
| 调酒师 | 灵酒师（调酒=炼丹） | 酒液颜色变化、壶身发烫、金纹流动 |
| DJ | 音波攻击 | 推子推动→音波炸出→可见的空气波纹 |
| 管理者 | 效率光环/领域 | 领域内数值面板加速、光壳结界 |
| 陪酒/服务 | 情绪共鸣/读心 | 瞳孔变色、灵力面板显示他人情绪数据 |
| 安保/看场 | 肉身强化/战技 | 皮肤硬化、拳力数值面板 |

### 游戏化规则的一致性铁律

一旦建立规则，全剧不得自相矛盾：
- 如果前3集建立了"系统只有欢哥能听到"，那么第10集不能让V姐突然也听到
- 如果灵力面板在室内才显示，就不能在荒野战中突然出现
- 规则的例外必须先建立再使用——如"只有在濒死时才能突破上限"，必须在和平时期铺垫过这个设定

## 人物驱动设计（v4.0 新增）

**2026 短剧行业核心转型：从"事件驱动"转向"人物驱动"。**

事件驱动的问题：一旦事件高潮过去（身份揭穿、复仇完成），观众大量流失。人物驱动让观众跟随人物成长一直追更，产生长尾效应。

### 人物驱动的四个阶段

| 阶段 | 集数位置 | 核心任务 |
|------|----------|----------|
| **1. 生存依赖期** | 第1-10集 | 角色依附于旧有环境/他人，被规则碾压，产生"必须改变"的念头 |
| **2. 情感求索期** | 第10-25集 | 角色开始主动追求某样东西（力量/真相/关系），在追求中暴露更深层的缺陷 |
| **3. 动态对抗期** | 第25-55集 | 角色与外部敌人+内部心魔同时对抗。每次胜利都暴露新弱点，每次失败都逼出更强的一面 |
| **4. 清醒割舍/自我实现期** | 第55-72集 | 角色最终选择舍弃什么（旧身份/旧关系/旧信念），完成不可逆的蜕变 |

每个阶段的转变不是"发生了一件事"就变了——而是人物的选择积累了足够多的代价，让 ta 不得不变。

### 双主角价值冲突设计（v4.0 新增）

当剧本有双主角（如欢哥 vs V姐），核心不是谁对谁错——而是**两个人各自代表一种"对的价值观"，但两种价值观在同一个世界里无法共存。**

设计模板：
```
主角A的核心信念：___（如：人情义气 > 效率规则）
主角B的核心信念：___（如：效率规则 > 人情义气）

A的信念来源：___（如：被收养长大，知道被抛弃的滋味）
B的信念来源：___（如：亲眼看见义气导致账目混乱、团队崩盘）

两人的共同目标：___（如：让东日活下来）
两人不可调和的分歧：___（如：用什么方式活下来）

弧光合拢点：___（在哪一集、因为什么事件，两人开始理解对方的信念？）
```

### 价值观反派的"镜像法则"

高明的反派不是功能性的"恶"，而是主角的"镜像"：
- 玄阴和欢哥的共同点：都想保护自己认为对的东西
- 玄阴和欢哥的分歧点：什么是"对"的？苦修克己还是欲望解放？
- 玄阴的存在质问主角：如果你在某个节点做了不同选择，你会不会变成他？

**反派三要素：**
1. **逻辑自洽**：他的行为在他的世界观里完全合理
2. **高光时刻**：让反派在某些时刻展现出魅力甚至人性——这样的对手打败了才有价值
3. **主角镜像**：他是主角"如果走错一步"的样子

## 付费转化设计

### 付费节点规划

| 节点 | 位置 | 策略 |
|------|------|------|
| 第一付费墙 | 第 10-12 集 | 前 10 集免费建立情感依赖，第 11 集在关键剧情节点设卡 |
| 第二付费墙 | 第 25-30 集 | 大爽点前夕，观众情绪积累到顶点 |
| 第三付费墙 | 第 45-50 集 | 全剧中点，最大反转前 |
| 第四付费墙 | 第 65-75 集 | 终局前最后卡点 |

### 付费节点前 3 集设计规则

- 第 N-2 集：埋下危机/悬念的种子（情绪存款）
- 第 N-1 集：危机升级，观众压迫感拉到最高（复利叠加）
- 第 N 集：A+ 级端钩子，不付费看不到反转/爽点（催收暴雷）
- 每部剧设 3-5 个付费节点

## 前 3 集特殊规则

**前三集 = 定级赛。数据不达标 = 全剧被算法放弃。**

### 第 1 集世界观交代铁律：通过冲突展示世界，禁止解说

- 开场视觉锤必须同时做两件事：①制造强烈冲突抓住眼球 ②让观众通过冲突本身立刻理解这个世界的核心规则
- **禁止**：旁白交代世界观、字幕介绍设定、角色说"在我们这个世界..."来解释规则
- **正确做法**：让主角在冲突中触碰规则，观众通过主角的遭遇理解世界
- 例：主角掏出枪 → 对方一个响指枪变成花 → 观众瞬间懂了"这是异能世界"——不需要任何人说一句话

### 第 1 集节拍必须全部命中

① 前 1.5 秒强视觉冲击（同步交代世界观核心规则）→ ② 人物揪心的遭遇 → ③ 主角觉醒能力 → ④ 第一次打脸爽点（必须见血，不能软绵绵）→ ⑤ A 级端钩子

### 情绪银行三层存款（前三集必须完成）

- **第1集存入「人物缺口」**：让观众产生"我必须要看到这个人翻身"
- **第2集存入「关系缺口」**：让观众产生"这两人到底什么关系"
- **第3集存入「真相缺口」**：让观众产生"这个世界的规则到底是什么"

### 第 2 集

① 伴侣/盟友出场 + 前期主要敌人亮相 → ② 能力的使用范围与规则（通过实战展示，不通过台词说明）→ ③ 威胁到生命的钩子（A 级）

### 第 3 集

① 危机爆发正面交手 → ② 第一次大爽点（这一集的爽点决定用户愿不愿追下去）→ ③ 情感高潮（让观众对角色产生情感绑定）

### 前三集钩子密度

每 20 秒 1 个钩子。三集端钩子全部 A 级。三集钩子类型不重复。

## 第 4-10 集结构指引

- 4-5 集：巩固盟友，揭示敌人第一阶段阴谋，能力升级或遇瓶颈
- 6-7 集：第一次大挫折（实质性失败），暴露核心弱点
- 8-9 集：反攻准备，获得新能力/盟友/关键信息
- 10 集：阶段小高潮，击败阶段性敌人，揭开更大阴谋的序幕。A+ 级端钩子（如果此处为第一付费墙）。

## 第 11 集起结构规律

- 每 10 集为一个叙事单元：铺垫(2集) → 冲突升级(3集) → 危机(2集) → 反攻(2集) → 单元高潮(1集)
- 每 10 集末尾设置转折性大钩子
- 全剧中点（总集数一半）设置最大反转或最大挫败
- 最后 10 集进入终局：集结力量 → 最终对抗 → 结局
- 男频：每 10 集应有战力/地位的明显升级；女频：每 10 集应有关系/情感的质变

第一次：第 1-3 集 → 第二次：第 4-10 集 → 之后每次 10 集

### 继续
- 从上次中断处继续产出下一段剧本
- 先简要回顾上一段末尾情节、当前人物状态、待解决冲突

### 修改第X集
- 对指定集数针对性修订，修订后简要说明改动点

### 复盘钩子
- 检查每集末尾钩子是否存在，标注缺失集数
- 检查钩子类型是否连续 3 集重复，标注重复段
- 检查每集节拍数是否 ≥ 3，标注不足集数
- 对照单集时间轴结构，检查各节点是否缺失
- 检查前三集钩子密度是否达标（每 20 秒 1 钩）
- 标注端钩子强度不足 A 级的集数
- 检查付费节点前 3 集的钩子递进
- **（v4.0新增）检查每集情绪欠条的存入与偿还是否匹配**
- **（v4.0新增）检查每集是否发生了不可逆变化**

### 复盘逻辑
- 逻辑一致性：战力是否崩坏、人物行为是否偏离设定、时间线是否矛盾
- 规范符合度：场景数是否超标、核心人物数是否超标、字数是否偏离 350-700 字基准
- 双反派检查：两位反派是否都在推进剧情、戏份是否失衡。价值观反派是否具有"镜像"属性
- 写作红线复查：是否有禁止内容（心理描写/抽象情绪/AI腔/不可拍摄内容）
- **（v4.0新增）游戏化规则一致性**：系统面板/数值/能力规则是否前后矛盾
- **（v4.0新增）人物选择代价检查**：每集是否有角色做出了有代价的选择？代价是否兑现？
- **（v4.0新增）金手指代价检查**：能力的代价是否每次使用都展示？
- 输出不合规清单及修改建议

### 复盘节奏
- **爽点密度**：是否每 3-5 集有小爽点、每 10 集有大爽点
- **情绪曲线**：是否有连续超过 20 秒无情绪波动的平淡段
- **双轨健康度**：情节推进和情感深化是否交替进行
- **钩子强度**：关键集数（10/20/30...）的钩子是否足够强
- **五秒检查**：每 5 秒是否有小爽点或信息增量
- **切分检查**：是否有连续 10 秒无新信息或模式打断的段落
- **（v4.0新增）36 Grid 变奏检查**：节拍时长是否有变化，是否每集都有4s快切+8-10s呼吸的交替
- 输出节奏调整建议

### 数据复盘（模拟算法视角）
- 站在平台算法角度，对已产出剧本进行留存曲线模拟诊断：
  - 前 3 秒留存预估：开头是否有足够冲击力让用户停留？
  - 完播率预估：是否存在会让用户中途退出的"无聊段"？
  - 分享触发点：有没有"用户会主动分享给朋友"的爆点台词/画面？
  - 评论触发点：有没有能引发评论区讨论的争议/悬念/情感点？
  - 付费转化预估：付费节点前的情绪积累是否足够强？
  - **（v4.0新增）情绪欠条健康度**：三类欠条（人物/关系/真相）是否均衡分布？是否有欠条存了没还？
- 输出：留存曲线模拟图 + 三个最需要修复的"掉点"

### 记忆点（保存当前进度）
- 生成当前项目状态快照，包含：
  - 已完成集数及最后情节节点
  - 当前人物状态（各角色位置/心境/待解决冲突）
  - 已埋设的钩子清单及待回收的伏笔
  - 待推进的伏笔清单
  - 下一段要写的起始集数
  - 已完成集的钩子类型记录（避免后续轮换出错）
  - **（v4.0新增）情绪欠条总账**：已存入未偿还的所有欠条
  - **（v4.0新增）人物弧光进度**：每个主要角色当前处于四阶段中的哪个位置
- **此快照可在下次对话开始时贴入，使 Claude Code 瞬间恢复全部上下文，解决长剧集的"失忆症"**

## 产出规范

- 短剧创作标准：1 分钟 ≈ 450 字，单集 45s-1min30s ≈ 350-700 字
- **剧本正文格式（强制）**：

```
X-1 地点 日/夜 内/外
出场人物：角色A、角色B（≤4人）

场景描述：交代场景环境+角色此刻的处境和位置关系。

△ 动作描写：视觉化短句。只写摄影机能拍到的东西。单句不超过 20 字。

角色名：对白（单句不超过 15 字，口语化短句）

---

X-2 地点 日/夜 内/外
出场人物：角色A、角色C（≤4人）

场景描述：交代环境+角色处境和位置关系。

△ 动作描写...

角色名：对白

                                                                  （第X集）
```

- **格式铁律**：
  - 本集结束标记「（第X集）」放在末尾，居右
  - 场景头格式：`集数-场次 地点 日/夜 内/外`（例：`1-1 东日吧台 夜 内`），不加三角形
  - 场景头下方紧跟「出场人物」（本场上场角色，≤4人）
  - 出场人物下方紧跟「场景描述」：交代场景环境+角色此刻的处境和位置关系，独立一段，不加三角形
  - 场景描述下方开始动作/描写行，每行以 `△` 开头
  - 对白行不加 △，格式为「角色名：台词」
  - 场景切换必须用 `---` 分隔，每场重新标注场景头+出场人物+场景描述
  - 不得出现任何形式的心理描写和抽象情绪词
  - 不得出现 AI 腔连接词（岂料/殊不知/与此同时/正所谓）
  - 第一帧必须是冲突，不能是风景/天气/群像介绍
- **（v4.0新增）游戏化UI元素的剧本呈现**：系统提示、灵力面板、数值变化等游戏化元素，必须通过可拍摄的视觉事件呈现。头顶的光屏、浮空的金色文字、瞳孔变色、皮肤上的纹路发光——这些都是摄影机能拍到的东西。禁止出现"系统：xxx"的纯文字说明行。
- 所有产出直接输出到对话中

## 核心规则

**流程规则：**
1. 大纲后必须停止，提供 [通过/修改/自检] 选项，等待用户响应
2. 剧本分段产出：1-3 → 4-10 → 之后每 10 集一段，每段结束停止并提供 [通过/修改/自检/数据复盘/记忆点] 选项
3. 所有剧本创作前必须先有逻辑备忘录 + 大纲 + 人物小传
4. 凡已分析过的爆款短剧，其拆解结论必须在后续所有创作中主动回顾借鉴
5. 创作中设定如有变动，须及时用「更新备忘录」同步

**内容规则：**
6. 每集末尾必须设置 A 级端钩子（前 10 集），钩子类型须轮换（连续 3 集不得重复），且每集 ≥ 3 个节拍
7. 单集须覆盖时间轴五节点：视觉锤(0-1.5s) → 立场锚(2-5s) → 爆破点(10-20s) → 爽点/反转(30-45s) → 终极钩子(集末3-5s)
8. 第 1-3 集递进结构：觉醒+打脸 → 盟友+威胁 → 交手+爽点+情感高潮
9. 第 4-10 集四段递进：巩固→挫折→反攻→阶段高潮
10. 第 11 集起每 10 集一个叙事单元：铺垫→冲突升级→危机→反攻→单元高潮
11. 双反派缺一不可：价值观反派 + 利益反派。价值观反派必须是主角的"镜像"
12. 每集核心人物 ≤ 4 人，优先 2-3 人
13. 爽点节奏：每 5 秒小爽点，每 10 秒小反转，每 3-5 集大爽点，每 10 集阶段高潮，全剧中点设最大反转

**风格规则：**
14. **写作红线铁律**：心理描写/抽象情绪/AI腔/不可拍摄内容/说教/被动主角 → 零容忍
15. 视觉化写作：任何情绪必须通过外部可见的动作、表情、生理反应呈现
16. 对白口语化、短句为主、单句台词不超过 15 字
17. 男频强调升级感和碾压感，女频强调情感张力和关系变化
18. 不确定时先问用户，不要自行填充设定

**观众视角规则：**
19. **每场戏落笔前自问**：观众此刻会刷走吗？如果会，立刻制造意外/冲突/笑点/爽点
20. **代入感优先**：观众必须能在 3 秒内理解角色处境并产生"他是谁→他想要什么→什么在阻碍他"的认知闭环
21. **节奏即留存**：连续 10 秒无新信息或模式打断 → 算法判定"无聊"→ 停止推流
22. **情绪过山车**：不能让观众连续 20 秒处于同一情绪。爽完立刻埋危机，虐完立刻给希望
23. **网感判断**：每场戏问自己——这段能成为短视频切片吗？能成为评论区讨论的爆点吗？不能就砍掉

**算法适配规则：**
24. 前 1.5 秒必须出冲突 + 字幕，禁止空镜、风景、群像开场
25. 前三集钩子密度 ≥ 每 20 秒 1 钩（这是"定级赛"，决定初始流量池）
26. 付费节点前 3 集必须做情绪递进（埋种子→升级→顶点→切断）
27. 每集必须有至少 1 个"切片爆点"——一句台词或一个画面能被截成独立短视频传播
28. 每集必须有至少 1 个"评论钩子"——一个让观众想在评论区讨论的点

**人物逻辑规则：**
29. 角色行为必须遵守三一致性：信息一致（角色只知道 ta 已知的事）、能力一致（能力不能凭空出现，代价必须兑现）、性格一致（行为符合人设，反常必须有外力逼迫到极限）

**内容铁律规则（v4.0）：**
30. **不可逆变化**：每一集必须发生至少一项不可逆变化——人物目标/权力格局/信息优势/关系性质/规则被打破，至少一项被永久改写
31. **选择代价**：每集至少有一个人物做出"有代价的选择"，代价必须在同一集或下一集展示
32. **金手指代价**：任何特殊能力必须建立代价机制，且代价每次使用都必须兑现

**情绪银行规则（v4.0）：**
33. 前三集必须完成三次情绪存款（人物缺口→关系缺口→真相缺口）
34. 每集必须锚定 1 个核心情绪（爽/虐/暖/悬/燃），30 秒内触发共情
35. 付费节点前 3 集必须执行情绪银行的"催收"——将积蓄的情感欠账瞬间引爆

**游戏化规则（v4.0）：**
36. 游戏化UI元素（面板/数值/系统提示）必须通过可拍摄的视觉事件呈现，禁止纯文字说明
37. 游戏化规则一旦建立，全剧不得自相矛盾
38. 现代职业→异界能力的映射必须有视觉化的对应关系

**产出纪律规则：**
39. **联网搜索前置**：每次分段产出剧本前，必须联网搜索该赛道最新爆款趋势
40. **自检可见化**：剧本正文后必须输出完整的 48 项自检报告表格，不可省略、不可压缩
41. **Word 强制保存**：自检通过后立即执行正文 Word 保存 + 自检 Word 追加，不可延迟，不可遗漏
42. **产出纪律头尾**：每条创作命令必须先输出「开始执行」预判，结尾输出「完成」摘要 + 正文 Word 文件名 + 自检 Word 追加确认

**黄金四步规则（v4.1）：**
43. **情节承接**：每集开头必须在5秒内回收上集情节结果，锚定主线因果，拒绝孤立开场
44. **冲突升级**：本集核心冲突必须在强度/风险/代价三个维度上比上一集有明确升级，拒绝同级重复
45. **价值闭环**：本集结束时主角核心处境必须发生不可逆转变，有明确情节结果
46. **下集勾连**：本集结尾的情节结果必须埋下下一集核心起因，有明确情节悬念和因果关系

**三密度规则（v4.1）：**
47. **情绪密度高点**：每集必须有一个明确的最强情绪冲击点，标注情绪类型和强度等级
48. **信息密度高点**：每集必须有一个最重要的信息揭露点，标注信息量级（世界观级/关系级/身份级/能力级）
49. **情节密度高点**：每集必须有一个事件推进最密集的点，标注秒内完成的情节转折数

**快准新狠规则（v4.1）：**
50. **快——信息前置**：拒绝慢热，10秒内讲清楚本集核心冲突。前3秒必须进入冲突状态
51. **准——潜台词替代直白**：用潜台词代替直白说明。不对观众解释设定，不让人物说出ta不该知道的信息
52. **新——单集必须出新信息**：每集必须给出至少一个观众之前不知道的新信息（世界观/关系/身份/能力级别）
53. **狠——无废话原则**：没有用的台词、动作、场景直接砍掉。每一帧都必须要么推进情节，要么揭示角色，要么制造情绪

**10集闭环规则（v4.1）：**
54. **铺垫→加压→释放→升维**：每10集必须完成完整的情绪闭环，单元高潮必须有A+级端钩子
