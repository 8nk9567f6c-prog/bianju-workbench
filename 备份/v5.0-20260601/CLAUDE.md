## 推荐工作流（五阶段审核制）

```
创建新项目 → 导入素材 → 内容检测 → 对标分析 → 逻辑备忘录 → 剧情简介 → 人物小传 → 大纲 → 分集梗概
    ↓
一审（1-3集） → 逻辑审核 → 通过后继续
    ↓
二审/一卡（4-10集） → 逻辑审核 → 通过后继续
    ↓
三审/二卡（11-30集） → 逻辑审核 → 通过后继续
    ↓
50集审核 → 逻辑审核 → 通过后继续
    ↓
100集完本 → 终审 → 导出
```

**每个审核节点必须执行：**
1. 剧情大纲一致性检查：已写剧情是否偏离大纲
2. 人物设定一致性检查：角色行为是否违反人设
3. 已有内容逻辑检查：前后矛盾、伏笔遗漏、能力崩坏
4. 平台红线检查：是否触碰内容安全边界
5. 格式规范检查：是否符合剧本格式要求

### 下沉剧核心特征

- **强冲突开场**：前 3 秒必须有耳光/砸杯/踹门等级别的外部冲突
- **简单人物关系**：好人 vs 坏人清晰分明，拒绝灰色人物
- **快节奏打脸**：每 3-5 集一次打脸爽点，反派不能"憋"超过 5 集
- **情感直给**：不搞含蓄、不玩隐喻，爱恨情仇直接演出来
- **身份反差**：核心爽点来自"你以为他是废物→他其实是王者"
- **口语化对白**：下沉市场的语言，拒绝文艺腔和书面语

## 平台红线——内容安全铁律（输出前必须自检，触碰即视为不合格）

### 绝对禁止内容

1. ❌ **涉政敏感** — 不得涉及国家领导人、政治体制批判、历史虚无主义
2. ❌ **色情低俗** — 不得出现露骨性描写、性暗示画面、软色情擦边
3. ❌ **暴力血腥** — 不得出现过于血腥暴力的详细描写（打脸见血可，分尸断肢不可）
4. ❌ **违法示范** — 不得详细展示犯罪手法、吸毒过程、赌博技巧
5. ❌ **封建迷信** — 算命/天师/修仙类需设定为"架空世界观"，不得宣扬现实封建迷信
6. ❌ **未成年不良引导** — 不得出现未成年人恋爱、校园暴力美化
7. ❌ **性别歧视** — 不得出现"女人就该xxx""男人必须xxx"等性别刻板说教
8. ❌ **地域黑/职业黑** — 不得以特定地域/职业为嘲讽对象
9. ❌ **版权侵权** — 不得直接搬运已有作品的标志性情节和金句

### 敏感内容需规避

- 警察/军人/政府官员不得作为纯反派（如需出现，必须有正面执法/正义动机）
- 医院/学校等公共机构不得被描写为系统性腐败
- 涉及真实地名/企业名/品牌名需用化名
- 结局必须"正义战胜邪恶"，主角不能以纯粹违法手段获胜

## 每集黄金四步公式（v4.1 保留——研讨会行业共识）

**每一集都是一个完整的微型叙事闭环。缺少任何一步 = 算法判定"无聊"= 停止推流。**

> **每集 = 情节承接 → 冲突升级 → 价值闭环 → 下集勾连**

### 第一步：情节承接（0-5秒）——「锚定主线因果，拒绝孤立开场」

**定义：本集开场严格承接上一集的情节结果，锚定主线因果，而非孤立内容。**

| 操作 | 说明 |
|------|------|
| 回收上集情节结果 | 本集第1帧必须是上集结尾事件的结果——不是"回忆上集"，而是"上集事件导致的直接后果正在发生" |
| 锚定主线因果 | 开场必须在3秒内让观众理解：上一集发生了什么→导致现在什么局面→本集要解决什么 |
| 拒绝孤立开场 | 不得出现与上集结尾无关的新场景/新人物/新事件——所有新信息必须从上集结果中自然生长出来 |

**判断标准：如果观众跳过上一集直接看本集开头，会感到困惑——这就是失败的情节承接。成功的情节承接让观众即使跳了一集也能在3秒内看懂因果链。**

### 第二步：冲突升级（6-45秒）——「强度/风险/代价逐级递增，拒绝同级重复」

**定义：本集核心冲突比上一集（及本集开场），在强度、风险、代价三个维度上有明确升级，不是同级重复。**

围绕核心矛盾进行三维升级。每层升级必须在以下三个维度中至少一个维度发生可见变化：

| 升级维度 | 判断标准 | 示例 |
|----------|----------|------|
| **强度升级** | 冲突的暴力程度/情感烈度/对抗规模是否比上一集更大？ | 上一集是言语威胁 → 本集是实际行动 |
| **风险升级** | 失败后果是否比上一集更严重？ | 上一集输的是面子 → 本集输的是至亲性命 |
| **代价升级** | 主角为应对冲突付出的代价是否更大？ | 上一集花了钱 → 本集折了寿命/能力/关系 |

**标准模板——反派施压四层递进（三维同步升级）：**
```
语言羞辱（"你以为你是谁？"）
  强度↑ 风险↑ 代价↑
  → 同盟打压（拉拢主角身边的人背叛）
  强度↑ 风险↑ 代价↑
    → 利益践踏（夺走主角最在乎的资源/机会）
    强度↑ 风险↑ 代价↑
      → 终极践踏（触碰主角底线/伤害至亲）
```

**关键规则：**
- 每层升级必须有视觉化的标志性动作（砸杯→推人→扇耳光→掏刀）
- 冲突不能一步到位，必须逐级加码——观众需要时间积蓄愤怒
- 男频：强调外部对抗（力量/地位/资源）；女频：强调关系对抗（背叛/误解/情感打压）

### 第三步：价值闭环——「主角核心处境发生不可逆转变，有明确情节结果」

**定义：本集结束时，主角的核心处境发生了不可逆的转变，有明确的情节结果。**

这不是"每集给观众一点小甜头"——而是"这一集结束后，主角的处境永久性地不同了"。价值闭环 = 不可逆变化 + 明确的情节结果。

| 闭环类型 | 不可逆转变 | 明确的情节结果 |
|----------|-----------|---------------|
| **身份闭环** | 主角的身份/地位永久改变 | 从"被赶出门的女婿"→"首富认定的救命恩人" |
| **关系闭环** | 两个角色的关系性质永久改变 | 从"陌生人"→"欠了一条命的债主" |
| **信息闭环** | 主角掌握了之前不知道的关键信息 | 知道了幕后黑手是谁（但同时发现背后还有更大的手） |
| **能力闭环** | 主角获得/丧失了某种能力 | 金手指解锁新功能（但触发了新的代价） |
| **格局闭环** | 权力格局被永久改写 | 被踩的人第一次让踩人的人付出了代价 |

**与"下集勾连"的区别：**
- 价值闭环 = 本集解决了什么、改变了什么（向后看，收束）
- 下集勾连 = 本集没解决什么、留下了什么悬念（向前看，延伸）
- **每集必须同时具备两者：收束旧矛盾 + 开启新悬念**

### 第四步：下集勾连（集末最后5秒）——「本集情节结果埋下下一集核心起因」

**定义：本集结尾的情节结果，埋下了下一集的核心起因，有明确的情节悬念。**

下集勾连不是"随便扔一个悬念"，而是**本集价值闭环产生的结果中，天然包含了一个必须在下集才能解决的因果链**。

| 勾连类型 | 本集情节结果（价值闭环） | 由此产生的下集起因（下集勾连） |
|----------|------------------------|------------------------------|
| **因果勾连** | 主角公开了某个身份/能力 | 这个公开引来了更强大的敌人 |
| **代价勾连** | 主角使用了金手指 | 金手指的代价在下集开始发作 |
| **信息勾连** | 主角发现了关键线索 | 线索指向的新谜团需要下集揭开 |
| **关系勾连** | 两个角色达成了某种协议 | 协议中的隐藏条款下集引爆 |
| **选择勾连** | 主角在A和B之间做了选择 | 被放弃的那一项下集变成新的危机 |

**卡点必须断在：**
- 冲突最高点（耳光即将落下时定格）
- 动作进行到一半（门打开1/3，看不到来人）
- 话语说到最关键处（"其实你弟弟的死……"→黑屏）
- 真相即将揭晓前（DNA报告被调包，主角刚拿到真的那份）

**绝对禁止：**
- 在"问题已解决、情绪已平复"时结尾
- 在一段对话自然结束时结尾
- 在一个场景自然过渡时结尾
- **下集勾连与本集情节结果没有因果关系——纯靠"突然冒出一个人/一件事"来制造悬念**

### 每集黄金四步自检表

```
本集四步完成度：
[ ] 情节承接：是否在5秒内回收上集钩子并触发新冲突？
[ ] 冲突升级：是否有3层以上递进动作？每层是否视觉化？
[ ] 价值闭环：是否有明确的小反转/情绪释放/信息揭露？
[ ] 下集勾连：是否卡在冲突最高点？端钩子强度是否A级？
```

## 逆向设计法（v4.1 保留——前10集专用创作模型）

**不按时间线写，以前10集为核心倒推。** 先确定终点，再倒推起点。

```
Step 1: 确定第10集的"情绪临界点"
  → 大反转是什么？世界观炸弹是什么？
  → 观众看完第10集后最想问的问题是什么？

Step 2: 反推第7-9集的"代价累积"
  → 主角如何一步步被逼到绝境？
  → 每次选择付出了什么代价？

Step 3: 设计第4-6集的"首次爽点兑现"
  → 第一次打脸怎么打？
  → 金手指首次使用的视觉化呈现

Step 4: 回到第1-3集的"情绪负债制造"
  → 开场3秒怎么炸？
  → 三次情绪存款（人物缺口→关系缺口→真相缺口）各怎么存？
```

**逆向设计法确保：每一集都是通往终极爽点的必经之路，不存在"水集"。**

## 快准新狠四字法则（v4.1 保留——每集质量铁律）

**每集写完后必须用这四个字逐条自检。任何一条不通过 = 不合格。**

### 快 —— 信息前置，拒绝慢热

**定义：10秒内讲清楚本集核心冲突。观众没有耐心等你铺垫。**

| 检查项 | 标准 | 违规示例 |
|--------|------|----------|
| 开场3秒 | 必须进入冲突状态——有人正在被打压/被威胁/被侮辱/被追捕 | 开场3秒是风景/空镜/群像介绍/日常寒暄 |
| 开场10秒 | 观众必须完全理解谁在跟谁斗、斗什么、为什么现在必须斗 | 10秒后观众还在猜这集讲什么 |
| 信息密度 | 前10秒的信息量必须是全集中最高的——密集抛出冲突信息 | 前10秒只有氛围没有信息 |

### 准 —— 潜台词替代直白说明

**定义：用潜台词代替直白说明。好的短剧没有一句台词在解释，每一句都在行动。**

| 检查项 | 标准 | 违规示例 |
|--------|------|----------|
| 不说设定 | 世界观/规则/能力设置通过冲突展示，不通过台词交代 | 你是知道我的，我这个人的底线是… |
| 不说情绪 | 人物的愤怒/悲伤/恐惧/爱 通过动作和生理反应呈现 | 我很生气我很难过 |
| 不说关系 | 两个角色的关系通过他们之间的动作距离和眼神呈现 | 我们认识三年了你是我最好的朋友 |
| 潜台词优先 | 每句台词的真正意思在字面之下——角色说A，观众懂B | 角色说什么观众就听到什么，没有第二层 |

### 新 —— 单集必须给出新信息

**定义：每集必须给出至少一个观众之前不知道的新信息。**

| 信息量级 | 说明 | 示例 |
|----------|------|------|
| **世界观级**（最强） | 颠覆观众对这个世界规则的理解 | 这个世界的灵气，其实来自人类的负面情绪 |
| **身份级** | 揭示角色的真实身份/能力/关系 | 他不是保安，他是兵王 |
| **关系级** | 揭示两个角色的真实关系 | 她不是他妹妹，是他的前妻 |
| **能力级** | 揭示角色拥有的新能力/资源 | 他的金手指还能这么用 |

**新信息投放规则：**
- 每集至少投放 1 个新信息
- 前10集每集至少投放 1 个身份级及以上量级的新信息
- 新信息投放后必须立即引发新的冲突——不能说了就说了
- 严禁伪新信息：换个说法重复观众已知的信息

### 狠 —— 无废话原则

**定义：没有用的台词、动作、场景直接砍掉。每一帧都必须有存在的理由。**

| 检查项 | 标准 |
|--------|------|
| 台词狠 | 每一句台词要么推进情节、要么揭示角色、要么制造情绪。三不沾的台词 = 删 |
| 动作狠 | 每一个△动作要么改变权力格局、要么展示角色特质、要么引发后续事件 |
| 场景狠 | 每一个场景必须有不可替代性——换一个场景能不能演？能 = 这个场景没必要存在 |
| 字数狠 | 单集字数须 ≥ 500 字（100集×500=5w+）。低于 500 字需检查节拍是否完整 |

## 剧本正文格式（v5.0 新版——强制执行）

**以下格式为唯一标准，所有剧本正文必须严格遵守。违反格式 = 不合格。**

### 格式模板

```
第x集

001 日/夜 内/外 场景

人物：角色A 角色B

服装：
- 角色A：服装描述
- 角色B：服装描述

道具：
- 道具1
- 道具2
- 道具3

【角色A站在xxx，角色B坐在xxx，两人之间距离约xxx。环境xxx，光线xxx。氛围xxx。】

△ 描述性内容——交代此刻正在发生的事情，角色的状态和情绪的外在表现。

△ 角色A走向xxx，伸手拿起xxx。

角色A：对白内容（口语化短句）

角色B：对白内容

△ 角色B转身，脚步急促。

△ 门被猛地推开。

---
```

### 格式要素详解

| 要素 | 格式 | 说明 |
|------|------|------|
| **集数标记** | `第x集` | 每集开头，独立一行，顶格 |
| **场景编号** | `001 日/夜 内/外 场景` | 三位数编号（001/002/003...），每集内独立编号。日夜 + 内外 + 场景名 |
| **人物** | `人物：角色A 角色B` | 本场出场人物，用空格分隔，≤4人 |
| **服装** | `- 角色名：服装描述` | 每个出场人物的服装，分别列出。需可拍摄 |
| **道具** | `- 道具名` | **本集所有出现道具必须在此罗列。禁止在正文中贸然出现未罗列的道具。** |
| **场景描述** | `【...】` | 本集/本场开头时角色的站位、动作关系、环境、氛围。必须用【】括起 |
| **描述/动作** | `△ ...` | 所有描述性内容和动作均以 △ 开头。只写摄影机能拍到的东西 |
| **对白** | `角色名：台词` | 不加 △，角色名+冒号+台词。口语化短句 |
| **分场** | `---` | 场景切换时使用，独立一行 |
| **集末标记** | `（第x集完）` | 每集结尾，独立一行 |

### 道具规则（v5.0 新增强制）

**道具必须在每场开头罗列完整。正文中出现的任何物品，必须预先出现在道具列表中。**

- 道具包括：手机、钥匙、文件、武器、酒杯、药品、照片、信件、行李箱、车辆（提及的）等所有被角色使用/提及/展示的物品
- 消耗性道具（如水杯被打翻→水和杯子都是道具）也需罗列
- 如果后文需要某个道具但前面未罗列 → 必须回头补充，或在场景切换时新增道具列表
- **违规后果：审核不通过，必须修正**

### 场景描述规则（v5.0 新增强制）

**【】中的场景描述必须包含以下信息：**
1. 角色的初始站位和姿势（站在哪、坐着还是站着、面向什么方向）
2. 角色之间的空间关系（距离多远、谁在上风谁在下风、有没有物理阻隔）
3. 环境要素（光线明暗、空间大小、有无其他人、天气）
4. 氛围（紧张/松弛/暧昧/冰冷——通过可见的环境线索呈现，不直接说"氛围紧张"）

**示例：**
```
✅ 【办公室内，灯光明亮。张三站在办公桌前，双手撑在桌面上，身体前倾。李四坐在办公桌后的转椅上，身体后仰，双腿交叠，右手转着一支钢笔。两人之间隔着一张红木办公桌，距离不到一米。窗外是阴天，光线灰白。】
```
```
❌ 【办公室，张三和李四在争吵。】——缺少站位、动作关系、环境细节
```

### △ 符号使用规则

- **所有描述性内容和动作均以 △ 开头**
- 场景编号、人物、服装、道具、【】场景描述、对白 —— 这些不加 △
- △ 后面的内容只写摄影机能拍到的东西
- 单句不超过 20 字
- 不得出现心理描写、抽象情绪词

## 钩子工程学（每集必须执行）

### 钩子密度标准（100集）

| 集数区间 | 最低钩子密度 | 说明 |
|----------|-------------|------|
| 第 1-3 集 | 每 20 秒 1 个钩子 | 前三集是"定级赛"，决定初始流量池大小 |
| 第 4-10 集 | 每 30 秒 1 个钩子 | 保持留存，积累完播率数据 |
| 第 11-30 集 | 每 45 秒 1 个钩子 | 已建立观看习惯，但不可松懈 |
| 第 31-100 集 | 每 45-60 秒 1 个钩子 | 保持节奏，单元高潮处加大密度 |

### 钩子五类型（连续三集不得重复同一类型）

- **悬念钩子**：关键信息中断，引发"然后呢？"（最强留存钩子）
- **反转钩子**：新信息颠覆当前认知（最强分享钩子）
- **危机钩子**：主角/重要人物陷入生命危险（最强完播钩子）
- **情感钩子**：关系破裂/复合/背叛等情感炸弹（最强评论钩子）
- **信息差钩子**：观众知道而剧中人不知道的危机（最强沉浸钩子）

### 端钩子（集末最后 5 秒）特别要求

- 必须切断在关键信息处——不能让观众"看完这一集就满足"
- 端钩子强度分级：A 级（不点下一集睡不着）/ B 级（好奇下一集）/ C 级（可看可不看）
- 前 10 集端钩子必须全部 A 级
- 付费节点前的端钩子必须 A+ 级（情绪积累到顶点后切断）
- 每 10 集单元高潮（第10/20/30/.../100集）必须 A+ 级端钩子

## 节奏工程学（每集必须执行）

### 单集时间轴结构（每集须覆盖以下节点）

| 时间节点 | 节拍 | 要求 |
|---------|------|------|
| 0-1.5 秒 | **视觉锤** | 强视觉/情绪冲击画面，直接进入冲突。0.2 秒内出字幕。 |
| 2-5 秒 | **立场锚** | 主角处境和核心冲突清晰确立。观众在 3 秒内完成认知闭环。 |
| 10-20 秒 | **爆破点** | 新信息/新事件打破当前平衡，推动情节拐弯。 |
| 30-45 秒 | **爽点/反转** | 小赢或认知颠覆。必须在最小字数内完成爆发。 |
| 集末 3-5 秒 | **终极钩子** | 强制切断在关键信息处。A 级强度。 |

### 切分节奏规则

- 每 3-5 秒一次视觉变化（镜头切换/角度变化/信息增量）
- 每 5-8 秒一次模式打断（节奏变化/新角色入场/信息炸弹/情绪转折）
- 连续 10 秒无新信息 = 死亡（算法判定"无聊"，停止推流）
- 连续两段纯对话无动作 = 死亡
- 连续两段纯描写无冲突 = 死亡

## 情绪银行机制（v4.0 保留）

**短剧不再卖"剧情"，而是卖"情绪欠条"。**

### 情绪欠条的三类来源

| 欠条类型 | 机制 | 操作 |
|----------|------|------|
| **人物缺口** | 观众想看他赢、想看她翻身 | 建立主角的"被亏欠感"——ta被夺走了什么、被冤枉了什么 |
| **关系缺口** | 他们之间到底什么关系、什么时候撕破脸 | 每集在两人关系上叠一层新谜 |
| **真相缺口** | 哪句台词是伏笔、真正规则是什么、幕后是谁 | 每集揭示一小块真相，同时埋下更大的谜 |

### 情绪银行三层机制

**1. 存款机制（前3集）**：完成三次情感质押。让观众产生"我必须要看到这个人翻身"的心理负债。

**2. 复利规则（第4-50集）**：每次反转都要叠加利息。主角每获得一次小赢，立刻埋一个更大的危机。

**3. 催收系统（付费节点前）**：关键节点设置"情绪暴雷"——将积蓄的情感欠账瞬间引爆，让观众不得不付费观看。

## 游戏化系统设计（v4.0 保留——适用于含系统/面板/数值的剧本）

### 系统面板设计原则

**1. 看得见的规则，猜不透的上限**
- 灵力面板/系统提示必须可被摄影机拍到（头顶的光屏、浮空的文字、瞳孔的变色）

**2. 数值不是装饰，是剧情引擎**
- 转化率变化 → 驱动情节
- 伤势数值 → 制造危机
- 境界突破 → 爽点和代价同时触发

**3. 系统提示的"示"与"不示"**
- 示：能力觉醒、代价触发、规则建立
- 不示：系统背后的真相、最终上限、隐藏条件
- **系统提示必须是视觉事件，不是文字说明**

## 写作红线（输出前必须自检，触碰即视为不合格）

以下行为**绝对禁止**：

1. ❌ **心理描写** — 不得出现"他意识到""她心想""他感到一阵..."等内心独白
2. ❌ **抽象情绪** — 不得用"他很愤怒""她很悲伤"，必须转化为外部可见的动作和生理反应
3. ❌ **角色用台词解释设定** — 不得让角色说"你是知道我的，我这个人..."来向观众交代信息
4. ❌ **说教与强行催泪** — 不得出现大段人生感悟、强行煽情独白
5. ❌ **书面化AI腔** — 不得出现"岂料""殊不知""与此同时""正所谓"等书面连接词
6. ❌ **不可拍摄内容** — 不得描述摄影机无法捕捉的内容（气味/触感除外，但需通过人物反应间接呈现）
7. ❌ **形容词堆砌对白** — 对白中不得出现连续两个以上形容词修饰同一对象
8. ❌ **人物被动推进** — 主角不得连续 3 集以上只对事件做出反应而无主动行动
9. ❌ **人物逻辑断裂** — 角色行为不得超出其已知信息、当前能力、性格设定

### 人物逻辑三问（每场戏每个角色行动前必须自问）

① **信息一致性**：角色知道什么？不知道什么？
② **能力一致性**：角色此刻能做到什么？能力代价是否兑现？
③ **性格一致性**：以这个角色的性格，ta 会这么做吗？

**通过标准：把你的剧本给任何一个摄影师，他能不提问就直接拍出来。**

## 核心规则

**流程规则：**
1. 所有剧本统一 100 集，全剧 ≥ 5w 字，单集 ≥ 500 字
2. 五阶段审核制：一审(3集) → 二审一卡(10集) → 三审二卡(30集) → 50集审核 → 100集完本终审
