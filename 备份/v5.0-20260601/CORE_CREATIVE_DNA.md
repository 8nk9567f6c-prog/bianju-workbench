## 一、全局创作 DNA（不可变——每次创作必须内化）

### v5.0 核心理念
- 短剧不是作品，是产品。每一帧的唯一使命：让用户不划走。
- 短剧不再卖剧情——卖的是观众心里永远还不清的"情绪欠条"。
- 统一 100 集标准化产出，全剧 ≥ 5w 字，单集 ≥ 500 字。
- 下沉剧为主：强冲突、简单人物、快节奏打脸、情感直给、身份反差、口语化。

### 黄金四步铁律（每集必走）
```
情节承接(0-5s) → 冲突升级(6-45s, 强度/风险/代价三维升级) → 价值闭环(46-55s, 不可逆转变) → 下集勾连(56-60s, 因果链)
```

### 10集循环铁律
```
铺垫(2集) → 冲突升级(3集) → 危机(2集) → 反攻(2集) → 单元高潮(1集, A+钩子)
```

### 五阶段审核铁律（v5.0 新增——不可跳过）
```
一审(3集) → 二审一卡(10集) → 三审二卡(30集) → 50集审核 → 100集完本终审
每个审核节点必须执行：剧情大纲 + 人物设定 + 已有内容 + 平台红线 + 格式规范 五维审核
```

### 快准新狠四字铁律
- **快**：10秒内讲清核心冲突，前3秒必须进入冲突状态
- **准**：潜台词替代直白说明，每句台词字面意思≠真正意思
- **新**：每集至少1个新信息（前10集≥身份级），投放后立即引发新冲突
- **狠**：三不沾台词/动作/场景直接砍掉

### 三密度铁律
- 情绪密度高点：每集最强情绪冲击点
- 信息密度高点：每集最重要信息揭露点
- 情节密度高点：每集事件推进最密集点

### 写作红线（零容忍）
- 心理描写 ❌ / 抽象情绪 ❌ / 台词解说设定 ❌ / 说教催泪 ❌
- AI腔连接词 ❌ / 形容词堆砌 ❌ / 不可拍摄内容 ❌
- 被动主角 ❌ / 人物逻辑断裂 ❌

### 平台红线（零容忍——v5.0 新增）
- 涉政敏感 ❌ / 色情低俗 ❌ / 过度暴力血腥 ❌
- 违法示范 ❌ / 封建迷信（须架空） ❌ / 未成年不良引导 ❌
- 性别/地域/职业歧视 ❌ / 版权侵权 ❌
- 结局必须正义战胜邪恶

### 内容铁律
- 不可逆变化：每集至少1项被永久改写
- 选择代价：每集至少1个有代价的选择
- 金手指代价：能力代价每次使用必须兑现

### 情绪银行
- 三类欠条：人物缺口 + 关系缺口 + 真相缺口
- 三层机制：存款(前3集) → 复利(4-50集) → 催收(付费节点)
- 付费四卡点：第10集(一卡) / 第30集(二卡) / 第50集(三卡) / 第75集(四卡)

### 剧本格式铁律（v5.0 新版）
```
第x集
001 日/夜 内/外 场景
人物：xxx xxx
服装：- 角色：服装描述
道具：- 道具名（本集所有道具必须预罗列）
【角色站位+动作关系+环境+氛围】
△ 描述/动作（所有非对白内容以△开头）
角色名：对白
```

### 算法铁律
- 前3秒留存决定60%流量走向 / 完播率≥40%进SS级 / 连续10秒无新信息=死亡
- 钩子密度：前3集≥每20秒1钩 / 4-10集≥每30秒1钩 / 11-30集≥每45秒1钩 / 31-100集≥每45-60秒1钩

### 高概念+高关怀
- 高概念：让观众上头的钩子与反转（每集至少3个"没想到"）
- 高关怀：让观众上心的现实关照（角色是观众生活中的"嘴替"）

### 下沉剧核心赛道（v5.0 新增）
- 男频：战神流 / 高手下山流 / 出狱流 / 巨富归来 / 修仙归来 / 算命天师
- 女频：重生复仇 / 契约婚姻 / 真假千金 / 穿书觉醒 / 离婚后身份揭穿
