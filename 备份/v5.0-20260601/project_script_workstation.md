## v5.0 核心升级（2026.6.1）—— 底层代码重写

### 结构性变化
- **统一 100 集**：取消 45/72/100 可选项，所有剧本固定 100 集
- **五阶段审核制**：一审(3集) → 二审一卡(10集) → 三审二卡(30集) → 50集审核 → 100集完本终审
- **每阶段强制五维审核**：剧情大纲一致性 + 人物设定一致性 + 已有内容逻辑 + 平台红线 + 格式规范
- **全剧 ≥ 5w 字**：单集 ≥ 500 字，确保内容充足

### 剧本格式全面升级
- 新增 `第x集` 每集开头标记
- 新增 `001 日/夜 内/外 场景` 三位数场景编号
- 新增 `人物：xxx xxx` 每场人物强制罗列
- 新增 `服装：` 每个出场人物服装分别描述
- 新增 `道具：` 本集所有道具预罗列，**禁止正文中出现未罗列的道具**
- 新增 `【场景描述】` 角色站位+动作关系+环境+氛围
- 强化 `△` 三角形符号用于所有描述性内容和动作

### 下沉剧题材聚焦
- 男频核心赛道：战神流 / 高手下山流 / 出狱流 / 巨富归来 / 修仙归来 / 算命天师

- 下沉剧特征：强冲突开场 / 简单人物关系 / 快节奏打脸 / 情感直给 / 身份反差 / 口语化对白

### 平台红线新增
- 9 条绝对禁止内容
- 敏感内容规避指南
- 每段产出强制平台红线自检

### 创作命令重构

## 剧本格式（v5.0 新版）

```
第x集

001 日/夜 内/外 场景

人物：角色A 角色B

服装：
- 角色A：服装描述
- 角色B：服装描述

道具：
- 道具1
- 道具2

【角色A站在xxx，角色B坐在xxx，两人之间xxx。环境xxx。氛围xxx。】

△ 描述性内容——角色状态和情绪的外在表现。

△ 角色A走向xxx，伸手拿起道具1。

角色A：对白内容

角色B：对白内容

△ 角色B转身离开。

---

（第x集完）
```

## 核心能力

- 对标分析：自动搜索同赛道爆款短剧并拆解
- 逻辑备忘录：金手指/战力/行为准则/双反派设定 + 游戏化规则设定
- 剧情简介：200字北极星，全剧审核基准
- 大纲：三幕结构 + 情绪曲线 + 双轨节奏 + 付费节点设计 + 10单元规划
- 人物小传：六段式（含"切片潜力"标签 + 角色能力映射表 + 人物弧光四阶段）
- 分集梗概：100集完整分集梗概，含钩子类型+不可逆变化类型+情绪锚点
- 剧本正文：v5.0新格式（服装/道具/【】/△），五阶段分段产出
- 五阶段审核：一审/二审/三审/50集/终审，每阶段五维逻辑审核
- 四种复盘：钩子/逻辑/节奏/数据（含情绪欠条健康度 + 平台红线复查 + 道具复查）
- 记忆点：长剧集进度保存与恢复
- 写作红线：9 条禁令 + 人物逻辑三问
- 平台红线：9 条绝对禁止 + 敏感内容规避 + 强制自检
- 内容铁律：不可逆变化 + 选择代价 + 金手指代价
- 情绪银行：3 类欠条 + 3 层机制
- 黄金四步：4 步单集公式 + 10 集循环结构 + 逆向设计法
- 热梗素材库：热梗名梗 + 爆款台词 + 名场面拆解
- Word 自动输出：正文/自检分离，审核报告独立保存

## 使用方式

双击桌面 `编剧工作台.lnk` 启动。

**Why:** 用户是职业短剧编剧，专注于下沉市场100集标准化短剧产出。v5.0 通过统一集数、五阶段审核制、新剧本格式和平台红线机制，确保每部剧本的商业竞争力和合规性。

**How to apply:** 当用户在编剧工作台目录下工作时，所有创作需遵循 CLAUDE.md v5.0 中定义的规则和流程。每次创作前必须加载 CORE_CREATIVE_DNA.md。所有产出必须经过五阶段审核，每阶段执行五维逻辑审核。
