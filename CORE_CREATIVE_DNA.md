## 1. 用户画像

**老张**，34岁外卖员，累了一天躺沙发刷手机。
他看不懂的句子删。他不兴奋的段落改。不要让他用脑子理解——让他用身体感受。

## 2. 格式铁律

### 2.1 场景头
```
{序号} {日/夜} {内/外} {地点}
```
示例：`001 日 外 古玩堂` / `010 夜 内 藏家交流会大厅`

### 2.2 人物/服装/道具段
每变场景必须列出：
```
人物：角色A，角色B，角色C
服装：角色A（描述），角色B（描述），角色C（描述）
道具：物品A，物品B，物品C
```

### 2.3 场景概述
每场景开头必须有【】概述：
```
【古玩堂门口，林北被几名保安架着扔出大门。】
```

### 2.4 三件套
| 元件 | 格式 | 规则 |
|------|------|------|
| △动作 | `△动作描述` | △后只能写动作或场景描述。不能有情绪词（冷笑/不屑/不慌不忙）、不能有小说体、不能有解释性话语。情绪通过动作和对白本身传递。 |
| 角色os | `角色名os：内容` | 小写os。少用。观众能通过画面看懂的内容，os全删。林北os基于黄金瞳信息透视有恃无恐。 |
| 【特效】 | `△【特效】描述` | 方括号。标注意象化。例：`△【特效（意象化）】碗中一道金光没入林北眼睛。` |

### 2.5 信息面板
黄金瞳鉴定结果以独立信息行展示：
```
△信息：明代铜鎏金释迦牟尼像，价值：1200w。
```
信息面板是画面的一部分，直接展示，不是角色说出来的。

### 2.6 格式范例（参照前20集基线）
```
001 日 外 古玩堂
人物：林北，保安，路人
服装：林北（破烂肮脏衣服），保安（制服），路人（华服）
道具：破碗，包
【古玩堂门口，林北被几名保安架着扔出大门。】
△保安队长拿着一个破碗走到林北面前，上下翻看两眼。
保安队长：一个臭乞丐，拿着个破碗就敢说是古董，糊弄鬼呢，（将碗扔在林北身前，碗碎）赶紧滚，再敢来，老子打断你的腿！
△保安队长啐了一口，带人离开。
△林北急忙爬过去捡碎片，被瓷片划破手指，血珠渗出。
林北：（愣神）我的碗！这是我爸留给我的唯一的东西...
```

### 2.7 集分界线
`第N集` 独占一行。跨集连续动作允许下集开头重复收尾作checkpoint。

## 3. 节奏规则

### 3.1 字数
每集正文 450-600 字。场号/人物/服装/道具不计入字数。超标时三刀自查：
1. 大纲是否过多？（一集只能1-2个核心事件）
2. 对白和动作是否足够精简？
3. 内容是否全部推动情绪或剧情？不能就删。

### 3.2 事件密度
- **一集只讲一个核心事件**。大纲里的一句话 = 一集的内容。
- 大纲格式：一句话包含事件+钩子。例：「林北穿旧西装到藏家交流会，门卫认出他是乞丐拦下羞辱，林北亮出苏怀远请柬打脸。赵四爷出现嘲讽，林北硬闯进入大厅。钩子：林北目光落在供桌铜佛上，眼底金光一闪。」
- 每个核心事件在正文中展开成3-5个△段，不是一个段落。

### 3.3 打脸公式
```
被看不起 → 亮出底牌 → 对方不信 → 验证底牌 → 全场震惊 → 对方吃瘪
```
每1.5集至少一次打脸事件。

### 3.4 钩子规则
| 类型 | 功能 | 占比 | 示例 |
|------|------|:--:|------|
| 预告型 | 告诉观众下集有好戏 | 40% | 「赵四爷盯着林北背影：查查。」 |
| 悬念型 | 制造疑问 | 30% | 「林北目光落在铜佛上，眼底金光一闪。」 |
| 威胁型 | 制造危险 | 15% | 「赵四爷冷笑：你走不出这个门。」 |
| 转折型 | 剧情翻转 | 15% | 「林北把银行卡拍桌上：一个亿，卖不卖。」 |

同一类型钩子不连续使用超过2集。钩子每集结尾必须强力——观众想点下一集。

### 3.5 节奏模块
每6-7集一个模块，模块尾端：打脸高潮 + 新线索（新目标+新困境）。
每个循环附带：新困境（更大的对手/更难的规则/更紧的时间）+ 新技能展示 + 新资源上线。

### 3.6 伏笔回收
- 要么当集回收，要么三集内回收。
- 超过三集未回收 → 砍掉。
- 伏笔回收计划写在大纲末尾标注。

## 4. 语言守则

### 4.1 禁词清单
出现1次警告，≥2次退稿：
`仿佛` `忽然` `竟然` `原来如此` `突然之间` `殊不知` `却不知` `只见` `就这样` `其实` `原来` `终于`

替代方式：动作直接写，不加修饰词。

### 4.2 对白铁律
- **每句对白必须推动剧情或展示人设**——不写过渡句、气氛句。
- **每句至少承载两个信息层**（身份+动作、威胁+反转、情感+悬念）。
- **对白要有精准信息**。错误：「一个臭乞丐，这地方也是你能来的？」正确：「站住！你不是上个月在天桥翻垃圾桶那个乞丐吗？这是你能来的地儿？」
- **观众能通过画面看懂的内容，对白全删**。林北捡碎片时沉默比说话强10倍。
- **林北对白风格**：嘴贱不出戏，口语化（妈的/卧槽/爽/给高了/想啥好事儿呢）。基于黄金瞳信息他有恃无恐，知道对手不知道的情报所以敢出手。

### 4.3 调度三原则（来自哥哥调度品味）
1. **先写环境对身体的冲击，再写角色反应。** 错误：三人手持棍棒来。正确：泥水溅起，一双双破烂胶鞋踩进浑浊的水坑。远处持枪守卫来回踱步。
2. **危险先于发现出现，制造延迟满足。** 守卫先砸铁皮桶，林北才低头发现金条。观众的心悬着：他能不能在被发现前藏好？
3. **超能力用身体反应表达，不写成光效大片。** 错误：一尊金龙冲天而起。正确：瞳孔一缩——眼底闪过一道极快的金色光晕。

### 4.4 动作描写
- 每个动作用3-5步完成，加入感官细节（泥浆从鞋底挤出、血珠渗出、铜屑飞溅）。
- 删掉所有「他说/她做」句式，改成身体在环境中的反应。
- △后不能有情绪词，情绪通过对白节奏和动作密度自然传递。

## 5. 人物设计

### 5.1 固定角色原型
| 角色类型 | 功能 | 出场时间 | 常见身份 |
|---------|------|:--:|------|
| 第一反派 | 初期看不起主角、被打脸 | 第1-2集 | 保安/摊主/守卫/前女友 |
| 行业反派 | 中期大反派、有资源背景 | 第4-8集 | 大少/会长/富二代 |
| 鉴宝人 | 打通经济循环、收宝变现 | 第2-4集 | 古董行老板/拍卖行 |
| 女性角色 | 商人型（求助+平台）+ 江湖型（引路+人脉） | 第2-6集 | 大小姐/老板 |
| 路人 | 捧哏、放大打脸效果 | 贯穿 | 围观群众 |
| 家人 | 情感锚点/驱动目标 | 第5-10集 | 父亲/母亲 |

### 5.2 反派设计原则
- **不降智**：反派贪但不蠢。王二有理由贪婪（干了十二年）、沈均有资源（雇风清扬）、风清扬有风骨（输了认）。
- **打脸后有后果**：丢工作、输产业、输名声。
- **反派链对应社会层级**：地头蛇→行会→官商→上层。

### 5.3 主角声音
- 男频：嘴贱、不吃亏、钱第一驱动力。用「愤怒→碾压」情绪弧。
- os基于黄金瞳信息——他知道别人不知道的，所以有恃无恐。
- 每集必须有一次「林北味」的对白或动作（讨价还价/耍贱/冷嘲热讽）。

## 6. 收入爬坡

每6-8集一次数量级跃迁：
- 第1-10集：10w-100w级
- 第11-30集：100w-1000w级
- 第31-60集：1000w-5000w级
- 第61-80集：资产型（店铺/矿山/人脉），不单纯靠捡漏

80集内主角净资产不超过5000w。保持「还在奋斗」的底层姿态，不让观众失去共情。

古董品类和赌石种类不得重复。

---

## 7. 能力成长路径

外挂不是一次觉醒给足。每10-15集解锁一个新能力面：
面板信息 → 远距透视 → 透视内部 → 慢动作 → 隐藏价值识别 → 暗格/夹层透视

---

## 8. 水字数自查

每△段必须满足三条之一，否则删除：
1. 推进剧情
2. 展示人物
3. 制造冲突

附加检查：
- 对白里有没有「过渡句」？有没有单纯解释画面内容的话？
- 两个△段能不能合并？三段对白能不能砍成一句？
- 老张能不能看懂这个场景？他会不会兴奋？
