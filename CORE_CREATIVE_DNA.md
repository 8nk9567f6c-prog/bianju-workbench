## 一、全局创作 DNA（每次创作必须内化）

### 核心理念
- 短剧不是作品，是产品。每一帧的唯一使命：让用户不划走。
- 短剧卖的是观众心里还不清的"情绪欠条"。
- **S+ 定位铁律**：每部剧本从立项到完本以 S+（≥85分）为最低标准。A 级 = 废稿。
- 统一 100 集，全剧 ≥ 5w 字（目标 55000+），单集 ~600 字（550-650 字区间）。下沉剧男频/女频爽文短剧。
- **自审系统**：审核不再独立执行——Agent 2 内置三维护大纲自审（剧情一致性12项/人物一致性10项/逻辑一致性11项），Agent 3 内置四维护剧本自审（写作红线15项/平台红线12项/格式规范15项/节奏情绪12项）+ MCP 17 工具自动验证。
- **全平台 SS/S+ 对标**：每部剧立项前完成 8 基因对标拆解（钩子/节奏/情绪/人物/爽点/台词/付费/切片）。

### 黄金四步铁律
```
情节承接(0-5s) → 冲突升级(6-45s, 强度/风险/代价三维升级) → 价值闭环(46-55s, 不可逆转变) → 下集勾连(56-60s, 因果链)
```


### 快准新狠
- **快**：10秒内讲清核心冲突，前3秒进入冲突状态
- **准**：潜台词替代直白说明，每句台词字面≠真意
- **新**：每集至少1个新信息（前10集≥身份级），投放后立即引发新冲突
- **狠**：三不沾台词/动作/场景直接砍掉

### 信息前置·废话后置（v5.4 吸收自 drama-creator v2.2）
> "台词的功能是传递信息，不是填充时间。"

- **信息前置**：每场戏第一句台词必须抛出身份/冲突/目标三者之一。观众3秒内必须知道"谁在跟谁斗，为了什么"。
- **废话后置**：寒暄/客套/过渡性对话一律砍掉或移到情绪释放后。如果一句台词删掉不影响剧情理解→立即删除。
- **功能化测试**：每句台词问三个问题——①推动情节？②揭示人设？③制造/化解冲突？三不沾→删。
- **钩子优先**：集末最后一句必须是押弹簧/放弹簧的信号，不能是平淡过渡句。

### 情绪银行（宏观跨集）+ 情绪弹簧（微观单集）—— v5.4 双模情绪引擎

**情绪银行（宏观——跨集情绪账本）**
- 三类欠条：人物缺口 + 关系缺口 + 真相缺口
- 三层机制：存款(前3集) → 复利(4-50集) → 催收(付费节点)
- 付费四卡点：10集(一卡) / 30集(二卡) / 50集(三卡) / 75集(四卡)

**情绪弹簧（微观——单集情绪二态，v5.4 吸收自 drama-creator v2.2）**
> "观众的情绪是根弹簧，每一集要么压到最紧，要么瞬间松开。没有中间状态。"

- **压弹簧**：制造误会、主角被压制、反派嚣张、危机降临 → 积蓄负面情绪(愤怒/憋屈/紧张/好奇)
- **放弹簧**：主角反转、打脸反派、揭露真相、化解危机 → 瞬间释放情绪制造爽感
- **硬性铁规**：每集结束必须是"压弹簧"或"放弹簧"二态之一，**不得中性收尾**
- **全剧扫描(v5.4+)**：不仅查集末200字，按"---"分场后分三段(开/中/尾)独立检测压/放状态 + 平区检测(连续20+字无情感信号)。每段不得中性，全剧≤2处平区。
- **五节点节拍映射(v5.4+)**：视觉锤(0-5%)→立场锚(5-20%)→爆破点(20-50%)→爽点(50-85%)→端钩(85-100%)，每段映射预期情感信号，≥3/5节点通过。
- **充值转化**：付费卡点前必须连续2-3集压弹簧 → 付费点放弹簧，免费用户看不到释放=转化动力
- **与情绪银行协作**：情绪银行决定"欠什么债、什么时候催收"；情绪弹簧决定"单集是压还是放"

**情绪锚点(v5.4+ 自动检测)**
- 五型锚点：爽(打脸反转碾压) / 虐(牺牲背叛绝望) / 暖(守护陪伴原谅) / 悬(秘密阴谋倒计时) / 燃(觉醒突破对决)
- 自动识别：全文关键词profile扫描 → 主导锚点判定 → 情绪转向次数(≥3次=合格)
- 锚点轮换：连续3集不得重复同一主导锚点(跨集自动验证)

**角色情感范围(v5.4+ 自动分析)**
- 每角色提取对白 → 迷你五锚点扫描 → 情感宽度(1-5)
- 宽度=0：无情感信号 / 宽度=1且≥5句对白：可能情感单调(确认是否刻意) / 宽度>4：可能情绪不一致
- 始终advisory——扁平角色(如冷面护卫)可能是刻意设计

- 前3秒留存决定60%流量 / 完播率≥40%进SS级 / 连续10秒无新信息=死亡

- 高概念(每集≥3个"没想到")+高关怀(角色是观众嘴替)，缺一不可

**10 维度 100 分制**：赛道匹配(10%)/开场冲击(15%)/钩子密度(12%)/情绪银行(10%)/人物驱动(10%)/黄金四步(10%)/节奏留存(10%)/付费转化(8%)/下沉适配(8%)/平台合规(7%)
