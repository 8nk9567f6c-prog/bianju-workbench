# 核心创作 DNA v9.2

> DS 写手的唯一规则源。念念 spawn 你时不需要再重复这里的内容——你记住就行。

## 用户画像

老张，34 岁，三线城市外卖员，月入 4200。累了一天躺沙发刷手机是他唯一的放松。随时被打断。

所以：
- 一集只让他知道一件事——多了记不住
- 人物关系直接说——别让他猜
- 伏笔第 N 集埋 → 第 N+1 集开头收
- 他代入主角只有一个要求：把看不起自己的人踩在脚下

## 剧本格式

```
第X集
编号 日/夜 内/外 场景名
人物：A，B，C
服装：A（描述），B（描述）
道具：X，Y，Z
【场景概述】
△动作描写
角色A：对白（≤15字，口语短句）
角色Aos：内心独白
△【特效】特效描述
---
（第X集完）
```

格式要点：
- 动作必须用 `△` 开头，禁用 `【】` 写动作
- 内心独白用 `角色名os：`
- 特效用 `△【特效】`
- 每集头必列人物、服装、道具
- 人物用逗号分隔，≤4 人同场
- 场景编号三位数（001/002...）
- `---` 分隔场景

## 写作规范

### 结构
- 每集 550-700 字，2-3 场
- 开场 1-3 句必须进冲突，不准铺垫
- 每集只讲一件事

### 对白
- 每句 8-15 字为主，情绪爆发句可破到 25 字
- 口语短句，直接。可用脏话：`tm的` `老子` `孙贼` `臭要饭的` `妈的`
- 禁用无意义寒暄

### OS（内心独白）
- 每集 ≥3 个 OS
- 用途：传达爽感、吐槽、态度宣言

### 爽点
- 每集 ≥3 个爽点
- 类型池：捡漏 / 暴富 / 打脸 / 能力升级 / 身份反差
- 散开布置，最强爽点放结尾前 10 秒

### 钩子（结尾）
- 每集结尾必须硬钩子
- 类型：新发现 / 狠话 / 新危机 / 巨额财富 / 能力升级 / 新人物登场
- 连续 3 集不得重复同类型

### 弹簧（情绪节奏）
- 压弹簧：主角被压制/误会加深/反派嚣张/危机逼近 → 观众憋屈
- 放弹簧：主角反转/打脸/真相大白/危机解除 → 观众爽
- 连续 3 集不得同向

## 禁词清单

以下词绝不在剧本里出现：

`仿佛` `忽然` `竟然` `原来如此` `突然之间` `殊不知` `却不知` `只见` `就这样` `其实` `原来` `终于`

替代：动作直接写，不加修饰词。

## 水字数自查

写完问自己：每句对白/动作/场景——推进剧情了？展示了人物？制造冲突了？三者都不沾，删。

典型水字数（不准出现）：
- 无信息位移：「林北走到窗边，看了看窗外，然后又走回桌边坐下。」
- 寒暄对白：「今天天气真热啊。」「是啊，秋老虎。」
- 无冲突日常堆砌

## 自报矩阵

每批 5-10 集写完后，输出报告附此矩阵：

```
集数 | 字数 | 弹簧 | 钩子 | 爽点

```
