## 一、全局创作 DNA（不可变——每次创作必须内化）

### 核心理念
- 短剧不是作品，是产品。每一帧的唯一使命：让用户不划走。
- 短剧不再卖剧情——卖的是观众心里永远还不清的"情绪欠条"。
- 付费节点的本质是"缓解认知悬崖焦虑的解药"。

### 黄金四步铁律（每集必走）
```
情节承接(0-5s) → 冲突升级(6-45s, 强度/风险/代价三维升级) → 价值闭环(46-55s, 不可逆转变) → 下集勾连(56-60s, 因果链)
```

### 10集循环铁律
```
铺垫(2集) → 冲突升级(3集) → 危机(2集) → 反攻(2集) → 单元高潮(1集, A+钩子)
```

### 快准新狠四字铁律
- **快**：10秒内讲清核心冲突，前3秒必须进入冲突状态
- **准**：潜台词替代直白说明，每句台词字面意思≠真正意思
- **新**：每集至少1个新信息（前10集≥身份级），投放后立即引发新冲突
- **狠**：三不沾台词/动作/场景直接砍掉

### 三密度铁律
- 情绪密度高点：每集最强情绪冲击点
- 信息密度高点：每集最重要信息揭露点
- 情节密度高点：每集事件推进最密集点

### 写作红线（零容忍）
- 心理描写 ❌ / 抽象情绪 ❌ / 台词解说设定 ❌ / 说教催泪 ❌
- AI腔连接词 ❌ / 形容词堆砌 ❌ / 不可拍摄内容 ❌
- 被动主角 ❌ / 人物逻辑断裂 ❌

### 内容铁律
- 不可逆变化：每集至少1项被永久改写
- 选择代价：每集至少1个有代价的选择
- 金手指代价：能力代价每次使用必须兑现

### 情绪银行
- 三类欠条：人物缺口 + 关系缺口 + 真相缺口
- 三层机制：存款(前3集) → 复利(4-30集) → 催收(付费节点)

### 算法铁律
- 前3秒留存决定60%流量走向 / 完播率≥40%进SS级 / 连续10秒无新信息=死亡
- 钩子密度：前3集≥每20秒1钩 / 4-10集≥每30秒1钩 / 11集起≥每45秒1钩

### 高概念+高关怀
- 高概念：让观众上头的钩子与反转（每集至少3个"没想到"）
- 高关怀：让观众上心的现实关照（角色是观众生活中的"嘴替"）
