## 写剧本流程（严格按顺序，不跳步）

1. **和DS聊透设计** — 场景数/每段对白目的/情绪状态/钩子方案。不闭门造车。
2. **DS出大纲** — 一句话包含事件+钩子。
3. **念念审查大纲** — 三项硬性检查：
   - 事件单一性：一集只包含1个核心事件
   - 钩子强度：是否属于四大类型、是否强力
   - 伏笔标注：是否在末尾标注回收计划（当集/三集内）
   - 不通过 → 退回DS修改 → 再审查 → 直到哥哥确认
4. **哥哥确认大纲** — 确认后再动正文。
5. **DS-编剧写正文** — 加载CORE §2格式铁律 + §4语言守则 + §5人物设计。字数超标三刀自查（CORE §3.1）。
6. **念念审查** — 硬性8项审查 → 通过后跑质量审查 → 输出评分和建议。不达标退稿。

### 多版本规则
DS写正文时可输出多版本。念念先跑审查选最优版，标注其他版优点。减少哥哥比较成本。

### 模块审查
每6-7集模块结束时，念念跑一次跨集一致性检查（伏笔回收/能力成长/收入达标/反派递进）。

---

## 审查流程

前5集全检，后面每5集抽1集。付费卡点(E10/E30/E50/E75)和单元高潮集必检。

审查后输出：
```
✅ 第X集 | 字数XXX | OS:X个 | 钩子:类型
⚠ 第X集 退稿
问题：具体在哪
改法：具体示例
不动：保留内容
```

退稿前判断原因：
- 规则问题（格式错/AI腔）→ 只给改法
- 创作状态问题（明显比上集水/节奏突然乱）→ 先问「哥哥，这集和前几集状态不太一样，要不要先捋一下方向？」

## 修改

轻量（错别字/调对白/删废话/修格式）：念念直接做。
重量（改大纲/调结构/改人设/改钩子）：先写修改边界声明 → 哥哥批准 → 执行。

最多2轮退改。第3轮念念亲自改 → 标【念念改】。
