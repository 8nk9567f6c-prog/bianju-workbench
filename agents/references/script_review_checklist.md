# 剧本自审清单 — 58 项完整检查标准

> **加载触发**: Agent 3 每集/每阶段写完后，执行自审时读取
> **父文件**: `agents/agent_content_expansion.md`
> **用途**: 四维护自审（写作红线+铁律 15 项 + 平台红线 12 项 + 格式规范 15 项 + 节奏与情绪 12 项）

## 维度一：写作红线 + 内容铁律（19 项）

| # | 检查项 | 判定标准 | 检查方式 |
|---|--------|---------|---------|
| 1 | 心理描写 | 无"他心想"、"他感到"、"他意识到"、"他明白" | MCP `check_writing_redlines` + 人工抽查 |
| 2 | 抽象情绪 | 无"愤怒涌上心头"、"悲伤弥漫"等——全部转为可拍动作 | MCP + 人工 |
| 3 | 台词解说设定 | 无"你忘了吗？我们家族世代..."型对白 | MCP + 人工 |
| 4 | 说教催泪 | 无"人生啊..."、"你一定要坚强..."型对白 | 人工 |
| 5 | AI 腔 | 无"然而/因此/于是/随后/与此同时/就这样/不仅如此" | MCP `auto_fix.py` + 人工 |
| 6 | 形容词堆砌 | 无"豪华气派金碧辉煌的大厅" → 改为物象罗列 | MCP + 人工 |
| 7 | 不可拍摄 | 所有描写摄影机能拍到？给摄影师能不提问直接拍？ | 人工逐句 |
| 8 | 被动主角 | 主角永远主动做选择，不能"被带到"、"被告诉" | 人工 |
| 9 | 人物逻辑断裂 | 三问通过：角色知道什么？此刻能做到什么？以ta性格会这么做吗？ | 人工 |
| 10 | 不可逆变化/集 | 每集 ≥ 1 项永久改写（人物目标/权力格局/信息优势/关系性质/规则打破） | MCP `check_content_rules` |
| 11 | 选择代价/集 | 每集 ≥ 1 个有代价的选择 | MCP `check_content_rules` |
| 12 | 金手指代价/次 | 金手指每次使用必须兑现代价，100 集 ≥ 3 次重大代价 | MCP + 人工累计 |
| 13 | 散文质感 | 可拍摄 ✓ / 感官 ≥ 2 种 ✓ / 节奏变化 ✓ / 声音区分 ✓ / 无 AI 虚词 ✓ | 人工 + MCP `check_dialogue_quality` |
| 14 | 重复短语检测 | 与前 10 集交叉比对，同一描写/比喻/口头禅 ≥ 3 次复用 → 替换 | 人工 |

| 16 | 一事一集 | 本集是否只讲一件事？一句话概括唯一事件，若需用"和"连接 → 事件过多 | 人工 |
| 17 | 对话主力 | 对白推进剧情占比 ≥ 70%？场面调度限于基础微动作（进门/出门/坐下/站起/递物/接物）？ | 人工 |
| 18 | 伏笔速回收 | 本集若埋设伏笔→下一集必须揭示；上一集伏笔未回收→本集必须处理 | 人工 + hooks_ledger.json |
| 19 | 剧情简化 | 支线 ≤ 1 条且仅服务主线？无多线并行叙事？ | 人工 |

## 严重程度分级与处理规则

| 等级 | 定义 | 处理 | 示例 |
|------|------|------|------|
| 极高（红线） | 不可交付，必须修正 | 立即修正 → 重审该维度 | 平台红线触碰/心理描写/正义结局缺失/格式严重错误 |
| 高（影响 S+） | S+ 将跌破 85 | 优先修正 → 重审该维度 | 钩子密度不足/情绪弹簧中性/10s 空白/被动主角 |
| 中（影响体验） | S+ 降档但不跌破 85 | 标注 → 建议修正 | AI 腔残留/形容词堆砌/对白过长 |
| 低（优化建议） | 不影响交付 | 标注 → 可选修正 | 散文质感可加强/感官种类可丰富 |

## 自审不通过处理流程

1. MCP 自动标记 ❌ 项 → `auto_fix.py` 自动修复 → 复验
2. MCP 无法自动修复项 + 人工核查 ❌ 项 → 列出清单 + 严重程度
3. 按优先级修正：极高 → 高 → 中
4. 修正后重新自审该维度
5. 全部 ✓ 或仅剩低问题 → 方可交付
6. **连续 2 次自审不通过 → 停止并向用户报告阻塞点，请求方向指导**
