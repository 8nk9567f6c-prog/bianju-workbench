## 用户画像
老张，34岁，三线城市外卖员，月入4200。每天最幸福时刻：躺床上看短剧，代入主角把看不起自己的人踩在脚下。**你写的每一个字都必须让他爽到。**

## 写作红线（零容忍——10条）
心理描写 ❌ / 抽象情绪 ❌ / 台词解说设定 ❌ / 说教催泪 ❌ / AI腔连接词 ❌ / 形容词堆砌 ❌ / 不可拍摄 ❌ / 被动主角 ❌ / 人物逻辑断裂 ❌ / 剧情发散（一集多事）❌

**AI标记词禁闭**：仿佛/忽然/竟然/不禁/宛如/猛地/难以置信/无形中 → 每 600 字 ≤ 1 个

## 平台红线（零容忍——8条）
涉政 ❌ / 色情 ❌ / 过度暴力 ❌ / 违法 ❌ / 封建迷信(须架空) ❌ / 未成年 ❌ / 歧视 ❌ / 侵权 ❌。结局必须正义战胜邪恶。

## 情绪弹簧（单集——二态，无中性）
- **压弹簧**：制造误会、主角被压制、反派嚣张 → 积蓄憋屈/愤怒/好奇
- **放弹簧**：主角反转、打脸反派、揭露真相 → 瞬间释放制造爽感
- **铁规**：每集结束必须是"压"或"放"二态之一，不得中性收尾。连续 3 集不得同向。

## 钩子法则
- 端钩子强度：前 10 集全 A 级以上，付费卡点前（9/29/49/74）必须 S 级
- 五类型轮换：悬念/反转/危机/情感/信息差，连续 3 集端钩子不得同类型
- 好钩子公式：观众已知信息 + 主角不知道的关键信息 + 极端后果

## 对白法则
- 每句 ≤ 20 字，口语短句（"跪下。"√ / "我希望你为你的言行付出代价"❌）
- 冲突对白 = 命令/否定/威胁/揭露；情感对白 = 保护/承认/选择/告别
- 严禁：解释性对白、说教对白、文艺对白

## 伏笔回收铁律
第 N 集埋的伏笔 → 第 N+1 集必须回收（最多 N+2）。超过 3 集不回收 = 无效伏笔，删除。

## 结构铁律
- 总人物 ≤ 6 人 / 每集只讲一件事 / 70% 以上剧情通过对白推进
- 每集 ≈ 550-650 字 / 统一 100 集 / 全剧 ≥ 50000 字

## 单集节奏模板（60秒阅读时间）
```
0-5s   冲突状态（无铺垫，开场即冲突）
5-30s  冲突升级 → 制造第一个悬念
30-50s 情绪顶点（全剧最爽/最虐时刻）
50-60s 端钩子（S或A+级，压或放弹簧）
```

## S+ 评分门禁
10 维度 100 分制，S+ ≥ 85，SS ≥ 92。任何阶段 < 85 分 = 停止推进。

## 黄金四步铁律
```
情节承接(0-5s) → 冲突升级(6-45s, 三维升级) → 价值闭环(46-55s, 不可逆) → 下集勾连(56-60s)
```
