# Agent 调度 v9.2

> 念念 spawn DS 写手时使用。分层加载：一次加载规则 → 每次只喂动态上下文。

## 分层加载

**一次加载**（整个项目不变，DS 开写前读一遍）：
- CORE_CREATIVE_DNA.md 全文（规则 + 格式 + 用户画像）
- ANCHOR.md 前三节（北极星 + 人物速查 + 金手指体系）

**每次必喂**（每批 spawn 时念念构建）：
- 哥哥原始输入（原文粘贴）
- 目标集数 + 当前弧光位置 + 大纲段
- 上一集结尾钩子内容
- 上次审查结论（如有）

## 念念审查速查

| # | 查什么 | 标准 |
|---|--------|------|
| 1 | 格式 | △/os/【特效】三件套齐全 |
| 2 | AI 腔 | 禁词 ≤1 次 |
| 3 | 结尾能量 | 硬钩子 + 弹簧方向，连续 3 集不重复 |
| 4 | 水字数 | 念念直觉，觉得水就标 |

前 5 集全检。第 6 集起每 5 抽 1。卡点(E10/E30/E50/E75)+单元高潮必检。
1 次不达标 → 回退一档。轻量修改仅查格式+AI腔。

## 异常处理

- 格式小错（≤3 处）→ 念念直接修，不退
- 字数超限 → 退 DS 精简
- 结构性错误（一事多集/人物超限）→ 退 DS 重写
- DS 中断 → 检查已有文件，从缺失最小集数继续
